٦۷ - سورة الملك
( ۱ ) تبارك الذي :تعالى و تمجّد أو تكاثر خَيْره
( ۱ ) بيده الملك :له الأمر و النّهي و السلطان
( ۲ ) خلق الموت :أوْدَه . أو قدّرَه أزلاً
( ۲ ) ليبلوكم :ليَختبركم فيما بين الحياة و الموت
( ۲ ) أحسن عملا :أصْوبَه و أخلَصَه أو أسْرَع طاعة
( ۳ ) طباقا :كلّ سَماءٍ مَقـْـــِبـيّة على الأخرى
( ۳ ) تفاوت :اختلاف و عَدَم تناسب
( ۳ ) فطور :شقوق و صدوع أو خلل
( ٤ ) كرتين :رَجْعَتين رَجعة بعد رجعة
( ٤ ) خاسئا :صاغرا لعدم ِوجْدان الفـُـطور
( ٤ ) هو حسير :كليلٌ من كثرة المراجعة
( ۵ ) بمصابيح :بكواكب عظيمة مُضيئة
( ۵ ) رجوما للشياطين :بانقـضاض الشّهب منها عليهم
( ۷ ) شهيقا :صَوْتا مُـنـْـكرًا كصوت الحمير
( ۷ ) تفور :تغلي بهم غليَان القِدْر بما فيها
( ۸ ) تكاد تميّـز :تتقطّع و تتفرّق و تنـْـشقّ
( ۸ ) فَـوْج :جماعة من الكفار
( ۱۱ ) فَسُحْقا :فبُعْدا من الرّحمة و الكرامة
( ۱۵ ) الأرض ذلولا :مُذلـّـلة ليّـنـَـة سَهْلة تستقـرّون عليها
( ۱۵ ) مَناكبها :جَوَانبها . أو طُرُقِها و فِجَاجها
( ۱۵ ) إليه النشور :إليه تبْعثون من القبور
( ۱٦ ) منْ في السّماء :أمْرُه و قَـضاؤُه و سلطانه
( ۱٦ ) يَخسف بكم :يُـغَوّر بكم
( ۱٦ ) هِي تمور :تـَـرْتجّ و تضطرب فتعلو عليكم
( ۱۷ ) حاصبا :ريحًا من السّماء فيها حَصباء
( ۱۷ ) كيف نذير :كيف إنذاري و قدْرتي على العِقاب
( ۱۸ ) كان نكير :إنكاري عليهم بالإهلاك
( ۱۹ ) صافّات و يَقبضْن :باسطاتٍ أجنحتهنّ في الجوّ عند الطيران و يَضْمُمْـنـَـها إذا ضربنَ بها جُنوبهنّ
( ۲۰ ) أمّن هذا ؟ ؟ :بَلْ مَنْ هذا ؟ ؟
( ۲۰ ) جُندٌ لكم :أعْوان لكم و مَنـَـعَة
( ۲۰ ) غرور :خديعة من الشيطان و جنده
( ۲۱ ) لجّوا في عُـتوّ :تمادَوْا في استكبار و عناد
( ۲۱ ) نفور :شِرَادٍ و تباعد عن الحقّ
( ۲۲ ) مُكِـبّا على وجهه :سَاقطا عليه لا يَـأمن العثور
( ۲۲ ) يَمْشي سَويّا :مُسْـتـَويًا مُنتصِبا سالمًا من العثور (مَثلٌ للمشرك و الموحّد)
( ۲٤ ) ذرأكم :خَلقكم و بَثـّـكم و فرّقكم
( ۲۷ ) رأوه زلفة :رَأوُا العذاب قريبا منهم
( ۲۷ ) سيئت :كَئِبَتْ و اسْوَدّتْ غمّا و ذلاّ
( ۲۷ ) به تدعون :تـَـطلبون أن يُعجّل لكم استهزاءً
( ۲۸ ) أرأيتم :أخبروني أو أروني
( ۲۸ ) يُجير الكافرين :يُنـَجّـيهمْ. أو يَمنعهم أو يؤَمّنهم
( ۳۰ ) غَوْرا :غائرا ذاهبا في الأرض لا يُنال
( ۳۰ ) بماء معين :جَار أو ظاهرٍ . سَهْل التـّـناوُل
