٤۸ - سورة الفتح
( ۱ ) فتحا مبينا :هو صُـلح الحديبية عام 6 هـ
( ٤ ) السكينة :السكون و الطمأنينة و الثبات
( ٦ ) ظن السوْء :ظنّ الأمر الفاسد المذموم
( ٦ ) عليهم دائرة السّوْء :دعاءٌ عليهم بالهلاك و الدّمار
( ۹ ) تعزروه :تـَـنصروه تعالى بنصْرَة دينه
( ۹ ) توقّـروه :تعظّموهتعالى و تبجّـلوه
( ۹ ) تسبّحوه :تنزّهوهعما لا يليق بجلاله
( ۹ ) بكرة و أصيلا :غدوة و عشيّـا، أو جميع النّهار
( ۱۰ ) نـَـكثَ :نـَـقـَـض البيعة و العَهْد
( ۱۱ ) المخلـّـفون :عن صحْبتك في عمرة الحديبية
( ۱۲ ) لنْ يَنـْقلب :لن يعود إلى المدينة
( ۱۲ ) قوْمًا بُورا :هالكين أو فاسدين
( ۱۵ ) ذرونا نتـّبعكم :اتـْركونا نخرجْ معكم لخيْـبَـر
( ۱۵ ) كلام الله :حُكمَهُ باختصاص أهل الحديبية بالمغانم
( ۱٦ ) أولي بأس شديد :أصحاب شدّة و قوّة في الحَرْب
( ۱۷ ) حَرَجٌ :إثم في التخلّـف عن الجهاد
( ۱۸ ) يبايعونك :بَيْعة الرضوان بالحديبيّة
( ۱۸ ) فتحا قريبا :فتح خيبر عام سبع ٍ
( ۲۱ ) أحاط الله بها :أعدّها لكم أو حَفظها لكم
( ۲٤ ) ببطن مكّة :بالحديبية قرب مكّة
( ۲٤ ) أظفركم عليهم :أظهركم عليهم و أعلاكم
( ۲۵ ) الهدْيَ :البُدْن التي سَاقها الرسول صلى الله عليه و سلم
( ۲۵ ) مَعْكوفا :مَحْبوسًا
( ۲۵ ) مَحلـّـه :المكان الذي يحلّ فيه نحرُه
( ۲۵ ) تطئوهم :تـُـهلكوهمْ مَعَ الكفـار
( ۲۵ ) مَعَرّة :مَكروهٌ و مشقّة، أو سُـبّـة
( ۲۵ ) تزيّـلوا :تميّـزوا من الكفارفي مكة
( ۲٦ ) الحميّة :الأنفة و الغضب الشديد
( ۲٦ ) سكينته :الإطمئنان و الوقار
( ۲٦ ) كلمة التقوى :كلمة التوحيد و الإخلاص
( ۲۷ ) فتحا قريبا :صلح الحديبية أو فتح خيبر
( ۲۸ ) ليُظهرَه :لِـيعليَه و يُقوّيَه
( ۲۹ ) سِـماهم :علامَتهُمْ
( ۲۹ ) مَثـلهمْ :وَصْـفهم العجيب
( ۲۹ ) أخرَج شطـْـأه :فِرَاخه المتفرّعة في جوانبه
( ۲۹ ) فآزره :فقوّى ذلك الشّطء الزّرع
( ۲۹ ) فاسْتغلظ :فصار غليظا
( ۲۹ ) فاستوى على سوقه :فاستقام على أصوله و جُذوعِه
