٤۷ - سورة محمد
( ۱ ) أضلّ أعمالهم :أحبطها و أبطلها فلا نفع لها
( ۲ ) كفّر عنهمْ :أزال و مَحَا عنهم
( ۲ ) أصلحَ بالهمْ :حالهمْ و شأنـَـهمْ في الدّين و الدنيا
( ٤ ) فضرْبَ الرقاب :فاضْربوا الرقاب ضَرْبـًـا
( ٤ ) أثخنتموهمْ :أوْسعتموهمْ قتلا و جراحا و أسرا
( ٤ ) فشدّوا الوَثاق :فأحكموا قـيْد الأسارى منهم
( ٤ ) مَنّــًا :بإطلاق الأسرى بغير عِـوَض
( ٤ ) فِدَاءً :بالمال أو بأسارى المسلمين
( ٤ ) حتى تضعَ الحَرْب أوزارها :آلاتها و أثقالها ، و المراد حتى تنقضي الحرب
( ٤ ) لـِـيَبْـلـُــوَ. . :لِـيَخْـتبر . . فيمحّص المؤمنين و يمحق الكافرين
( ٤ ) فلن يضلّ أعمالهمْ :فلن يبطلها بل يوفّـيهم ثوابها
( ۸ ) فتـَـعْسًـا لهمْ :فَهَلاكا . أو عِـثـَارًا أو شقاءً لهم
( ۹ ) فأحْـبَـط أعمالهم :فأبْطلها لكراهتهم القرآن
( ۱۰ ) دمّر الله عليهم :أطبق الهلاك عليهم
( ۱۱ ) مولى . . :وليّ و ناصر . .
( ۱۲ ) مثوًى لهم :موْضع ثواء و إقامة لهم
( ۱۳ ) كأين من قرية :كثيرٌ من القرى
( ۱۵ ) مَثل الجَنة :وصفها – ما تسمعون
( ۱۵ ) غير آسِـن :غير متغيّر و لا منـْتن
( ۱۵ ) عسل مصفّى :مُـنـَـقّى من جميع الشوائب
( ۱۵ ) ماءً حميما :بالغا الغاية في الحرارة
( ۱٦ ) ماذا قال آنفا :ماذا قال الآن ، أو الساعة القريبة
( ۱۸ ) جاء أشراطها :علاماتها و منها مبعثه صلى الله عليه و سلم
( ۱۸ ) فأنى لهم ؟ :فكيف . أو منْ أين لهم ؟
( ۱۸ ) ذِكراهم :تذكّرهم ما ضيعوا من طاعة الله
( ۱۹ ) يعلم متقلبكم :متصرّفكم حيث تتحركون
( ۱۹ ) مثواكم :مقامكم حيث تستقرون
( ۲۰ ) المغشيّ عليه :منْ أصابته الغشْـيَـة و السّـكرة
( ۲۰ ) فأولى لهم :قاربهمْ ما يهلكهم و اللام مزيدةٌ أو العقاب أحقّ و أوْلى لهم
( ۲۱ ) طاعة :خيرٌ لهم أو أمرنا طاعة ٌ
( ۲۱ ) عزم الأمر :جدّ و لزم الجهاد
( ۲۲ ) فهل عسيتمْ :فهل يُتوقّع منكم ؟ (أي يُتوقّع)
( ۲۲ ) توليتم :الحُكم و كنتم ولاة أمر الأمّة
( ۲٤ ) أقفالها :مَغاليقها التي لا تفتح
( ۲۵ ) سوّل لهم :زيّن و سهّـل لهم خطاياهم و منّـاهم
( ۲۵ ) أملي لهم :مدّ لهمْ في الأماني الباطلة
( ۲٦ ) يعلم إسرارهم :إخفاءهم كل قبيح
( ۲۹ ) أضغانـَـهم :أحقادهم الشديدة الكامنة
( ۳۰ ) بسماهمْ :بعلامات نسِمهم بها
( ۳۰ ) في لحن القول :بفحوَى و أسلوب كلامهم الملتوي
( ۳۱ ) لنبلونّـكم :لنختبرنّكم بالتكاليف الشاقة
( ۳۱ ) نبْـلوَ أخباركمْ :نظهرها و نكشفها
( ۳۵ ) فلا تهنوا :فلا تضعفوا عن مقالتة الكفار
( ۳۵ ) السّـلم :الصّـلح و المُـوَادَعَة
( ۳۵ ) يَتركم أعمالكم :ينـْقـصكم أجورَها
( ۳۷ ) فـيُـحْـفكم :يجْهدكم بالطلب كلّ المال
( ۳۷ ) أضغانـَكم :أحقادكم الشديدة على الإسلام
