۸۱ - سورة التكوير
( ۱ ) الشّمس كوّرت ْ
أزيل ضِـياؤها أو لـُـفّـت و طُويَتْ
( ۲ ) النجوم انكدرت
تـَسَاقـَطتْ و تـَـهاوَت
( ۳ ) الجبال سُـيّـرت
أزيلتْ عن مَوَاضعها
( ٤ ) العِشار عُطلت
النـّوق الحَوَامل أهْـمِلـَتْ بلا رَاع
( ۵ ) الوُحوش حُشِرت
جُمِعَـتْ مِنْ كلّ صَوْب
( ٦ ) البحار سُجّـرت
أوقِدَتْ فصَـارتْ نـارًا تـَـضْـطرم
( ۷ ) النـّـفـوس زُوّجَت ْ
قـُرنـَتْ كلّ نـَفـس بـشـَـكْـلِـها
( ۸ ) المَوْءودة
البنـْـت التي تـُـدْفـن حَـيّـة
( ۱۰ ) الصّـحُف نشِرَت
صحف الأعمال فـُرقـَـتْ بين أصحابها
( ۱۱ ) السّماء كُشِطت
قـُـلِعَتْ كما يُـقـْـلع السّـقف
( ۱۲ ) الجَحيم سُعّـرَتْ
أوقِدَتْ و أضْرمَتْ للكفـّـار
( ۱۳ ) الجَـنـّـة أزلفتْ
قـُـرّبَتْ و أدْنِـيَـتْمن المتـّـقين
( ۱٤ ) عَلِمَـتْ نـَـفـْـسٌ ما أحْضَرَتْ
ما عَـمِلـَـتْ منْ خير أو شرّ (جواب إذا)
( ۱۵ ) فلا أقسِم
(أقـْسِم) و"لا" مزيدة ...
( ۱۵ ) بالخُـنـّـس
...بالكواكب السّـيّـارة تخْـنُـس نهارا و تخْـتـفي عن البَصر و هي فوق الأفق ، و تظهر ليلا ثم...
( ۱٦ ) الجَوَار الكنـّس
...تكنس و تستتر في مغيبها تحت الأفق
( ۱۷ ) و الليل إذا عسعس
أقبل ظلامه . أو أدْبر
( ۱۸ ) و الصّـبْح إذا تـنفّـس
أقبل أو أضاء و تـَـبَـلـّج
( ۱۹ ) إنـّـه لقـوْل رَسول
جبريل عن الله ( جواب القسم)
( ۲۰ ) مكين
ذي مكانة رفيعة و شرف
( ۲۳ ) رَآه
رأى الرسول جبريل بصورته الخِـلقـيّـة
( ۲٤ ) الغَيْب
الوَحْي و خبَـر السّماء
( ۲٤ ) بضَـنين
ِببَخيل فـيُـقـصّـر في تبليغه
