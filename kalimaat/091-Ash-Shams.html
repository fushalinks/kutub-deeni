۹۱ - سورة الشمس
( ۱ ) و الشمس
(قسمٌ بها و بما بعدها)
( ۱ ) ضُـحاها
ضَوئها إذا أشرقـَتْ
( ۲ ) تـَـلاها
تـَـبـِعَـها في الإضاءة بَـعْدَ غروبها
( ۳ ) جَـلاّها
أظهَر الشمس للرّائين
( ٤ ) يَـغْـشاها
يُغطّيها حين تغيب فـَـتـُـظلِـم الآفاق
( ۵ ) و ما بناها
و الذي خلقها و هو الله تعالى
( ٦ ) و ما طحاها
و الذي بَـسَـطها و وَطّـأها
( ۷ ) و ما سَوّاها
و الذي عدّل أعضاءها و مَنـَـحَها قـُـوَاها
( ۸ ) فـُـجُـورها و تـَـقـْـواها
مَـعْصِـيَتها و طاعتها و خَـيْرَها و شَرّها
( ۹ ) قدْ أفـلح
فاز بالبغية و ظفَـر(جواب القسم)
( ۹ ) منْ زكّـاها
طهّرها و أنماها بالتـّـقوى
( ۱۰ ) قدْ خاب
خَسر
( ۱۰ ) منْ دسّـاها
نقـّـصها و أخـْـفاها و أخْـمَلها بالفجور
( ۱۱ ) بطغْواها
بسَبَبِ طغيانِها و عُدْوانِها
( ۱۲ ) انبعَثَ أشقاها
قام مُسْرعًا يَـعْـقِـرُ النّـاقة
( ۱۳ ) ناقة الله و سُقياها
احْذروا عـقـْـرَها و نـَـصيـبَها من الماء
( ۱٤ ) فدمدم عليهم
أهْـلـكهمْ و أطبَق العذاب عليهم
( ۱٤ ) فسَوّاها
فجَعَل الدمْدمة عليهم سواءً
( ۱۵ ) عُـقـْـبَـاها
عاقِـبة هذه العقـوبة
