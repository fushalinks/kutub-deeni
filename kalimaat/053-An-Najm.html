۵۳ - سورة النجم
( ۱ ) و النّجم إذا هوَى :(قسَم) بالنّجم إذا غرب و سَقط
( ۲ ) ما ضلّ صاحبكم :ما عدَل الرّسول عن الحقّ و الهُـدَى (جواب القسم)
( ۲ ) ما غوَى :ما اعتقد باطلا قطّ
( ۵ ) شديد القوى :أمين الوحي جبريل عليه السلام
( ٦ ) ذو مرّة :قوة أو خَلْق ٍ حَسَن . أو آثار بديعة
( ٦ ) فاستوى :فاسْتقام على صورته الخِلقـيّة
( ۸ ) دنا :قـَرُبَ جبريل من النبي صلى الله عليه و سلم
( ۹ ) قاب قوسين :قـَـدْرَ قـَوْسَيْن أو ذراعَـيْن من النبي صلى الله عليه و سلم
( ۱۰ ) عبده :عبد الله و هو محمد صلى الله عليه و سلم
( ۱۲ ) أفتمارونه :أفَتـُـكذبونه فتجادلونه صلى الله عليه و سلم
( ۱۳ ) نزلـَة أخرى :مرّة أخرى في صورته الخِـلـْـقـيّة
( ۱٤ ) سدرة المنتهى :التي تنتهي إليها علومُ الخلائق
( ۱۵ ) جنّة المأوى :مُـقام أرواح الشهداء
( ۱٦ ) يغشى السّدرة :يُغَـطّيها و يسْـتـرها
( ۱۷ ) ما زاغ البصر :ما مال بَصَره عمّا أمِرَ برُؤيته
( ۱۷ ) ما طغى :ما جاوزه إلى ما لم يُؤمَر برُؤيته
( ۱۸ ) لقد رآى :ليلة المِعراج
( ۱۹ ) أفرأيْتم :فأخبروني ألهذه الأصنام قدرَة
( ۱۹ ) اللاّت و العزّى :أصنام كانوا يعبدونها في الجاهليّة
( ۲۰ ) و مناة :أصنام كانوا يعبدونها في الجاهليّة
( ۲۲ ) قسمة ضيزى :جائِـرَة . أو عَوْجَاء
( ۲٤ ) أمْ للإنسان ما تمنّى :بل ألـَـهُ كل ما يشتهيه – لاَ
( ۲٦ ) لا تـُـغني شفاعتهم :لاَ تدفَع . أو لا تنفع
( ۳۲ ) الفوَاحش :ما عَظم قـُـبْحه من الذنوب
( ۳۲ ) اللـّـمم :صغائر الذنوب
( ۳۲ ) فلا تزكّوا أنفسكم :فلا تمدحوها بحُسْنِ الأعمال
( ۳٤ ) أكْـدَى :قطعَ عطيّته بخْلاً
( ۳۷ ) الذي وفّى :أتمّ و أكمل ما أمِر به
( ۳۸ ) لا تـزر وازرة . . :لا تحْمِل نفسٌ آثمَة . .
( ٤۲ ) المنتهى :المصير في الآخرة للجزاء
( ٤٦ ) تـُمْنـَى :تدْفق في الرّحم
( ٤۷ ) النّشأة الأخرى :الإحياء بعد الإماتة كما وَعَدَ
( ٤۸ ) أقْـنى :أفقرَ. أو أرْضى بما أعْـطَى
( ٤۹ ) الشِـّــعْـرى :كوكبٌ معروف كانوا يعبدونه في الجاهليّة
( ۵۰ ) عادًا الأولى :قـَـوْم هودٍ عليه السلام
( ۵۱ ) ثمود :قوم صالح عليه السلام
( ۵۳ ) المؤتـفـكة :قُرى قوم لوطٍ عليه السلام
( ۵۳ ) أهْوى :أسْـقـَـطها إلى الأرض بعد رفعها
( ۵٤ ) فغشّاها :ألبَسَها و غطّـاها بأنواع من العذاب
( ۵۵ ) آلاك ربـّــك :نِعَـمِـهِ تعالى و منها دلائل قدرته
( ۵۵ ) تتمارى :تتشكّـك
( ۵۷ ) أزفتِ الآزفة :اقترَبَتْ
( ۵۸ ) كاشفة :نـَفـْـسٌ تكشف أهوالها و شدائدها
( ٦۱ ) أنتم سامِدون :لاهون غَافـلون
