۵۷ - سورة الحديد
( ۱ ) سبَّح لله . . :نزه الله و مجده و دلّ عليه
( ۱ ) العزيز :القادر الغالب على كل شيء
( ۳ ) الأوّل :السابق على جميع الموجودات
( ۳ ) الآخر :الباقي بعد فَـنائِها
( ۳ ) الظاهر :بوجوده و مصنوعاته و تدبيره
( ۳ ) الباطن :بكُنه ذاته عن العقول
( ٤ ) استوى على العرش :استواءٍ يليق بكماله تعالى
( ٤ ) ما يلج :ما يدخل من مطر و غيره
( ٤ ) ما يعرج فيها :ما يصعد إليها من الملائكة و الأعمال
( ٤ ) و هو معكم :بعلمه المحيط بكل شيء
( ٦ ) يولج الليل :يدخله
( ۱۰ ) قبْل الفتح :فتح مكة أو صلح الحديبية
( ۱۰ ) الحسنى :المثوبة الحسنى (الجنّة)
( ۱۱ ) قرْضا حسنا :محتسبا به ، طيبة به نفسه
( ۱۳ ) انظرونا :انتظرونا
( ۱۳ ) نقتبس :نـُـصِـبْ و نأخذ و نستضيء
( ۱۳ ) بسور :حاجز بين الجنة و النار (الأعراف)
( ۱٤ ) ينادونهم :ينادي المنافقون المؤمنين
( ۱٤ ) فتنـْـتم أنفسكم :محنتموها و أهلكتموها بالنفاق
( ۱٤ ) تربّـصتم :انتظرتم بالمؤمنين النوائب
( ۱٤ ) غرتكم الأماني :خدعتكم الأباطيل
( ۱٤ ) الغرور :الشيطان و كل خادع
( ۱۵ ) هي مولاكم :النار أولى بكم . أو ناصركم
( ۱٦ ) ألمْ يأن . . :ألمْ يجيء . .
( ۱٦ ) أن تخشع :وقت أن تخضع و ترق و تلين
( ۱٦ ) الأمد :الأجل أو الزمان
( ۲۰ ) تكاثرٌ . . :مباهاة و تطاول بالعَدد و العُدد
( ۲۰ ) أعْجَب الكفار :راق الزّراع
( ۲۰ ) يهيج :يَيْبس في أقصى غايته
( ۲۰ ) يكون حطاما :فتاتا هشيما متكسّرا بعد يُـبْـسِه
( ۲۱ ) سابقوا :سارعوا مسارعة المتسابقين في المضمار
( ۲۲ ) نبرأها :نـَـخْـلـُـقَ هذه الكائنات
( ۲۳ ) لكيلا تأسَوْا :لكيلا تحزنوا حزن قنوط
( ۲۳ ) لا تفرحوا :فَرَحَ بَطَــٍر و اختيال
( ۲۳ ) مختال فخور :متكبّر مُـبَاهٍ متطاول بما أوتي
( ۲۵ ) الميزان :العَدْل و أمَرْنا به أو الآلة المعروفة
( ۲۵ ) و أنزلنا الحديد :خلقناه . أو هيّـأناه للنّاس
( ۲۵ ) بأس شديد :قوة شديدة
( ۲۷ ) قـفـينا على آثارهم :أتبَعناهم و بعثنا بعدهمْ
( ۲۷ ) الإنجيل :و قد حرّفوه بَعد
( ۲۷ ) الذين اتبعوه :على دينه الذي أرسل به
( ۲۷ ) رأفة و رحمة :مودّة و لينا ، و شفـَقة ً و تعطّـفا
( ۲۷ ) رهبانية :مغالاة في التعبّد و التـّـقـشّـف
( ۲۷ ) ما كتبناها عليهم :ما فرضناها عليهم بل ابتدعوها
( ۲۷ ) فما رعوها :بلْ ضيّعها أخلافهم و كفروا بدين عيسى عليه السلام
( ۲۸ ) يؤتكم كـفــلـيـن :نصِـيبَـين (أجْرَيْن)
( ۲۹ ) لئلاّ يَعلم :ليعلم و "لا" مزيدة
