٦۸ - سورة القلم
( ۱ ) و القلم :( قَسَمٌ) بالقلم الذي يُكتب به
( ۱ ) و ما يسطرون :و الذي يكتبونه بالقلم
( ۲ ) ما أنت :يا محمد (جواب القسم)
( ۳ ) غير ممنون :غير مقطوع عنك
( ٦ ) بأيكم المفتون :في أيّ الفريقين منكم المجنون
( ۹ ) ودّوا لو تدهن :أحبّوا لـَوْ تلايـنُهمْ و تصانِعُهمْ
( ۹ ) فيُدهِنون :فَهُمْ يُلاينونـَـك و يُصانعُونك
( ۱۰ ) حلاف :كثير الحَلِفِ في الحقّ و الباطل
( ۱۰ ) مهين :حقير في الرأي و التمييز أو كَذاب
( ۱۱ ) همّاز :عيّـاب أو مُغتاب للنّـاس
( ۱۱ ) مشّـاء بنميم :بالسِّعَايَة و الإفساد بين الناس
( ۱۳ ) عُـتلّ :فاحِش لئيم ، أة غليظ جافٍ
( ۱۳ ) زنيم :دَعِيّ مُـلصَق بقوْمِه أو شِرّير
( ۱۵ ) أساطير الأولين :أباطيلهم المسطرة في كتبهم
( ۱٦ ) سنسمه على الخرطوم :سَنلحِق به عارًا لا يُـفارقه كالوَسْم على الأنف
( ۱۷ ) بَلوناهم :امْتحنّـا اهل مكّة بالقحْط
( ۱۷ ) الجنّة :بُسْـتان بالقرب من صنعاء
( ۱۷ ) ليَصْرمنها :لـَـيَـقـْـطعنّ ثمارها بعد الإستواء
( ۱۷ ) مصبحين :داخلين في وقت الصّباح
( ۱۸ ) لا يستثنون :حِصّة المساكين مُخالِفـِينَ لأبيهم
( ۱۹ ) فطاف عليها :أحاط نازلا عليها
( ۱۹ ) طائف :بَلاءٌ و عذابٌ (نارٌ مُحْرقة)
( ۲۰ ) كالصريم :كالليل الأسود أو البستان المّصروم
( ۲۱ ) فتنادوْا مصحبين :نادى بَعضهم بَعْضا حين أصبحوا
( ۲۲ ) اغدوا على حرْثِكم :بَـاكِرُوا مُقبلين على ثماركم
( ۲۲ ) صارمين :قاصدين قطعها
( ۲۳ ) يَتخافتون :يَتسارّونَ بالحديث فيما بينهم
( ۲۵ ) غَدوْا :ساروا غُـدوة إلى حرثهم
( ۲۵ ) على حرْد :على انفراد عن المساكين
( ۲۵ ) قادرين :على الصّرام
( ۲٦ ) إنّـا ضالون :الطّريق ، و ما هذه جنّـتـنا
( ۲۸ ) أوْسَطهم :أحسنهم رأيا و أرجحهم عقلا
( ۲۸ ) لولا تسبّحون :هلاّ تستغفرون الله من فعلكم و خبْث نيّـتكم
( ۳۰ ) يَتلاومون :يَلوم بَعضهم بَعضا على قصْدهم
( ۳۲ ) إلى ربـّـنا راغـبون :طالبون منه الخير و العفو
( ۳۸ ) لمَـا يتحيّرون :لـلـّـذي تختارونه و تشتهونه
( ۳۹ ) لكم أيمان علينا :عهود مؤكّدة بالأيْمَان
( ۳۹ ) لما تحْكمون :لـلـّذي تحكمون به لأنفسكمْ
( ٤۰ ) زعـيم :كفيلٌ بأن يكون لهم ذلك
( ٤۲ ) يُكشف عن ساق :كنايَة عن شدّة هوْل يوم القـيامة
( ٤۳ ) خاشعة أبْصارهم :ذليلة مُنكسِرَة
( ٤۳ ) ترْهقهم ذلـّـة :يَغشاهم ذلّ و خسْران و نـَـدامة
( ٤٤ ) فذرني :دَعْـني و خَلـّـني (تهديد شديد)
( ٤٤ ) سنستدرجهم :سَنُدنيهم من العذاب دَرَجَة فدَرجة حتى نوقِعَهم فيه
( ٤۵ ) أمْلِي لهمْ :أمْهِلهمْ ليزدادوا إثما
( ٤٦ ) مَغرَمٍ :غَرامة ذلك الأجر
( ٤٦ ) مُثقلون :مُكلفون حِمْلاً ثقيلاً
( ٤۸ ) كصاحب الحوت :يونس عليه السّـلام
( ٤۸ ) مَكظوم :مَمْـلوءٌ غَـيْظا في قـلبه على قَوْمه
( ٤۹ ) لنـُـبذ بالعراء :لـَـطرح من بَــْطن الحوت بالأرض الفضاء المُهلِكة
( ۵۰ ) فاجتباه رَبّه :فاصْـفاه بعَوْدة الوَحْي إليه
( ۵۱ ) ليـُــزلقـونك :ليُزلـّـونَ قـَدَمَك فيَرمونـَـك
