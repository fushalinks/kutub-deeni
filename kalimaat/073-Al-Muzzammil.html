۷۳ - سورة المزمل
( ۱ ) المزّمل :المتلفّـف بثيابه (النبي صلى الله عليه و سلم)
( ٤ ) رتـّـل القرآن :اقرأه بتمهّل ، و تبْيين حروف
( ۵ ) قوْلاً ثقيلا :شاقّـاعلى المكلـّـفـين
( ٦ ) ناشِئة الليل :العبادة التي تنشأ به و تحدث
( ٦ ) أشدّ وطأ :ثباتا للقدَم و رسوخا في العبادة
( ٦ ) أقوَم قيلا :أثبتُ قِراءة لحضور القلب فيها
( ۷ ) سَبْحًا :تصرّفا و تقـلـّـبا في مُهمّـاتك
( ۸ ) تبتـّـل إليه :انقطع إلى عبادته تعالى ، و استغرق في مراقـبته
( ۱۰ ) هجرًا جميلا :اعتزالا حَسَنا لا جزع فيه
( ۱۱ ) ذرني و المكذبين :دَعْـني و إياهم فسأكفـيكهم
( ۱۱ ) أولي النعمة :أرباب التنعّم ، و غضارة العيش
( ۱۱ ) مهّـلهم قليلا :أمْهلهم زمانا قليلا بعده النّـكال
( ۱۲ ) أنـْـكالا :قـيودا شديدة ثِقالا
( ۱۳ ) طعامًـا ذا غصّة :ذا نُشوب في الحلق فلا يَـنـْـساغ
( ۱٤ ) يوم ترجُف الأرض :تضطرب و تتزلزل (يوم القيامة)
( ۱٤ ) كثيبا مهيلا :رَمْلا مُجْـتمِعًـا – سائلا مُنهالا
( ۱٦ ) أخذا وبيلا :شديدا ثقيلا وَخيم العُقـبى
( ۱۸ ) السماء منفطر به :شيء مُنشقّ في ذلك اليوم لِهوْله
( ۲۰ ) لنْ تـُحصوه :لنْ تـُـطيقـوا ضَبْط وَقتِ قِـيامه
( ۲۰ ) فـتـَـاب عَليكم :بالتـّرخيص في ترك قيامه المقدّر
( ۲۰ ) فاقرءُوا ما تيسر من القرآن :فصلـّـوا ما سَهُـلَ عليكم منْ صلاة اللـّـيل ، و في الصّلاة قرآن
( ۲۰ ) يَضربون :يُـسافـرون للتجارة و نحوها
( ۲۰ ) أقيموا الصّـلاة :المفروضة
( ۲۰ ) قرضا حسنا :احتسابا بطيبة نفـْـس
