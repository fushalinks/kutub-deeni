۹۰ - سورة البلد
( ۱ ) لا أقسِم :(أقسِم) و "لا" مزيدة
( ۱ ) بهذا البلد :بمكّة المكرّمة
( ۲ ) حِلّ بهذا البلد :حلالٌ لك ما تـَـصْـنـَـع به يومئذ
( ۳ ) والد و ما وَلد :آدم و جميع ذريّـته أو الصالحين منهم
( ٤ ) لقد خلقـنا الإنسان :(جواب القسم)
( ٤ ) كبد :نـَـصَـب و مشقـّة و مُكابَدة للشّدائد
( ٦ ) أهلكتُ مالا لـُـبَدًا :كثيرا في المَكْرُمات مباهاةً و تـَـعاظما
( ۱۰ ) هدَيْنـاه النـّـجْدَين :بَـيـَّـنـَّـا له طريقيْ الخيْر و الشرّ
( ۱۱ ) فلا اقـتحم العقبة :فهلاّ جَاهد نفسَـه في أعمال البرّ
( ۱۳ ) فكّ رَقـبَة :تخليصها من الرّق و العبوديّة
( ۱٤ ) ذي مسغبة :مَـجَـاعَة
( ۱۵ ) يتيما ذا مقربة :قرَابَة في الـنّـسَب
( ۱٦ ) مسكينا ذا متربَة :فاقة شديدة لـَـصِـقَ منها بالتراب
( ۱۷ ) بالمَرْحمَـة :بالرّحمة فيما بينهم
( ۱۸ ) أصحاب الميمنة :اليُـمْن . أو ناحية اليمين
( ۱۹ ) أصحاب المشأمة :الشّؤم . أو ناحية الشمال
( ۲۰ ) نارٌ مؤصَدة :مُـطبَقة ٌ مُـغْـلقة أبْوابها
