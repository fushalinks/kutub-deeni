۹۲ - سورة الليل
( ۱ ) و الليل إذا يغشى
يُغطي الأشياء بظلمته (قسم)
( ۲ ) و النهار إذا تجلـّى
ظهر بضوْئه و وَضح
( ٤ ) إنّ سعيكم لشتـّـى
إنّ عَمَـلكمْ لمخْـتلِف في الجزاء (جواب القسم)
( ٦ ) صَدّق بالحسنى
بالمِلـّـة الحُسْنـى و هي الإسلام
( ۷ ) فسنـُـيَـسّره
فـسَـنـُـوفّـقهو نـُـهيـّـئـُـه
( ۷ ) لليُسرى
للخَصـلة المؤَدّية إلى اليُسْر و الرّاحَة
( ۱۰ ) للعُسْرى
للخصلة المؤدية إلى العُسْر و الشدّة
( ۱۱ ) ما يُـغْنى
ما يَدْفع العذاب عنه
( ۱۱ ) ترَدّى
هَـلـَـكَ . أوْ سَـقـَـط في النـّـار
( ۱۲ ) إنّ علينا للهدى
الدّلالة على الحقّ أو بيان طريقه
( ۱٤ ) نارًا تـَـلظى
تتلهّب و تتوقّد
( ۱۵ ) لا يصْلاها
لا يَدْخلها أو لا يُقاسي حَرّها
( ۱۷ ) سَـيُـجَـنّـبُها
سَـيُـبْعَدُ عنها
( ۱۸ ) يتزكّى
يتـطهّر به منَ الذنوب
( ۱۹ ) تـُـجْـزى
تكافَـأ ، نزلت في الصّديق رضي الله عنه
