۵۲ - سورة الطور
( ۱ ) و الطور :(قسَمٌ) بجبَل طور سيناء الذي كلـّـم الله عنده موسى
( ۲ ) و كتاب مسطور :مكتوب على وَجه الانتظام
( ۳ ) في رقّ :ما يُـكْـتـَب فيه جلْدًا أو غيره
( ۳ ) منشور :مبْسوط غير مختوم عليه
( ٤ ) و البيت المعمور :هو الضّراح في السّماء أو الكعبة
( ۵ ) و السّقف المرفوع :السّماء
( ٦ ) و البحر المسجور :الموقَـد نارًا يوْمَ القيامة
( ۷ ) إنّ عذاب . . :(جَوَاب القسَم)
( ۹ ) تمور السماء :تـَـضْـطرب و تـَـدور كالرّحَى
( ۱۱ ) فويْـل :هَلاكٌ أوحَسْـرَة أو شدّة عذاب
( ۱۲ ) خوْض :انـْدفاع في الأباطيل و الأكاذيب
( ۱۳ ) يُدعّون :يُدفعون بـِعُـنـْـفٍ و شدّة
( ۱٦ ) اصلوها :ادخلوها . أو قاسوا حرّها
( ۱۸ ) فاكهين :مُتلذذين نـَـاعمين مَسْرورين
( ۲۰ ) سُرُرٍ مصفوفة :مَوْصولٍ بعضها ببعـض باستواءٍ
( ۲۰ ) زوّجناهم :قرَنّـاهمْ
( ۲۰ ) بحور عين :بـنساءٍ بيض نُـجْـل العيون حِـسَانِـها
( ۲۱ ) ألتناهم :ما نـَـقـَـصنا الآباء بهذا الإلحاق
( ۲۱ ) رَهين :مَرْهونٌ عِـنـْد الله تعالى
( ۲۳ ) يتنازعون :يَتجاذبون و يَتعاورون
( ۲۳ ) كأسًـا :خَمْرًا . أو إناءً فيه خمرٌ
( ۲۳ ) لا لغوٌ فيها و لا تأثيم :لا كلامٌ سَاقط في أثناء شربها و لا فِعْـلٌ يوجب الإثم
( ۲٤ ) لؤلؤ مكْـنونٌ :مَسْـتور مَصونٌ في أصدافه
( ۲٦ ) مشفـقـين :خائفـين من العاقـبة
( ۲۷ ) عذاب السّموم :نار جهنّم النّـافذة في المَـسَامّ
( ۲۸ ) هو البرّ الرحيم :المحسِن العَطوف، العظيم الرحمة
( ۳۰ ) رَيْبَ المَنون :صُرُوف الدّهْـر المُهلكَة
( ۳۲ ) قوْمٌ طاغون :متجاوزون الحَدّ في العِناد
( ۳۳ ) تقوّله :اختلق القرآن منْ تلقاء نفسه
( ۳۷ ) خزائن ربّـك :خزائن رزقه و رَحمته أو مقـْـدوراته
( ۳۷ ) هم المسيطرون :الأرباب الغالبون أو المسلـّـطون
( ۳۸ ) لهم سُلـّـمٌ :مَرقـًـى إلى السماء يَصعدون به
( ٤۰ ) منْ مَغْرَمٍ مثقلون :من التزام غرْمٍ متعبون
( ٤۲ ) هم المكيدون :المَـجْـزيون بكيْدهم و مَكْرهم
( ٤٤ ) كِسْـفـًـا :قطْعَة عَـظيمة
( ٤٤ ) سحابٌ مَرْكوم :مجموعٌ بَعضه على بعض يُمْطِرُنا
( ٤۵ ) فيه يصعقون :يُهلكون (يوم بَدْر)
( ٤٦ ) لا يغني عنهمْ :لا يَدْفع عـنهمْ
( ٤۷ ) عذابًـا دون ذلك :عذابا قبل ذلك هو القحط
( ٤۸ ) بأعيننا :في حِفظنا و حراستنا
( ٤۸ ) سبّحْ بحمد ربـّـك :نزّهْه تعالى حامِدًا له
( ٤۹ ) إدبار النجوم :وَقْت غَـيْـبَتها بضوْءِ الصّباح
