۲۵ - سورة الفرقان
( ۱ ) تبارك الذي . . :تعالى و تمجّد . أو تكاثر خيره . .
( ۱ ) نزل الفرقان :القرآن الفاصل بين الحقّ و الباطل
( ۲ ) فقدّره :فهيّأه لما يصلح له و يليق به
( ۳ ) نشورا :بعثا بعد الموت في الآخرة
( ٤ ) إفك افتراه :كذب إخترعه من عند نفسه
( ٤ ) زورا :كذبا عظيما لا تـُبلَـغ غايته
( ۵ ) أساطير الأوّلين :أكاذيبهم المسطورة في كتبهم
( ۵ ) بكرة و أصيلا :أول النهار و آخره : أي دائما
( ٦ ) يعلم السّر :يعلم كلّما يغيب و يخفى
( ۸ ) جنّة يأكل منها :بستان مثمر يتعيّش منه
( ۸ ) رجلا مسحورا :غلب السّحر على عقله
( ۱۱ ) سعيرا :نارا عظيمة شديدة الإشتعال
( ۱۲ ) تغيّظا :صوت غليان كصوت المتغيّظ
( ۱۲ ) زفيرا :صوتا شديدا كصوت الزّافر
( ۱۳ ) مقرّنين :مقرونه أيديهم إلى أعناقهم بالأغلال
( ۱۳ ) ثبورا :هلاكا فقالوا وَا ثبوراه
( ۱٦ ) وعدا مسئولا :موعودا حقيقا أن يُسأل و يُطلب
( ۱۸ ) نسوا الذكر :غفلوا عن دلائل الوحدانية
( ۱۸ ) قوما بورا :هالكين . أو فاسدين
( ۱۹ ) صرفا :دفعا للعذاب عن أنفسكم
( ۲۰ ) فتنة :ابتلاء و محنة
( ۲۱ ) لا يرجون لقاءنا :لا يأملونه لكفرهم بالبعث
( ۲۱ ) عتوا :تجاوزوا الحدّ في الطّغيان و الظلم
( ۲۲ ) حجرا محجورا :حراما محرّما عليكم البُشرى
( ۲۳ ) هباءً :كالهباء ( ما يُرى في الكُوى مع ضوء الشمس كالغبار)
( ۲۳ ) منثورا :مفرّقا ذاهبا
( ۲٤ ) مقيلا :مكان استرواح و تمتّع ظهيرة
( ۲۵ ) تشقّق السّماء :تتفتح السموات
( ۲۵ ) بالغمام :بالسّحاب الأبيض الرقيق
( ۲۷ ) سبيلا :طريقا إلى الهدى أو إلى النجاة
( ۲۹ ) للإنسان خذولا :كثير الخذلان لمن يواليه
( ۳۰ ) مهجورا :متروكا مهملا
( ۳۲ ) رتـّلناه :فرّقناه آية بعد آية . أو بيّـناه
( ۳۳ ) أحسن تفسيرا :أصدق بيانا و تفصيلا
( ۳٦ ) فدمّرناهم :فأهلكناهم
( ۳۸ ) أصحاب الرس :البئر – قتلوا نبيّهم و دسوه فيها
( ۳۸ ) قرونا :أُمما
( ۳۹ ) تبّرنا تتبيرا :أهلكنا إهلاكا عجيبا
( ٤۰ ) مطر السّوء :حجارة من السماء مهلكة
( ٤۰ ) لا يرجون نشورا :لا يتوقّعون بعثا بل ينكرونه
( ٤۱ ) هزوا :مهزوءً به
( ٤۳ ) أرأيت :أخبرني
( ٤۳ ) وكيلا :حفيظا تمنعُه من عبادة ما يهواه
( ٤۵ ) مدّ الظلّ :بسطه بين الفجر و طلوع الشّمس
( ٤۷ ) الليل لباسا :ساترا لكم بظلامه كاللباس
( ٤۷ ) النوم سُباتا :راحة لأبدانكم ، بقطع أعمالكم
( ٤۷ ) النهار نشورا :إمبعاثا من النّوم للسعي و العمل
( ٤۸ ) الرّياح بشرا :مبشّرات بالرّحمة و هي المطر
( ۵۰ ) صرّفناه بينهم :أنزلنا المطر على أنحاء مختلفة
( ۵۰ ) كفورا :جحودا و كفرانا بالنّعمة
( ۵۳ ) مرج البحرين :أرسلهما في مجاريهما أو أجراهما
( ۵۳ ) عذب فرات :حلو شديد العذوبة
( ۵۳ ) ملح أجاج :شديد الكلوحة و الحرارة أو المرارة
( ۵۳ ) برزخا :حاجزا عظيما يمنع اختلاطهما
( ۵۳ ) حجرا محجورا :حراما محرّما تغيّر صفاتهما
( ۵٤ ) نسبا :ذوي نسب ذكورا يُنسب إليهم
( ۵٤ ) صهرا :ذوات صهر إناثا يُصاهر بهن
( ۵۵ ) على ربّه ظهيرا :مُعينا للشّيطان على ربّه بالشّرك
( ۵۸ ) سبّح :نزّه تعالى عن جميع النقائص
( ۵۸ ) بحمده :مُثنيا عليه بأوصاف الكمال
( ۵۹ ) استوى على العرش :إستواء يليق بكماله تعالى
( ٦۰ ) زادهم نفورا :تباعدا عن الإيمان
( ٦۱ ) تبارك الذي . . :تعالى و تمجّد أو تكاثر خيره
( ٦۱ ) بروجا :منازل للكواكب السّيارة
( ٦۲ ) خِـلفة :يخلُف أحدهما الآخر و يتعاقبان
( ٦۳ ) هونا :بسكينة و وقار و تواضع
( ٦۳ ) قالوا سلاما :قولا سديدا يسلمون به من الأذى
( ٦۵ ) كان غراما :لازما أو ممتدّا ، كلزوم الغريم
( ٦۷ ) لم يقتروا :لم يُضيّقوا تضييق الأشحّاء
( ٦۷ ) قواما :عدلا وسطا بين الطّرفين
( ٦۸ ) يلقى أثاما :عقابا و جزاء في الآخرة
( ۷۲ ) مرّوا باللغو :بما ينبغي أن يُلغى و يُطرح
( ۷۲ ) مرّوا كراما :مُكرمين أنفسهم بالإعراض عنه
( ۷۳ ) لم يخروا :لم يسقطوا و لم يقعوا
( ۷٤ ) قرّة أعين :مسرّة و فرحا
( ۷٤ ) إماما :قدوة و حجّة أو أئمّة
( ۷۵ ) يُجزون الغرفة :أعلى منازل الجنّة و أفضلها
( ۷۷ ) ما يعبأ بكم :ما يكترث و ما يبالي بكم
( ۷۷ ) دعاؤكم :عبادتكم له تعالى
( ۷۷ ) يكون لزاما :يكون جزاء تكذيبكم عذابا دائما مُلازما لكم
( الآية ) الكلمة :التفسير
