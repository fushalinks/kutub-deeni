۹٤ - سورة الشرح
( ۱ ) ألم نشرحْ :ألم نـُـفـسِـحْ بالحكمة و النبوّة – قد أفـْـسَحْـنا
( ۲ ) وَضَعْنا :خَـفـّـفـنا عنك و سَـهّـلنا عـليك
( ۲ ) ِوزرك :حِمْـلك "أعباءَ النبوّة و الرّسالة
( ۳ ) الذي أنقض ظهرك :أثقـله حتـّـى سُـمِعَ له نقـيض "صَوْتٌ
( ۷ ) فإذا فرغت :مِـنْ عِـبَـادة أدّيتها
( ۷ ) فانصبْ :فاجْتهِدْ و أتـْـبعْها بعِبادة أخرى
( ۸ ) فارْغَبْ :فاجْعَـلْ رَغْـبَـتـَك في جَميع شؤونك
