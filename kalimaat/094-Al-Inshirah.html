۹٤ - سورة الشرح
ألم نشرحْ ( ۱ )
ألم نـُـفـسِـحْ بالحكمة و النبوّة – قد أفـْـسَحْـنا
وَضَعْنا ( ۲ )
خَـفـّـفـنا عنك و سَـهّـلنا عـليك
ِوزرك ( ۲ )
حِمْـلك "أعباءَ النبوّة و الرّسالة
الذي أنقض ظهرك ( ۳ )
أثقـله حتـّـى سُـمِعَ له نقـيض "صَوْتٌ
فإذا فرغت ( ۷ )
مِـنْ عِـبَـادة أدّيتها
فانصبْ ( ۷ )
فاجْتهِدْ و أتـْـبعْها بعِبادة أخرى
فارْغَبْ ( ۸ )
فاجْعَـلْ رَغْـبَـتـَك في جَميع شؤونك
