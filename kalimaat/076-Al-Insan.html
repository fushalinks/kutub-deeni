۷٦ - سورة الانسان
( ۲ ) أمشاج :أخلاط ممتزجة مُتباينـَـة الصّـفات
( ۲ ) نبتليه :مُتبَلين له بالتّكاليف فيما بعد
( ۳ ) هديناه السّبيل :بيّـنا له طريق الهداية و الضّلال
( ٤ ) سلاسل :بها يُقادون و في النّار يُسحبون
( ٤ ) أغلالا :بها تـُـجْمَع أيديهم إلى أعناقهم و يُقـيّدون
( ۵ ) كأس :خَمْر أو زُجاجة فيها خَمر
( ۵ ) مزاجُها :ما تـُمـْـزَج الكأس به و تـُخْـلـَط
( ۵ ) كافورا :ماءً كالكافور في أحسن أوصافه
( ٦ ) عيْنا :ماءَ عَـيْن أو خَمْر عيْن
( ٦ ) يَشرب بها :يَشرب منها . أو يَرتوي بها
( ٦ ) يُـفجّرونها :يُجْرونها حيث شاءوا من منازلهم
( ۷ ) مستطِرا :فاشِيًـا مُنتشِرًا غاية الإنتشار
( ۱۰ ) يوما عبوسا :تـَـكْـلـَحُ فيه الوُجُوه لِهَوْله
( ۱۰ ) قمطريرا :شديد العُبوس
( ۱۱ ) لقـّـاهم نضرة :أعطاهُم حُسْـنـًا و بَهْجَة في الوُجوه
( ۱۳ ) الأرائك :السّرُر في الحِجال(جمع حِجلة مُحرّكة- بيت يُزيّن بالقـِباب و الأسِرّة و السّـتور)
( ۱۳ ) زمهريرا :بَرْدًا شديدا . أو قَمَرًا
( ۱٤ ) دانِية عليهم ظلالها :قريبة منهم ظلال أشجارها
( ۱٤ ) ذلـّلتْ قطوفها :قُرّبت ثِمارُها لمتناولِها
( ۱۵ ) أكواب :أقداح بلا عرى و خراطيم
( ۱۵ ) قوارير :كالزّجاجات في الصّـفاء
( ۱٦ ) قدّروها :جَعَلوا شرابَها على قدْر الرِّيّ
( ۱۷ ) كأسًا :خَمْرًا أو زجاجة فيها خَمْر
( ۱۷ ) مزاجُها :ما تـُـمزَج به و تـُخْـلط
( ۱۷ ) زنجبيلا :ماءً كالزنجبيل في أحْسَن أوصافه
( ۱۸ ) تُسَمّى سلسبيلا :يُوصَف شرابُها بالسّلاسة في الإنسياغ
( ۱۹ ) ولدان مُخَلـّدون :مُـبَقـَّـوْن على هيْئة الولدان في البَهاء
( ۱۹ ) لؤلؤا منثورا :اللؤلؤ المُـفـرّق في الحُـسْن و الصّـفاء
( ۲۱ ) ثياب سندس :ثِـيَاب من ديباج رقيق
( ۲۱ ) إستبرق :ديباج غليظ
( ۲۵ ) بكرة و أصيلا :أوّل النّهار و آخِره . أو دائِما
( ۲۷ ) يوما ثقيلا :شديد الأهوال (يوم القيامة )
( ۲۸ ) شددنا أسرهم :أحْكَمْـنا خلقهم
