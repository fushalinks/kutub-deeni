۳۷ - سورة الصافات
( ۱ ) و الصّـافات صفّا :قـَسَمبالجماعات تصطفّ للعبادة
( ۲ ) فالزّاجرات زجرا :تزْجر عن المعاصي بالأقوال و الأفعال
( ۳ ) فالتـّـاليات ذِكرا :تتلوا آيات الله للعلم و التّعليم
( ٤ ) إنّ إلهكم لواحد :جواب القسم
( ۷ ) شيطان مارد :متمرّد خارج عن الطاعة
( ۷ ) على آثارهم يُهرعون :يُزعجون و يُحثّون على الإسراع الشّديد على أثارهم
( ۸ ) يقذفون :يُرْجمون
( ۹ ) دحورا :إبعادا و طردا
( ۹ ) عذابٌ واصبٌ :دائمٌ لا ينـْـقطع
( ۱۰ ) خَطِـف الخطفة :اختلس الكلمة مسارقة بسرعة
( ۱۰ ) شهاب :ما يرى كالكوكب منقضّا من السّماء
( ۱۰ ) ثاقبٌ :مُضيءٌ . أو مُـحْـرقٌ
( ۱۱ ) طين لازب :ملتزق بَعْـضه ببَعض
( ۱۲ ) و يسخرون :و هم يَهزءون بتعَجّبكَ
( ۱٤ ) بستسخِرون :يُبالِغونَ في سُخْريَتهمْ
( ۱۸ ) أنتم داخرون :صاغِـرون أذلاّء
( ۱۹ ) زجْرَة واحدة :صيحة واحدة " نـَفخة البَعث
( ۲۰ ) ياويْـلنا :يا هلاكنا احضر
( ۲۰ ) يوم الدّين :يوم الجَزاء و الحساب
( ۲۲ ) أزواجَهم :أشباههم . أو قُرناءهمْ
( ۲٤ ) قِـفوهم :احبسوهم في موقف الحساب
( ۲۸ ) عن اليمين :من جهة الدّين فتصدّوننا عنه
( ۳۰ ) قوما طاغين :مُجاوزين الحدّ في العصيان
( ۳۱ ) فحقّ علينا :ثبت ووجب علينا
( ۳۲ ) فأغويناكم :فدعوناكم إلى الغيّ فاستجبتم
( ٤۰ ) المخلصين :الذين أخلصهم الله لطاعته
( ٤۵ ) بكأسٍ :بخمر . أو بقدح فيه خمر
( ٤۵ ) مِنْ مَعين :من شراب نابع من العيون
( ٤۷ ) لا فيها غوْلٌ :ليس فيها ضرر ما كخمر الدنيا
( ٤۷ ) عنها يُنزفون :بسببها يسكرون و تُنزع عقولهم
( ٤۸ ) قاصرات الطّرف :حور لا ينظرن إلى غير أزواجهنّ
( ٤۸ ) عِـينٌ :نُـجْـلُ العيون حِسانها
( ٤۹ ) بيْضٌ مكنونٌ :مصون مستور لم يُصبه غبار
( ۵۳ ) لَمَدينون :لمجزيون و محاسبون ؟
( ۵۵ ) سَوَاء الجحيم :وسطها
( ۵٦ ) إنْ كِدْت لترْدين :إنّك قاربت لتهلكني بالإغواء
( ۵۷ ) المُحضرين :للعذاب مثلك
( ٦۲ ) خيرٌ نُزُلا :ضيافة و تكرمة و لذة
( ٦۲ ) شجرة الزّقوم :شجرة من أخبث الشّجر بتهامة
( ٦۳ ) فتنة للظّالمين :مَحنة و عذابا لهم في الآخرة
( ٦٤ ) أصْـل الجحيم :قعر جهنّم
( ٦۵ ) طلـْعها :ثمرها الشبيه بطلع النّخل
( ٦۵ ) كأنّه رءوس الشياطين :تمثيل لتناهيه في البشاعة و القبح
( ٦۷ ) لشَوبا :لخَـلـْـطاً و مزاجا
( ٦۷ ) منْ حميم :ماءٍ بالغ غاية الحرارة
( ۸۳ ) منْ شيعَتِه :ممّن شايعه على منهاجه و ملّـته
( ۸٦ ) أإفْـكا ؟ :أكذبا و باطلا ؟
( ۸۸ ) فَـنـَظر :تأمَـل تأمُـلَ الكاملين
( ۸۹ ) إنّي سقيمٌ :يُريد أنه سقيم القلب لكفرهم
( ۹۱ ) فرَاغ إلى آلهتهم :فمال إليها خفية ليُـحطّمها
( ۹۳ ) ضربا باليمين :يضربهم ضربا مُلتبسا بالقوّة
( ۹٤ ) يزفّون :يسرعون في مشيهم
( ۱۰۱ ) بغلام حليم :رجّح كثيرٌ أنّه إسماعيل عليه السلام
( ۱۰۲ ) بَلغَ مَعَه السّـعْي :درجة العمل معه في حوائجه
( ۱۰۳ ) أسلما :استسلما و انقادا لأمره تعالى
( ۱۰۳ ) تلّه للجبين :أضجعه على جبينه على الأرض
( ۱۰٦ ) البلاء المبين :الإختبار البيّن. أو المحنة البيّـنة
( ۱۰۷ ) بذِبْح :بكبشٍ يُذبح
( ۱۲۵ ) أتدْعون بَعْلا :أتعبدون الصّـنم المسمّى بعلا
( ۱۲۷ ) لمُحضرون :تُحضرهم الزّبانية في النّار
( ۱۳۰ ) إلياسين :إلياس . أو إلياس و أتباعه
( ۱۳۵ ) في الغابرين :في الباقين في العذاب
( ۱۳٦ ) دمّرنا الآخرين :أهلكناهم
( ۱۳۷ ) مُصبحين :داخلين في وقت الصّباح
( ۱٤۰ ) أبق :هَرَبَ
( ۱٤۰ ) المشحون :المملوء
( ۱٤۱ ) فَساهم :فقارع من في الفلك
( ۱٤۱ ) المدْحضين :المغلوبين بالقرعة
( ۱٤۲ ) فالتـَـقـَـمَه الحوت :ابتلعه
( ۱٤۲ ) هو مُـليم :آتٍ بما يُلام عليه
( ۱٤۳ ) المسبّحين :الذاكرين الله كثيرا بالتسبيح
( ۱٤۵ ) فنبذناه بالعراء :طرحناه بالأرض الفضاء الواسعة
( ۱٤٦ ) يقطين :هو القرع المعروف و قيل غيره
( ۱۵۱ ) إفكهمْ :كذبهم على الله
( ۱۵۳ ) أصطفَى؟ :أختار ؟ (استفهام توبيخ)
( ۱۵٦ ) سُلطان ٌ :حجّة و برهان
( ۱۵۷ ) الجـِـنّة :الملائكة . أو الشّياطين
( ۱۵۸ ) إنّهم لمُحضرون :إنّ الكفار لمحضرون للنّار
( ۱٦۲ ) عليه بفاتنين :بمضلّين أو مفسدين على الله أحدا
( ۱٦۳ ) صَال الجحيم :داخلها أو مقاسٍ حرّها
( ۱٦۵ ) الصّـافون :أنفسنا في مقام العبادة
( ۱٦٦ ) المسبّحون :المنزّهون الله تعالى عمّا لا يليق بجلاله
( ۱۷۷ ) بساحَتهمْ :بفنائهم . و المراد : بهم
( ۱۸۰ ) ربّ العزّة :الغلبة و القدرة و البطش
