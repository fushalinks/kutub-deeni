۳۸ - سورة ص
( ۱ ) و القرآن :(قسم) جوابه ما الأمر كما تزعمون
( ۱ ) ذي الذكر :ذي البيان لما يُحتاج إليه في الدين
( ۲ ) عزة :حميّـة و تكبر عن الحقّ
( ۲ ) شقاق :مشاقّة و مخالفة لله و لرسوله
( ۳ ) كم أهلكنا :كثيرة أهلكنا
( ۳ ) قرن :أمة
( ۳ ) فنادوْا :استغاثوا حين عاينوا العذاب
( ۳ ) لات حين مناص :ليس الوقت وقت فرار و خلاص
( ۵ ) عجاب :بالغ الغاية في العَجَب
( ٦ ) الملأ منهم :الوجوه من كفار قريش
( ٦ ) امشوا :سيروا على طريقتكم و دينكم
( ۷ ) الملّة الآخرة :دين قريش الذي هم عليه
( ۷ ) اختلاق :كذب و افتراء منه
( ۱۰ ) الأسباب :المعارج إلى السماء
( ۱۱ ) جندٌ ما :هم مجتمع حقير و "ما" زائدة
( ۱۱ ) هنالك :بمكّة يوم الفتح أو يوم بَدْر
( ۱۲ ) ذو الأوتاد :الجنود أو المباني القويتين
( ۱۳ ) أصحاب الأيكة :سكان الغيضة الكثيفة الملتفّة الشجر (قوم شعيب)
( ۱۵ ) ما ينظر :ما ينتظر
( ۱۵ ) صيحة واحدة :نفخة البعث
( ۱۵ ) ما لها من فواق :مالها توقـّـف قدْر فواق ناقة ، و هو ما بين حلبتيها
( ۱٦ ) قطّـنا :نصيبنا من العذاب الذي أوعدته
( ۱۷ ) ذا الأيد :ذا القوّة في الدين و العبادة
( ۱۷ ) إنه أوّاب :رجّاع إلى الله تعالى و طاعته
( ۱۸ ) بالعشيّ و الإشراق :من الزّوال للغروب ، ووقت الضّحى
( ۲۰ ) شددنا ملكه :قوّيناه بأسباب القوّة كلّها
( ۲۰ ) آتيناه الحكمة :النبوّة و كمال العلم و اتقان العمل
( ۲۰ ) فصل الخطاب :علم فصل الخصومات
( ۲۱ ) الخصم :ملكين في صورة إنسانين
( ۲۱ ) تسوّروا المحراب :علوْ سور مصلاّه و نزلوا إليه
( ۲۲ ) بغى بعضنا :تعدّى و ظلم و جار
( ۲۲ ) لا تشطط :لا تجر في حكمك
( ۲۲ ) سواء الصّراط :وسط الطريق و هو عين الحقّ
( ۲۳ ) اكفلنيها :انزلْ لي عنها حتى أكفلها
( ۲۳ ) عزني في الخطاب :غلبني و قهرني في المُحاجّـة
( ۲٤ ) الخلطاء :الشركاء
( ۲٤ ) فتنّـاه :ابتليناه و امتحنّاه
( ۲٤ ) خرّ راكعا :ساجدا لله تعالى
( ۲٤ ) أناب :رجع إلى بالتـّـوبة
( ۲۵ ) لَزلفى :لقـُـربَة و مكانة
( ۲۵ ) حسن مآب :حسن مرجع في الآخرة (الجنّة)
( ۲۷ ) باطلا :لَعِـبًا و عبَثا
( ۲۷ ) فويل :هلاكٌ . أو وادٍ في جهنم
( ۳۰ ) إنه أواب :رجّاعٌ إليه تعالى بالتوبة
( ۳۱ ) بالعشي :ما بعد الزوال إلى الغروب
( ۳۱ ) الصافنات :الخيول الواقفة على ثلاث قوائم و طرف حافر الرابعة
( ۳۱ ) الجياد :السّراع السوابق في العدو
( ۳۲ ) أحببت حب الخير :آثرتُ حبّ الخيل
( ۳۲ ) عن ذكر ربي :على صلاتي العصر لله تعالى
( ۳۲ ) توارت بالحجاب :غرَبَت الشمس . أو غابت الخيل عن بصره لظلمة الليل
( ۳۳ ) ردّوها علي :رُدّوا الخيل عليّ
( ۳۳ ) فطفق مسحا بالسّوق و الأعناق :فَشرعَ يقطع سُوقها و أعْـنـَـاقها بالسّيف قربانا لله تعالى و كانَ ذلك مشروعا في ملّـتِه
( ۳٤ ) فتنّـا سليمان :ابْـتليْناهُ و امتحنّـاه و عاقبْناه
( ۳٤ ) جسدا :شِقّ إنسان ولِـد له
( ۳٤ ) أناب :رَجَع إلى الله تعالى بالتـّـوبة
( ۳٦ ) رخاءً حيث أصاب :لـَــيّـنة . أو مُنقادة حيث أرَاد
( ۳۷ ) غوّاص :في البحر لاستخْراج نفائسهِ
( ۳۸ ) الأصفاد :الأغلال تجمع الأيدي إلى الأعناق
( ۳۹ ) بغير حساب :غير مُحاسَبعلى شيء من الأمْـرَين
( ٤۰ ) لزلفى :لقـُرْبا و كَرَامة
( ٤۰ ) حسن مآب :حُسْن مَرْجع في الآخرة
( ٤۱ ) بنصْب و عذاب :بتعب و مشقـّـة ، و ألـَـم و ضرّ
( ٤۲ ) اركض برجلك :اضربْ بها الأرض
( ٤۲ ) هذا مغتسل :ماءٌ تغتسل به ، فيه شفاؤك
( ٤٤ ) ضِغثا :قبضة من قضْبان أو عثكال النّخل بشماريخه
( ٤۵ ) أولي الأيدي :أصحاب القوّة في الطّاعة
( ٤۵ ) و الأبصار :و البصائر في الدّين و العلْم
( ٤٦ ) أخلصناهم بخالصة :خَصصناهم بخصْـلـَـة لا شَوْب فيها
( ٤۹ ) هذا ذكر :المذكور من محاسنهم شَرَفٌ لهُم
( ۵۲ ) قاصرات الطرف :حُورٌ لا ينظرْن إلى غير أزواجهنّ
( ۵۲ ) أتراب :مستوياتٌ في الشّباب
( ۵٤ ) نفاد :انقطاع و فَـناءٍ
( ۵۵ ) لشرّ مآب :لأسْوَأ مُنقلبٍ و مَصير
( ۵٦ ) جهنّم يصلونها :يَدْخلونـَـها أو يقاسونَ حَرّها
( ۵٦ ) فبئس المهاد :فبئس الفِـراش ، أي المستقرّ جهنّم
( ۵۷ ) حميم :ماءٌ بالغ نهاية الحرارة
( ۵۷ ) غسّاق :صَديدٌ يسيل من أجسامِهم
( ۵۸ ) و آخر :و عذابٌ آخر
( ۵۸ ) من شكله أزواج :مِنْ مثلِه أصْنافٌ في الفظاعَة
( ۵۹ ) هذا فوج :جَمعٌ كَثيف مِنْ أتباعِكم الضّـالين
( ۵۹ ) مقتحم معكم :داخلٌ مَعَكم النار قـَهْرًا عنه
( ۵۹ ) لا مرحبا بهم :لا رَحُـبَـتْ بهم النار و لا اتـّـسعتْ
( ۵۹ ) صلوا النّار :داخِلوها . أو مُقـَاسو حرّها
( ٦۰ ) فبئس القرار :فبئسَ المَقرّ للجميع جَهنّم
( ٦۳ ) أتـّـخذناهم سخريّـا ؟ :مَهزوءًا بهم في الدنيا فأخطأنا ؟؟
( ٦۳ ) زاغت عنهم الأبصار :مَـالتْ عنهمْ فلم نعلم مكَانـَـهم
( ٦۹ ) بالملأ الأعلى :المَلائكة
( ٦۹ ) إذ يختصمون :في شأن آدم و خَلقِهِ و خِـلافته
( ۷۲ ) سوّيته :أتمَمْتَ خَلقـَـه بالصّورة الإنسانية
( ۷۲ ) ساجدين :تحيّة له و تكريمًا
( ۷۵ ) العالين :المستحقـّـين للعلُوّ و الرّفعَة – كَلاّ
( ۷۷ ) رجيم :مطرودٌ من كلّ خيْرٍ ة كَرَامة
( ۷۹ ) فأنظرني :أمهلني و لا تمـتـْـني
( ۸۱ ) يوم الوقت المعلوم :وَقـْـتِ النّـفخة الأولى
( ۸۲ ) فبعزتك :فبسُـلطانك و قـَـهْرك (قـَـسَم)
( ۸۲ ) لأغوينّهم :لأضلـّـنـّـهمْ بتزْيين المعاصي لهم
( ۸٦ ) المتكلّفين :المـتـَـصَـنّعِـين المُـتقـَـوّلينَ على الله
( ۸۸ ) نبأه :صدق أخباره
