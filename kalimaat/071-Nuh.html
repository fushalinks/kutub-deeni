۷۱ - سورة نوح
( ٤ ) إنّ أجل الله :وَقـْـتَ مَجيء عذابه إنْ لم تُؤمِنوا
( ٦ ) فِرارًا :تـَـبَاعُـدًا و نِفـَـارًا عن الإيمان
( ۷ ) استـَـغْـشَوْا ثِيابهم :بالغُوا في التـّـغَطّـى بها كَرَاهة لي
( ۷ ) أصرّوا :تشدّدوا و انـْـهَمَـكُوا في الكُـفـر
( ۱۱ ) يُرْسِل السّماء :المطر الذي في السّحاب
( ۱۱ ) مِدْرارًا :غزيرًا مُـتـتابعًا
( ۱۳ ) لا تـَـرْجونَ لله وَقارًا :لا تعتقِـدون أو لا تخافون عَظمَة الله
( ۱٤ ) خلقـَـكمْ أطوارا :مُدَرّجًا لكم في حالاتٍ مُختلِفة
( ۱۵ ) سموات طِباقا :كلّ سَمَـاءٍ مُقـْـبيَّة على الأخْرى
( ۱٦ ) نورًا :مُنـَـوّرًا لِوَجْهِ الأرض في الظلام
( ۱٦ ) الشّمس سِرَاجًـا :مِصْباحًـا مُضيئا يمحُو الظلام
( ۱۷ ) أنـْـبَـتـَـكم من الأرض :أنـْـشأكم من طينتها
( ۱۹ ) الأرض بساطًا :فِـراشا مبسوطا للاستقـرار عليها
( ۲۰ ) سُبُلاً فِجَاجًـا :طرُقـًـا وَاسِعات
( ۲۱ ) خَسَـارًا :ضلالا في الدنيا و عِقابًـا في الآخرة
( ۲۲ ) مَكْرًا كُـبّـَارًا :بَالِغَ الغَـايةِ في الكِـبَر
( ۲۳ ) وَدًّا :أصنامٌ عَبَدوها ثمّ انتقلت إلى العَرَب .، فكان وَدّ لِكـَــلـْـب
( ۲۳ ) سُوَاعًـا :و سُوَاعٌ لِهُـذيْـل
( ۲۳ ) يَغوثَ :وَ يَغُوث لِغَطفان
( ۲۳ ) يَعُوق :و يَعُوق لِهَمَذان
( ۲۳ ) نـَـسْرَا :و نـَـسْـرٌ لآلِ ذي الكَلاع مِنْ حِمْير
( ۲۵ ) ممّا خطيئاتِهم :من أجل ذنوبهم و "ما" زائدة
( ۲٦ ) ديّـارًا :أحدًا يَدور ة يَتحرّك في الأرض
( ۲۸ ) تبارًا :هَلاكًا وَ دَمارًا
