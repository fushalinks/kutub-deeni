٤٦ - سورة الأحقاف
( ۳ ) أجل مسمّى :بتقدير أجل مسمّى و هو يوم القيامة
( ٤ ) أرأيتم :أخبروني
( ٤ ) لهم شرك :شركة و نـَـصيبٌ مع الله تعالى
( ٤ ) أثارةٍ من علم :بَقيّة من علم عندكم
( ۸ ) تفيضون فيه :تندفعون فيه طعنا و تكذيبا
( ۹ ) بدْعًا :بديعا منفردًا فيما جئتُ به
( ۱۰ ) أرأيتم :أخبروني ماذا حالكم
( ۱۱ ) إفكٌ قديم :كذبٌ متقادمٌ
( ۱۵ ) وصّينا الإنسان :أمرناه و ألزمناه
( ۱۵ ) كُرْها :ذات كرْهٍ و مشقة
( ۱۵ ) حمله و فِصاله :مدّة حمله و فطامه من الرّضاع
( ۱۵ ) بَلغ أشدّه :بَلـَغ َ كمال قوته و عقله
( ۱۵ ) ربّي أوزعني :ألهمني ووفقني و رغّبني
( ۱۷ ) أفّ لكما :كلمة تضجّرٍ و تبرّمٍ و كراهيّة
( ۱۷ ) أن أخرَج :أُبْعَثَ من القبر بعد الموت
( ۱۷ ) خلـَـتْ القرون :مضتِ الأمم و لم تبعَث
( ۱۷ ) ويلكَ :هلكتَ و المراد حثـّه على الإيمان
( ۱۷ ) آمنْ :صدّقْ بالله و بالبعْث
( ۱۷ ) أساطير الأوّلين :أباطيلهم المسطّرة في كتبهم
( ۱۸ ) حقّ عليهم القول :وَجَبَ عليهم وعيد العذاب
( ۱۸ ) قد خَلـَتْ :مَضتْ و تقدّمت
( ۲۰ ) عذاب الهون :الهوان و الذلّ
( ۲۱ ) أخا عاد :هودا عليه السّلام
( ۲۱ ) بالأحقاف :وادٍ بين عُمان و أرض مَهْرَة
( ۲۲ ) لتأفكنا :لتصرفنا . أو لتزيلنا بالإفك
( ۲٤ ) عارضا :سحابا يعرض في الأفق
( ۲۵ ) تـُدمّر :تـُهلِك
( ۲٦ ) مكّـنّاهم :أقدرناهم و بسطنا لهم
( ۲٦ ) فيما إن مكّـنّـاكم فيه :في الذي ما مكّـنّـاكم فيه
( ۲٦ ) فما أغنى عنهم :فما دفع عنهم
( ۲٦ ) حاق بهم :أحاط أو نزل بهم
( ۲۷ ) صرّفنا الآيات :كررناها بأساليب مُختلفة
( ۲۸ ) قربانـًا آلهة ً :متقرّبـًـا بهم إلى الله
( ۲۸ ) إفكُهُمْ :أثر كذبهم في اتخاذها آلهة
( ۲۸ ) يفترون :يختلقونه في قوْلهم إنها آلهة
( ۲۹ ) صرفنا إليك :أمَلْـنا ووجهنا نَحْوَك
( ۲۹ ) أنصتوا :اسكتوا و اصغوا لنسمعه
( ۲۹ ) قضِيَ :أتمّ و فُــِرغ من قراءة القرآن
( ۳۲ ) فليس بمعجزٍ :لله فائت منه بالهَرَب
( ۳۳ ) لم يعيَ بخلقِهنّ :لم يتعب به أو لم يعجز عنه
( ۳۳ ) بلى :هو قادر على إحياء الموتى
( ۳۵ ) أولوا العزم :ذوو الجـِــدّ و الثبات و الصّبر
( ۳۵ ) بلاغ :هذا تبليغ من رسولنا
