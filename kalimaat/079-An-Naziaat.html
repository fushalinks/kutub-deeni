۷۹ - سورة النازعات
و النازعات ( ۱ )
(أقسم) الله بالملائكة تنزع أرواح الكفار من أقاصي أجسامهم
غرْقا ( ۱ )
نـَزعا شديدا مُؤْلما بالغ الغاية
و الناشطات نشطا ( ۲ )
الملائكة تـَـسُـلّ أرواح المؤمنين برفق
و السابحات سبْـحا ( ۳ )
الملائكة تنزل مسرعة لما أمِرتْ به
فالسابقات سبقا ( ٤ )
الملائكة تسبق بالأرواح إلى مستقرّها نارًا أو جَـنّـة
فالمدبرات أمرا ( ۵ )
الملائكة تنزل بالتدبير المأمور به
يوم ترجف الراجفة ( ٦ )
لتبعثنّ (جواب القسم) يَوم تضطرب الأجْرَام بالصيحة الهائلة (نفخة الموت)
تتبعها الرادفة ( ۷ )
نفخة البعث التي تردف الأولى
واجفة ( ۸ )
مُضطربة . أو خائفة وَجلة
أبصارها خاشعة ( ۹ )
ذليلة مُنكسِرة من الفزَع
في الحافرة ( ۱۰ )
إلى الحالة الأولى (الحياة)
كنا عظاما نخِرة ( ۱۱ )
باليَة مُـتـفـتـّـتة
كرّة خاسِرة ( ۱۲ )
رَجْعة غابـِـنة
زجْرة واحدة ( ۱۳ )
صيحة واحدة (نفخة البَعْث)
همْ بالساهرة ( ۱٤ )
هم أحياءٌ على وجه الأرض
طوى ( ۱٦ )
اسم الوادى المقـدّس
طغى ( ۱۷ )
عـتا و تجبّر و كَـفـَر و الطغيان
تـزكى ( ۱۸ )
تطهّر من الكفر و الطغيان
الآية الكبرى ( ۲۰ )
معجزة العصا و اليد البيضاء
يَسعى ( ۲۲ )
يَــجـِدّ في الإفساد و المعارضة
فحَشر ( ۲۳ )
جَمَعَ السّـحَرَة . أو الجُـنـْـد
نكال . . ( ۲۵ )
عُقوبة . أو بعقوبَة . .
رفع سمكها ( ۲۸ )
جعل ثِـخَـنـَـها مُرْتـَـفِعًـا جهة العلوّ
فسوّاها ( ۲۸ )
فجَعَلها مُسْـتـَـوية الخلـْـق بلا عَيْب
أغطش ليلها ( ۲۹ )
أظلمه
أخرج ضحاها ( ۲۹ )
أبْرز نهارها المضيء بالشمس
دحاها ( ۳۰ )
بَسَطها و أوْسَعها لسُكنى أهلها
مَرْعاها ( ۳۱ )
أقواتَ الناس و الدّواب
الجبال أرساها ( ۳۲ )
أثبَتها في الأرض ، كالأوتاد
الطامة الكبرى ( ۳٤ )
الدّاهـية العظمى (القيامة)
بُرزت الجحيم ( ۳٦ )
أظْهِرَتْ إظهارًا بَيّـنـًـا
هي المأوى ( ۳۹ )
هي المرْجع و المُقام له لا غيرها
أيان مرساها؟ ( ٤۲ )
متى يقيمها الله و يُثبتها
