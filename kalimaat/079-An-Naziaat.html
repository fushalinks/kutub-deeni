۷۹ - سورة النازعات
( ۱ ) و النازعات :(أقسم) الله بالملائكة تنزع أرواح الكفار من أقاصي أجسامهم
( ۱ ) غرْقا :نـَزعا شديدا مُؤْلما بالغ الغاية
( ۲ ) و الناشطات نشطا :الملائكة تـَـسُـلّ أرواح المؤمنين برفق
( ۳ ) و السابحات سبْـحا :الملائكة تنزل مسرعة لما أمِرتْ به
( ٤ ) فالسابقات سبقا :الملائكة تسبق بالأرواح إلى مستقرّها نارًا أو جَـنّـة
( ۵ ) فالمدبرات أمرا :الملائكة تنزل بالتدبير المأمور به
( ٦ ) يوم ترجف الراجفة :لتبعثنّ (جواب القسم) يَوم تضطرب الأجْرَام بالصيحة الهائلة (نفخة الموت)
( ۷ ) تتبعها الرادفة :نفخة البعث التي تردف الأولى
( ۸ ) واجفة :مُضطربة . أو خائفة وَجلة
( ۹ ) أبصارها خاشعة :ذليلة مُنكسِرة من الفزَع
( ۱۰ ) في الحافرة :إلى الحالة الأولى (الحياة)
( ۱۱ ) كنا عظاما نخِرة :باليَة مُـتـفـتـّـتة
( ۱۲ ) كرّة خاسِرة :رَجْعة غابـِـنة
( ۱۳ ) زجْرة واحدة :صيحة واحدة (نفخة البَعْث)
( ۱٤ ) همْ بالساهرة :هم أحياءٌ على وجه الأرض
( ۱٦ ) طوى :اسم الوادى المقـدّس
( ۱۷ ) طغى :عـتا و تجبّر و كَـفـَر و الطغيان
( ۱۸ ) تـزكى :تطهّر من الكفر و الطغيان
( ۲۰ ) الآية الكبرى :معجزة العصا و اليد البيضاء
( ۲۲ ) يَسعى :يَــجـِدّ في الإفساد و المعارضة
( ۲۳ ) فحَشر :جَمَعَ السّـحَرَة . أو الجُـنـْـد
( ۲۵ ) نكال . . :عُقوبة . أو بعقوبَة . .
( ۲۸ ) رفع سمكها :جعل ثِـخَـنـَـها مُرْتـَـفِعًـا جهة العلوّ
( ۲۸ ) فسوّاها :فجَعَلها مُسْـتـَـوية الخلـْـق بلا عَيْب
( ۲۹ ) أغطش ليلها :أظلمه
( ۲۹ ) أخرج ضحاها :أبْرز نهارها المضيء بالشمس
( ۳۰ ) دحاها :بَسَطها و أوْسَعها لسُكنى أهلها
( ۳۱ ) مَرْعاها :أقواتَ الناس و الدّواب
( ۳۲ ) الجبال أرساها :أثبَتها في الأرض ، كالأوتاد
( ۳٤ ) الطامة الكبرى :الدّاهـية العظمى (القيامة)
( ۳٦ ) بُرزت الجحيم :أظْهِرَتْ إظهارًا بَيّـنـًـا
( ۳۹ ) هي المأوى :هي المرْجع و المُقام له لا غيرها
( ٤۲ ) أيان مرساها؟ :متى يقيمها الله و يُثبتها
