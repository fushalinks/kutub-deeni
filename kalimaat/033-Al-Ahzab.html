۳۳ - سورة الأحزاب
( ۱ ) اتّقِ الله :دُمْ على تقواه أو ازدد منها
( ۳ ) وكيلا :حافظا مفَـوّضـًـا إليه كلّ أمر
( ٤ ) تُظاهرون منهنّ :تُحرّمونهنّ كحُرمة أمّهاتكم
( ٤ ) أدعياءكم :من تتبنّونهم من أبناء غيركم
( ۵ ) أقْسط :أعدل
( ۵ ) مواليكم :أولياؤكم في الدّين
( ٦ ) أولى بالمؤمنين :أرأف بهم ، و أنفع لهم
( ٦ ) أزواجه أمّهاتهم :مثلهنّ في تحريم نكاحهنّ و تعظيم حُرمتهنّ
( ٦ ) أولوا الأرحام :ذوو القرابات
( ۷ ) ميثاقهم :العَهدَ على الوفاء بما حُـمّـلوا
( ۷ ) ميثاقا غليظا :عهدا وثيقا قويّا على الوفاء
( ۹ ) جاءتكم جنود :الأحزاب يوم الخندق سنة خمس
( ۱۰ ) زاغت الأبصار :مالتْ عن سَنـَـنها حَيْرة و دهشة
( ۱۰ ) بلغتِ القلوب الحناجر :نهايات الحلاقيم ( تمثيل لشدّة الخوف )
( ۱۱ ) ابتـُـلِـيَ المؤمنون :اختـُـبـِـروا بالشّدائد و مُحّصوا
( ۱۱ ) زلزلوا :اضطربوا كثيرا من شدّة الفزع
( ۱۲ ) غرورا :قولا باطلا . أو خداعا
( ۱۳ ) يثرب :اسم المدينة المنوّرة قديما
( ۱۳ ) لا مُقام لكم :لا إقامة لكم هـهنا
( ۱۳ ) إن بيوتنا عورة :قاصية يخشى عليها العدوّ
( ۱۳ ) فِرارًا :هرَبا من القتال مع المؤمنين
( ۱٤ ) من أقطارها :نواحيها و جوانبهاد
( ۱٤ ) سُــئـلوا الفتنة :طـُـلِـبَ منهم مُقاتلة المسلمين
( ۱٤ ) ما تلبّـثوا بها :ما أخّروا المقاتلة
( ۱۷ ) يعصِمكم من الله :يمنعكم من قَدَره تعالى
( ۱۸ ) المعوّقين منكم :المُثَبّطين منكم عن الرّسول صلى الله عليه و سلم
( ۱۸ ) هلمّ إلينا :أقبـِـلوا أو قرّبوا أنفسكم إلينا
( ۱۸ ) البأس :الحرب و القتال
( ۱۹ ) أشحّةً عليكم :بُخلاء عليكم بكلّ ما ينفعكم
( ۱۹ ) يُغشى عليه من الموت :تُصيبه الغشية من سكراته
( ۱۹ ) سلقوكم :آذوكم و رموكم
( ۱۹ ) بألسنة حدادٍ :ذربة سليطة قاطعة كالحديد
( ۱۹ ) أشحّة على الخير :بُخلاء حريصين على المال و الغنيمة
( ۱۹ ) فأحبط الله :كأبطل الله
( ۲۰ ) بادون في الأعراب :كانوا معهم في البادية
( ۲۱ ) أُسوة حسنة :قُدْوة صالحة في كلّ الأمور
( ۲۳ ) قضى نحبه :وفيّ بنـَذره . أو مات شهيدا
( ۲٦ ) الذين ظاهروهم :يهود قريظة الذين عاونوا الأحزاب
( ۲٦ ) صياصيهم :حصونهم و معاقلهم
( ۲٦ ) الرّعب :الخوف الشّديد
( ۲۸ ) أمتّعكنّ :أعطكنّ مُتعة الطّلاق
( ۲۸ ) أسرّحكنّ :أطلّقكنّ
( ۲۸ ) سراحا جميلا :طلاقا حسنا لا ضِرار فيه
( ۳۰ ) بفاحشة مُـبَـيّـنة :بمعصية كبيرة ظاهرة القُبح
( ۳۱ ) يقنت منكنّ :تثطِعْ أو تخضعْ من كنّ
( ۳۲ ) فلا تخضعن بالقول :لا تلنّ القول و لا ترقّـقـنه للرّجال
( ۳۳ ) قَرْنَ في بيوتكنّ :الزمنَ بيوتكنّ و كذا جميع النّساء
( ۳۳ ) لا تبرّجن :لا تُبدين الزّينة الواجب سترُها
( ۳۳ ) الجاهليّة الأولى :ما كان قبل الإسلام من الجهالات
( ۳۳ ) الرّجس :الذنب. أو الإثم أو النّقص
( ۳٤ ) الحكمة :هدي النّبوّة أو أحكام القرآن
( ۳۵ ) القانتين :المطيعين الخاضعين لله
( ۳٦ ) الخِـيَـرَة :الإختيار
( ۳۷ ) وطرا :حاجته المهمّة ، كناية عن الطّلاق
( ۳۷ ) حرجٌ :ضيق أو إثم
( ۳۷ ) أدعيائهم :من تبنّوهم (قبل نسخ التّبنّي)
( ۳۸ ) فرض الله له :قسَمَ له أو قدّر أو أحلّ له
( ۳۸ ) خلوْا من قبل :مضوْا من قبلك من الأنبياء
( ۳۸ ) قدَرا مقدورا :مُرادا أزلاً . أو قضاءً مقضيّا
( ۳۹ ) حسيبا :مُحاسبا على الأعمال
( ٤۲ ) بُكرة و أصيلا :أوّل النّهار و آخره
( ٤۹ ) سراحا جميلا :عاريا عن أذى و منع واجب
( ۵۰ ) آتيت أجورهنّ :أعطيتهنّ مهورهنّ
( ۵۰ ) أفاء الله عليك :رجعه إليك من الغنيمة
( ۵۱ ) تُرجي :تُأخّر و لا تضاجع
( ۵۱ ) تُـؤوي إليك :تَضمّ إليك و تضاجع
( ۵۱ ) إبتغَيْتَ :طلبت
( ۵۱ ) عَزَلـتَ :اجتنبت بالإرجاء
( ۵۱ ) ذلك أدنى أن تقرّ أعينهنّ :التفويض إلى مشيئتك أقرب إلى سرورهنّ لعلمهنّ أنّه بحُكم الله
( ۵۲ ) رقيبا :حفيظا و مضطلعا
( ۵۳ ) غير ناظرين إناه :غير منتظرين نضجه و استواءه
( ۵۳ ) فانتشروا :فتفرّقوا و لا تمكُثوا عنده
( ۵۳ ) سألتموهنّ متاعا :حاجة يُنتفع بها
( ۵٦ ) يُصلّون على النبيّ :يُـثنون عليه بإظهار شرفه و تعظيم شأنه صلى الله عليه و سلم
( ۵۸ ) بُهتانا :فعلا شنيعا . أو كذبا فظيعا
( ۵۹ ) يُدنين عليهنّ :يُرخين و يُـسْدِلن عليهن
( ۵۹ ) جلابيبهنّ :ما يستتِـرن به كالمِلاءة
( ٦۰ ) المرجفون :المشيعون للأخبار الكاذبة
( ٦۰ ) لَنُغرينّك بهم :لنسلّطنّك عليهم
( ٦۱ ) ثقِـفوا :وُجدوا و أدركوا
( ٦۸ ) ضعفين :مِثلين
( ٦۹ ) وجيها :ذا جاهٍ و قدر مُستجاب الدّعوة
( ۷۰ ) قولا سديدا :صوابا . أو صدقا. أو قاصدا إلى الحقّ
( ۷۲ ) عرضنا الأمانة :التّكاليف من أوامر و نواهٍ
( ۷۲ ) فأبين :امتنعنَ
( ۷۲ ) أشفقن منها :خِفـْـنَ من الخيانة فيها
