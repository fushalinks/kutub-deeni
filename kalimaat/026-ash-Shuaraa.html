۲٦ - سورة الشعراء
( ۳ ) باخعا نفسك :مُهلِكُها حسرة و حُزنا
( ٤ ) أعناقهم :جماعاتهم أو رؤساؤهم و مقدّموهم
( ۷ ) زوج كريم :صِنفٍ حسن كثير النّــفع
( ۱۹ ) الكافرين :الجاحدين لِنعمتي
( ۲۰ ) الضّالين :المخطئين لا المتعمّدين
( ۲۲ ) عبّدت بني إسرائيل :اتـّخذتهم عبيدا لك مستذلـّين
( ۳۳ ) نزع يده :أخرجها من جيبه
( ۳۳ ) هي بيضاء :بياضا نورانيا يغشى الأبصار
( ۳٤ ) للملأ :وجوه القوم و سادتهم
( ۳٦ ) أرجه و أخاه :أخِرْ أمرهما و لا تعجل بعقوبتهما
( ۳٦ ) حاشرين . . :الشّـرَط يجمعون كل السّحرة
( ۳۹ ) هل أنتم مجتمعون :حثّ على الإجتماع و استعجالٌ له
( ٤٤ ) بعزّة فرعون :بقوّته و عظمته
( ٤۵ ) تلقف :تبتلِع بسرعة
( ٤۵ ) ما يأفكون :ما يقلبونه عن وجهه بالتـّمْـويه
( ۵۰ ) لا ضير :لا ضرر علينا فيما يُـصيبنا
( ۵۲ ) إنّكم مُـتـّــبعون :يتّـبعُكُم فِرعون و جُنُوده
( ۵۳ ) حاشرين :جامعين للجيش ليتبَعوهُم
( ۵٤ ) لشرذمة :لطائفة قليلة بالنّسبة إلينا
( ۵٦ ) حاذرون :مُحترزون . أو مُتأهّبون بالسلاح
( ٦۰ ) مشرقين :داخلين في وقت الشّروق
( ٦۱ ) تراءى الجمعان :رأى كلّ منهما الآخر
( ٦۳ ) فانفلق :انشقّ اثنيْ عشر فِرقا
( ٦۳ ) فرق :قطعة من البحر مُرتقعة
( ٦۳ ) كالطّود العظيم :كالجبل المُنطاد في السماء
( ٦٤ ) أزلفنا ثمّ الآخرين :قرّبنا هنالك آل فرعون من البحر
( ۷۵ ) أفرأيتم . . :أتأمّـلتـُـم فعلِمتم . .
( ۸٤ ) لسان صدق :ثناءً حسنا و ذكرا جميلا
( ۸۷ ) لا تخزني :لا تفضحني و لا تذلـّـني بعقابك
( ۸۹ ) بقلب سليم :بريء من مرض النفاق و الكفر
( ۹۰ ) أزلفة الجنّة :قُـرّبت بحيث يُـرى نعيمها
( ۹۱ ) برّزت الجحيم :أُظهرت بحيث تُرى أهوالها
( ۹۱ ) للغاوين :الضّالين عن طريق الحقّ
( ۹٤ ) فكبكبوا :فأُلقي الأصنام على وجوههم مرارا
( ۹۸ ) نسوّيكم بربّ العالمين :نجعلكم و إياه سواءً في استحقاق العبادة و أنتم أعجز الخلْق
( ۱۰۱ ) حميم :قريب أو شفيق يهتمّ بأمرنا
( ۱۰۲ ) كرّة :رجعة إلى الدنيا
( ۱۱۱ ) اتبعك الأرذلون :السِفلة الأدنياء من النّاس
( ۱۱۸ ) فافتح :فاحكم
( ۱۱۹ ) المشحون :المملوء بالناس و الدوابّ و المتاع
( ۱۲۸ ) ريع :طريق . أو مكان مرتفع
( ۱۲۸ ) آية :بناءً شامخا كالعلَم في الإرتفاع
( ۱۲۸ ) تعبثون :ببنائها . أو بمن يمرّ بها
( ۱۲۹ ) مصانع :حصونا أو قصورا أو حِياضا للماء
( ۱۳۲ ) أمدّكم :أنعم عليكم
( ۱۳۷ ) خلُق الأولين :عادتهم في اعتقاد أن لا بعث
( ۱٤۸ ) طلعها :ثمرُها الذي يؤول إليه الطّـلْـع
( ۱٤۸ ) هضيم :رُطب نضيج أو متدلّ لكثرته
( ۱٤۹ ) فارهين :حاذقين بـِــنحتها أو متجبّرين
( ۱۵۳ ) من المسحّرين :المغلوب على عقولهم بكثرة السّحر
( ۱۵۵ ) لها شرب :نصيب مشروب من الماء
( ۱٦٦ ) قوم عادون :متجاوزون الحدّ في المعاصي
( ۱٦۸ ) من القالين :من المبغضين أشدّ البغض
( ۱۷۱ ) في الغابرين :في الباقين في العذاب كأمثالها
( ۱۷۲ ) دمّرنا الأخرين :أهلكناهم أشدّ إهلاك
( ۱۷۳ ) مطرا :حجارة من سجيل مُهلكة
( ۱۷٦ ) أصحاب الأيكة :أصحاب الغيْضَة الكثيفة الملتفة الشجر (قرب مدين)
( ۱۸۱ ) من المخسرين :من النّاقصين للحقوق بالتطفيف
( ۱۸۳ ) لا تبخسوا :لا تنقُصوا
( ۱۸۳ ) لا تعثوا :لا تُفسِدوا أشدّ الإفساد
( ۱۸٤ ) و الجبلّة الأوّلين :و خلَق الخليقة و الأمم الماضين
( ۱۸۵ ) المسحّرين :المغلوبة عقولهم بكثرة السّحر
( ۱۸۷ ) كسفا :قطَعَ عذاب
( ۱۸۹ ) الظّلة :سحابة أظلّــتهم ثم أمطرتهم نارًا
( ۱۹٦ ) زبر الأولين :كتب الرّسل السابقين
( ۲۰۲ ) بغتة :فجأة
( ۲۰۳ ) هل نحن منظرون :ممهلون لِنؤمن ؟ كلاّ
( ۲۰۵ ) أفرأيت :أخبرني
( ۲۰۷ ) ما أغنى عنهم :أي شيء أغنى عنهم – لم يُغنِ
( ۲۱۵ ) اخفظ جناحك :ألِن جانبك و تواضع
( ۲۱۹ ) و تقلّبك في السّاجدين :و يرى تقلـّـبك في الصلاة مع المصلـّين
( ۲۲۲ ) أفاك أثيم :كثير الكذب و الإثم كالكهنة
( ۲۲۵ ) يهيمون :يخوضون و يذهبون كل مذهب
