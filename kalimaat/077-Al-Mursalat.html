۷۷ - سورة المرسلات
( ۱ ) و المرسلات عُرْفا ( لهذه الأقسام الخمسة تفسيرات كثيرة اخترنا هذا منها ) :(أقسم الله) برياح العذاب ممتابعة كَعُرْف الفَـرس
( ۲ ) فالعاصفات عَصفا :الرّياح الشديدة الهبوب المُهلكة
( ۳ ) و الناشرات نشرا :الملائكة تنشر أجنحتها في الجَوّ عند النّـزول بالوحي
( ٤ ) فالفارقات فرقا :الملائكة تأتي بالوحي فُرقانا بين الحقّ و الباطل
( ۵ ) فالمُلقِـيات ذكرا :الملائكة تُـلقي الوحْي إلى الأنبياء
( ٦ ) عُذرا :للإعذار من الله للخَلـْـق
( ٦ ) نـُـذرا :للإنذار و التـّـخويف بالعِقاب
( ۷ ) إنما توعدون :من البَعْثْ ( جَوَاب القـَـسَم)
( ۸ ) النجوم طُمِسَتْ :مُـحِـيَ نورُها و أذهِبَ ضوؤها
( ۹ ) السّماء فُـرجَت :شُـقّـت أو فُتِحَتْ فكانت أبوابا
( ۱۰ ) الجبال نُسِفت :قُـلِعَت من أماكِنِها بسُرْعَة
( ۱۱ ) الرّسل أقـّـتت :بُلـّـغَتْ ميقاتـَها ( يوم القيامة)
( ۱۲ ) لأي يوم أجّـلت :يُقال لأيّ يوم أخِّرَتْ
( ۱۳ ) ليوم الفـَـصْـل :بين الخلائق أو الحقّ و الباطل
( ۱۵ ) ويْـلٌ يومئذ :هلاكٌ في ذلك اليوم الهائل
( ۲۰ ) ماءٍ مهين :مَنِـيّ ضعيف حَقير
( ۲۱ ) قرار مكين :مُتمَكّن ، و هو الرّحِم
( ۲۳ ) فقـَدَرنا :فـقـَـدّرْنا ذلك تقديرا
( ۲۵ ) الأرض كِفاتا :وعاءًا تضمّ الأحياء على ظهرها
( ۲٦ ) أحْياءً و أمْوَاتا :و الأموات في بطنها
( ۲۷ ) رواسِيَ شامخات :جـِبالا ثوابتَ مُرْتـَفِعات
( ۲۷ ) ماءً فـُرَاتـًا :حُـلوًا عَذبًـا
( ۳۰ ) ظلّ :هو دُخان جهنّم
( ۳۰ ) ثلاث شُعَب :فِرَقٍ ثلاث كالذوائب
( ۳۱ ) لا ظليل :لا مظلـّـل من الحرّ
( ۳۱ ) لا يُغني من اللـّهَب :لا يَدْفـَعُ شيْئا من حرّه
( ۳۲ ) ترمي بشَرَر :هو ما تطايَر من النّار مُتفـرّقًـا
( ۳۲ ) كالقـَـصْر :كلّ شرَرَة كالبناء المُشيّد في العِظم و الإرتفاع
( ۳۳ ) كأنّه جـِمالة ٌ صُـفر :كأنّ الشّرر إبل سودٌ "و تـُسـمّيها العَرَب صفرا" في الكثرة و التـّـتابع و سُرْعة الحركة و اللون
( ۳۹ ) لكم كيْدٌ :
( ) :حيلة لاتـّـقاء العذاب
