۲۲ - سورة الحج
( ۱ ) زلزلة الساعة :أهوال القيامة وشدائدها
( ۲ ) تذهل :تغفُلُ وتُشغلُ لشدة الهول
( ۳ ) مريد :متمرّدٍ عاتٍ متجرد للفساد
( ٤ ) تولاّه :اتخذه وليا وتبعه
( ۵ ) نطفة :منيّ
( ۵ ) علقة :قطعة دم جامدة
( ۵ ) مُضغة :قطعة لحم قدر ما يمضغ
( ۵ ) مُخلقة :مستبينة الخلق مُصوّرة
( ۵ ) لتبلغوا أشدكم :كمال قوتكم وعقلكم
( ۵ ) أرذل العمر :أخسّـِـه , أي الخَرَف والهَرَم
( ۵ ) هامدة :ميتة يابسة قاحلة
( ۵ ) اهتزتْ :تحركت بالنبات
( ۵ ) رَبَتْ :ازدادت وانتفخت
( ۵ ) زوج بهيج :صنف حسن نضير
( ۹ ) ثانيَ عِطفه :لاويا لجانبه تكبرا وإباء
( ۹ ) خزيٌ :ذلّ وهوان
( ۱۱ ) على حرْف :شكّ وقلق و تزلزل في الدين
( ۱۳ ) المولى :الناصر
( ۱۳ ) العشير :المصاحب المعاشر
( ۱۵ ) ينصره الله :ينصر الله رسوله صلى الله عليه وسلم
( ۱۵ ) بسبب إلى السماء :بحبل إلى سقف بيته
( ۱۵ ) ثم ليقطع :ثم ليختنق به حتى يموت
( ۱۵ ) كيده :صنيعه بنفسه
( ۱۷ ) الصّـابئين :عبدة الملائكة أو الكواكب
( ۱۸ ) يسجد له :يخضع وينقاد لإرادته تعالى
( ۱۸ ) حق عليه :ثبت ووجب عليه
( ۱۹ ) خصْـمان :المؤمنون وسائر الكفار
( ۱۹ ) الحميم :الماء البالغ نهاية الحرارة
( ۲۰ ) يُصهر به :يُذاب به
( ۲۱ ) مقامِع :مطارق أو سِـياط
( ۲٤ ) صراط الحميد :الإسلام الذي ارتضاه لعباده دينا
( ۲۵ ) المسجد الحرام :مكة (الحرم)
( ۲۵ ) العاكف فيه :المقيم فيه الملازم له
( ۲۵ ) الباد :الطارئ غير المقيم
( ۲۵ ) بإلحاد بظلم :بميل عن الحق إلى الباطل
( ۲٦ ) بوأنا لإبراهيم :وطـّـأنا . أو بـيّـنا له
( ۲۷ ) أذن في الناس :نادِ فيهم وأعـلِـمهم
( ۲۷ ) رجالا :مشاة على أرجلهم
( ۲۷ ) ضامر :بعير مهزول من بُعد الشـقـّـة
( ۲۷ ) فج عميق :طريق بعيد
( ۲۸ ) بهيمة الأنعام :الإبل والبقر والضأن والمعز
( ۲۹ ) ثم ليقضوا تـفـثهم :ثم ليزيلوا بالتحلل أوساخهم أو ثم ليؤدوا مناسكهم
( ۳۰ ) حُرمات الله :تكاليفه من مناسك الحج وغيرها
( ۳۰ ) الرّجس . . :القذر والنجس وهو الأوثان
( ۳۰ ) قول الزور :قول الباطل والكذب القبيح
( ۳۱ ) حُنفاء لله :مائلين عن الباطل الى دين الحقّ
( ۳۱ ) تهوى به الريح :تُسقطه وتقذفه
( ۳۱ ) مكان سحيق :موضع بعيد مُـهـلـِـك
( ۳۲ ) شعائر الله :البُدن المهداة للبيت المُعظم
( ۳۳ ) محلها :وجوب نحرها
( ۳۳ ) إلى البيت العتيق :منتهية إلى أرض الحرم كله
( ۳٤ ) منسكا :نُسكا وعبادة ( الذبح قُربة لله )
( ۳٤ ) بَشـّر المُخبتين :المُطمئنين إلى الله أو المُتواضعين له
( ۳۵ ) وجلت قلوبهم :خافت هـيْـبة و إجلالا منه تعالى
( ۳٦ ) البُدن :الإبل . أو هي البقر المهداة للبيت
( ۳٦ ) شعائر الله :أعلام شريعته في الحج
( ۳٦ ) صوافّ :قائمات صففن أيديَهُن وأرجـُـلهُن
( ۳٦ ) وجبت جُنُوبُها :سقطت على الأرض بعد النحر
( ۳٦ ) أطعموا القانع :السائل
( ۳٦ ) المُعترّ :الذي يتعرض لكم دون سؤال
( ۳۸ ) خوان كفور :خائن للأمانات – جاحد للنعم
( ٤۰ ) صوامعُ :معابدُ رُهبان النصارى
( ٤۰ ) بـِــيَع :كنائس النصارى
( ٤۰ ) صلواتٌ :كنائس اليهود
( ٤۰ ) مساجدُ :للمسلمين
( ٤٤ ) أصحابُ مدين :قومُ شعيب عليه السلام
( ٤٤ ) فأمليتُ للكافرين :أمهلتهم وأخـّـرت عقوبتهمْ
( ٤٤ ) كان نكير :إنكاري عليهم بإهلاكهم
( ٤۵ ) فكأين من قرية :فكثير من القرى
( ٤۵ ) خاوية على عروشها :ساقطة حيطانها على سقوفها المتهدمة
( ٤۵ ) قصر مشيد :مرفوع البنيان خالٍ من ساكنيه
( ٤۸ ) أمليت لها :أمْـهَـلـتـُـها
( ۵۱ ) مُعاجزين :ظانين أنهم يُعجزوننا ويفُوتوننا
( ۵۲ ) تمنّى :قرأ الآيات المنزلة عليه
( ۵۲ ) ألقى الشيطان في أمنيته :ألقى في قلوب أوليائه الشُـبَهَ فيما يقرؤه
( ۵٤ ) فـتـُخبت له :فتطمئن وتسْكن للقرآن
( ۵۵ ) ِمرية منه :شك وقلق من القرآن
( ۵۵ ) يوم عقيم :لا يوم بعده ( يوم القيامة )
( ۵۹ ) مُدخلا :الجنة . أو درجات رفيعة فيها
( ٦۰ ) ثم بُغي عليه :ظلم بمعاودة العقاب
( ٦۱ ) يُولج :يُدخل
( ٦۷ ) منسكا :شريعة خاصة . أو نـُسُـكا وعبادة
( ۷۱ ) سُلطانا :حُجة وبرهانا
( ۷۲ ) المنكر :الأمر المستقبح من العبوس والتجهّم
( ۷۲ ) يسْطون :يَــثـِـبـُـون ويَبْطشُون غيظا وغضبا
( ۷٤ ) ما قدروا الله :ما عظّـموه . أو ما عَرَفوه
( ۷۸ ) هو اجتباكم :اختاركم لدينه وعبادته ونُصرته
( ۷۸ ) حرج :ضيق بتكليف يشقّ ويعسُرُ
( ۷۸ ) هو مولاكم :مَالِكِكُم وناصِرُكم و متولّي أموركم
