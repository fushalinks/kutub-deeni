۲۱ - سورة الأنبياء
( ۱ ) اقترب :قرُب و دنا
( ۲ ) مُحدَث :تنزيله بالوحي
( ۳ ) أسرّوا النجوى :بالغوا في إخفاء تناجيهم
( ۵ ) أضغاث أحلام :تخاليط أحلام رآها في نومه
( ۸ ) جسدا :أجسادا ، أو ذوي جسد
( ۱۰ ) فيه ذكركم :موعظتكم أو شرفكم و صيتكم
( ۱۱ ) كم قصمنا :كثيرا أهلكنا
( ۱۲ ) أحسّوا بأسنا :أدركوا بحاسّتهم عذابنا الشديد
( ۱۲ ) يركضون :يهربون مسرعين
( ۱۳ ) أترفتم فيه :نعّمتم فيه فبَطِرتم
( ۱۵ ) حصيدا :كالنّبات المحصود بالمناجل
( ۱۵ ) خامدين :ميّتين كانّار التي سكن لهبها
( ۱۷ ) نتّخذ لهوا :ما يُـتـَـلهّى به من صاحبة أو ولد
( ۱۸ ) نقذف بالحق :نرمي به و نورده
( ۱۸ ) فيدمغه :يمحقه و يدحضه
( ۱۸ ) زاهق :ذاهب مُضمحلٌ
( ۱۸ ) الويل :الهلاك أو الخزي أو وادٍ بجهنّم
( ۱۹ ) لا يستحسرون :لا يكلّون و لا يعيوْن
( ۲۰ ) لا يفترون :لا يسكنون عن نشاطهم في التسبيح و العبادة
( ۲۱ ) هم ينشرون :هم يُحيون الموتى – كلاّ
( ۲۲ ) لفسدتا :لأختلّ نظامهما و خربتا بالتـّـنازع
( ۲٦ ) ولدا :قالوا الملائكة بنات الله
( ۲۸ ) مُشفقون :خائفون حذرون
( ۳۰ ) كانتا رتقا :كانتا مُلتصِقتين بلا فصْـلٍ
( ۳۰ ) ففتقناهما :ففصلنا بينهما بالهواء
( ۳۰ ) كلّ شيء حيّ :كل شيء نامٍ حيوانا أو نباتا
( ۳۱ ) رواسي :جبالا ثوابت
( ۳۱ ) أن تميد بهم :لئلاّ تضطرب بهم فلا تثبُت
( ۳۱ ) فِجاجا سُبُلا :طُرُقا واسعة مسلوكة
( ۳۲ ) سقفا محفوظا :مصونا من الوُقوع أو التّغيّر
( ۳۳ ) كلّ :من الشّمس و القمر
( ۳۳ ) في فلك يسبحون :يدورون . أو يجرون في السّماء
( ۳۵ ) نبلوكم :نختبركم مع علمِنا بحالكم
( ۳۹ ) لا يكفّون :لا يمنعون و لا يدفعون
( ٤۰ ) بغتة :فجأة
( ٤۰ ) فتبهتهم :تحيّرُهم و تدهشهم
( ٤۰ ) يُنظرون :يُمهلون و يُؤخّرون
( ٤۱ ) فحاق :أحاط . أو نزل
( ٤۲ ) يأكلؤكم :يحفظكم و يحرسكم
( ٤۳ ) يُصحَبون :يُجارون و يمنعون أو يُنصرون
( ٤٦ ) نفحة :دُفعة يسيرة . أو نصيب يسير
( ٤۷ ) القسط :العدل . أو ذوات العدل
( ٤۷ ) مثقال حبّة :وزن أقلّ شيء
( ٤۹ ) مُشفقون :خائفون خذرون
( ۵۲ ) التّماثيل :الأصنام المصنوعة بأيديكم
( ۵٦ ) فطرهنّ :خلقهنّ و أبدعهنّ
( ۵۸ ) جُذاذا :قِطعا و كِسرا
( ٦۱ ) على أعين النّاس :ظاهرا بمرأى من النّاس
( ٦۵ ) نُكِسوا على رءوسهم :رجعوا إلى الباطل و العناد
( ٦۷ ) أفٍ لكم . . :كلمة تضجّر و كراهيّة و تبرّم
( ۷۱ ) إلى الأرض :مُنهيا إلى أرض الشّـام
( ۷۲ ) نافلة :عطيّة أو زيادة عمّا سأل
( ۷٤ ) قوم سوْء :فساد و فعلٍ مكروه
( ۷۸ ) الحرث :الزّرع . أو الكرم
( ۷۸ ) نفشت فيه :انتشرت فيه ليلا بلا راعٍ فرعته
( ۸۰ ) صنعة لَبوس :عمل الدّروع تلبس في الحرب
( ۸۰ ) لِتُحصنكم :لتحفظكم و تقيكم
( ۸۰ ) بأسكم :حرب عدوّكم و إصابتكم بسلاحه
( ۸۱ ) عاصفة :شديدة الهبوب
( ۸۲ ) يغوصون له :في البحار لاستخراج نفائسها
( ۸۲ ) لهم حافظين :من الزّيغ عن أمره أو الإفساد
( ۸۵ ) ذا الكفل :قيل هو إلياس عليه السّلام
( ۸۷ ) ذا النّون :صاحب الحوت يونس عليه السّلام
( ۸۷ ) مُغاضبا :غضبان على قومه لكُـفرهم
( ۸۷ ) لم نقدِر عليه :لن نضيّق عليه بحبس و نحوه
( ۹۰ ) رغبا و رهبا :رجاءً في الثّواب و خوفا من العِقاب
( ۹۰ ) خاشعين :متذلّلين خاضعين
( ۹۱ ) أحصنت فرجها :حفظته من الحلال و الحرام
( ۹۱ ) من روحنا :من جهة روحنا و هو جبريل
( ۹۲ ) أمّتكم :ملّـتكم (الإسلام)
( ۹۳ ) تقطّعوا أمرهم :تفرّقوا في دينهم فِرقا و أحزابا
( ۹۵ ) حرام على قرية :مُمتنِعٌ البتـّة على أهل قرية
( ۹۵ ) أنّهم لا يرجعون :إلينا بالبعث للجزاء
( ۹٦ ) حَدَبٍ :مرتفع من الأرض
( ۹٦ ) ينسِلون :يُسرعون المشي في الخروج
( ۹۷ ) الوعد الحقّ :البعث و الحساب و الجزاء
( ۹۷ ) شاخصة أبصار . . :مُرتفعة لا تكاد تطرف أبصار . .
( ۹۸ ) حصب جهنّم :حطبها ووقودها الذي به تهيَّج
( ۹۸ ) لها واردون :فيها داخلون
( ۱۰۰ ) زفير :تنفّس شديد تنتفخ منه الضّلوع
( ۱۰۲ ) حسيسها :صوت حركة تلهّبها
( ۱۰۳ ) الفزع الأكبر :حين نفخة البعث
( ۱۰٤ ) السجلّ :الصّحيفة التي يُكتب فيها
( ۱۰٤ ) للكتب :على ما كُتِب في السّجل
( ۱۰۵ ) الزّبور :الكُتب المنزّلة
( ۱۰۵ ) الذكر :اللّوح المحفوظ
( ۱۰٦ ) لبلاغا :كِفاية، أو وُصولا إلى البُغية
( ۱۰۹ ) آذنتكم :أعلمتكم ما أمرت به
( ۱۰۹ ) على سواءٍ :مُستوين جميعا في الإعلام به
( ۱۰۹ ) و إن أدري :و ما أدري و ما أعلم
( ۱۱۱ ) فتنة لكم :امتحان لكم
