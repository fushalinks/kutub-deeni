۱۰٤ - سورة الهمزة
( ۱ ) وَيْل :عذاب أو هَلاك أوْ وَادٍ في جَهنّم
( ۱ ) هُمَزةٍ لُـمَزةٍ :طَعّـان غَـيّـاب عَـيّـاب للناس
( ۲ ) عَدّده :أحصاه . أو أعدّه للنّوائب
( ۳ ) أخلده :يُخـلـّـدُهُ في الدنيا
( ٤ ) لـَـيُنـْـبَذنّ :لـَـيُطرَحَنّ
( ٤ ) الحُطمة :جَهنّم . لِحَطْمِها كلّ ما يُـلْـقى فيها
( ۷ ) تطّـلع على الأفـئدة :تـَـغْشـَى حَرَارتها أوْساط القلوب
( ۸ ) مؤْصَدَة :مُـطبَـقـَـة مُـغْـلـقـَة ٌ أبْوابها
( ۹ ) في عَمَدٍ مُمَدّدة :بأعمِدةٍ ممدودةٍ على أبْوابها
