۱۰۱ - سورة القارعة
( ۱ ) القارعة :القيامة تـقـْـرَعُ القـلوب بأهوالِها
( ٤ ) كالفراش :هوَ طيْرٌ كالبَعوض يَتهافتُ في النّار
( ٤ ) المبْثوث :المُتـفـرّق المُـنـتـَـشِر
( ۵ ) كالعِهن :كالصّوف المصْبوغ بألوان مُختلفة
( ۵ ) المنفوش :المُـفرّق بالأصابع و نحْوها
( ٦ ) ثقـُـلتْ موازينه :رَجَحتْ مَقادير حسناته
( ۸ ) خَـفـّـت موازينه :رَجَحت مقادير سَيّـئاته
( ۹ ) فأمّه هاوية :فمأواه جهنّم يَهْوي فيها
( ۱۰ ) مَاهِـيَهْ :ما هيَ – و الهاء للسّكت
