٦٦ - سورة التحريم
( ۱ ) ما أحلّ الله لك
شرب العسل
( ۱ ) تبْـتغي
تـَـطلب
( ۲ ) تحلّة أيْمانكمْ
تحليلها بالكفّارة
( ۲ ) الله مولاكمْ
ناصِركم و متولّي أموركم
( ۳ ) نبّـأت به
أخْـبَرَتْ به غيرها
( ۳ ) أظهَرَه الله عليه
أطلعه الله تعالى على إفشائه
( ٤ ) صَغَتْ قلوبكما
مَالتْ عن حَقـّه صلى الله عليه و سلم عليكما
( ٤ ) تظاهَرَا عليه
تتعاونـَا عليه بما يسوءُه
( ٤ ) هو مولاه
وَليّـه و ناصِره
( ٤ ) ظهير
فوْجٌ مُظاهر معينٌ له
( ۵ ) قانتات
مطيعات خاضعاتٍ لله
( ۵ ) سائحات
مهاجرَاتٍ . أو صائمات
( ٦ ) قوا أنفسكم
جَـنّـبوها بالطاعات
( ٦ ) غِلاظ شِداد
قساةٌ أقوياء و هم الزبانية
( ۸ ) توبة نـَـصوحا
خالصة . أو صادقة . أو مقبولة
( ۸ ) لا يُخزي الله النبي
لا يُـذلّه بل يُعزّه و يكرمه
( ۹ ) اغلظ عليهم
شدِّدْ. أو اقـْـسُ عليهم
( ۱۰ ) فخانتاهما
بالنّـفاق أو بالنميمة
( ۱۰ ) فلم يغنيا عنهما
فـلمْ يَدْفَعا و لم يمْنعاعنهما
( ۱۲ ) أحصنت فرجها
عَـفّـتْ و صانته من الرّجال
( ۱۲ ) من روحنا
روحًا مِنْ خَلقِـنـَـا بلا توسّط أبٍ (عيسى عليه السلام)
( ۱۲ ) من القانتين
من القوم المطيعين لربّهم
