۵۱ - سورة الذاريات
( ۱ ) و الذاريات ذروا
(قسَمٌ) بالرياح تذرو و تفرّق التراب و غيره ذروا
( ۲ ) فالحاملات وقرا
السّحب تحمل الأمطار حَمْلاً
( ۳ ) فالجاريات يُسرا

( ۹ ) يؤفك عنه

( ۱۰ ) قتل الخرّاصون

( ۱۳ ) يُـفـتـنون

( ۱۷ ) يَهجعون

( ۲٤ ) ضيف إبراهيم
أضيافه من الملائكة
( ۲۵ ) قوم منكرون
قاله في نفسه لغرابَتهمْ
( ۲٦ ) فراغ إلى أهله
ذهَبَ إليهم في خِفـيَة من ضَيفه
( ۲۸ ) فأوجس منهم

( ۲۸ ) بغلام عليم

( ۲۹ ) صرّة
صَيْحة و ضَجّـة
( ۲۹ ) فصكّت وجهها
لطمته بيدها تعجّبا
( ۳۱ ) فما خطبكم ؟
فما شأنكم الخطير ؟
( ۳٤ ) مسَوّمة
مُعْـلـَـمَة بأنّها حجارة عذاب
( ۳۸ ) و في موسى
و جعلنا في قصّة موسى آية
( ۳۹ ) فتولـّـى بركنه
فأعرض فرعون بقـوّتِه و سلطانه عن الإيمان
( ٤۰ ) هو مليم
آتٍ بما يُلام عليه الكُـفْـر
( ٤۱ ) الريح العقيم
المُهلكة لهمْ، القاطعة لنـَـسْـلهمْ
( ٤۲ ) كالرّميم
كالشيء البالي المفـتـّـت الهالك
( ٤٤ ) فعَتوْا
فاستكبروا

لقادرون
