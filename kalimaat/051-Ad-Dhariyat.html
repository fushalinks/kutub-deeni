۵۱ - سورة الذاريات
و الذاريات ذروا ( ۱ )
(قسَمٌ) بالرياح تذرو و تفرّق التراب و غيره ذروا
فالحاملات وقرا ( ۲ )
السّحب تحمل الأمطار حَمْلاً
فالجاريات يُسرا ( ۳ )
السّـفن تجري على الماء جَرْيا سهلا
فالمقسمات أمرا ( ٤ )
الملائكة تقسّم المقدّرات الرّبانيّة
إن ما توعدون ( ۵ )
مِنَ البعث (جواب القسم)
إن الدين ( ٦ )
الجزاء بعد الحساب
ذات الحُبك ( ۷ )
الطرق التي تسير فيها الكواكب
قوْل مختلف ( ۸ )
متناقضٍ فينا كلـّـفتم الإيمان به
يؤفك عنه ( ۹ )
يُصْرَف عن الحقّ الآتي به الرسول
قتل الخرّاصون ( ۱۰ )
لـُعِنَ و قُــبّـح الكذابون
غمرة ( ۱۱ )
جهالة غامرة بأمور الآخرة
ساهون ( ۱۱ )
غافلون عمّا أمروا به

متى يوم الجزاء ؟ (إنكارٌ له)
يُـفـتـنون ( ۱۳ )
يُحْرَقونو يعذبون
يَهجعون ( ۱۷ )
يَنامون
بالأسحار ( ۱۸ )
أواخر الليل
المحروم ( ۱۹ )
الذي حُرم الصدقة لتعفّـفه عن السؤال مع حاجته
ضيف إبراهيم ( ۲٤ )
أضيافه من الملائكة
قوم منكرون ( ۲۵ )
قاله في نفسه لغرابَتهمْ
فراغ إلى أهله ( ۲٦ )
ذهَبَ إليهم في خِفـيَة من ضَيفه
فأوجس منهم ( ۲۸ )
فأحسّ في نفسه منهم
بغلام عليم ( ۲۸ )
هو هنا إسحاق عند الجمهور
صرّة ( ۲۹ )
صَيْحة و ضَجّـة
فصكّت وجهها ( ۲۹ )
لطمته بيدها تعجّبا
فما خطبكم ؟ ( ۳۱ )
فما شأنكم الخطير ؟
مسَوّمة ( ۳٤ )
مُعْـلـَـمَة بأنّها حجارة عذاب
و في موسى ( ۳۸ )
و جعلنا في قصّة موسى آية
فتولـّـى بركنه ( ۳۹ )
فأعرض فرعون بقـوّتِه و سلطانه عن الإيمان
هو مليم ( ٤۰ )
آتٍ بما يُلام عليه الكُـفْـر
الريح العقيم ( ٤۱ )
المُهلكة لهمْ، القاطعة لنـَـسْـلهمْ
كالرّميم ( ٤۲ )
كالشيء البالي المفـتـّـت الهالك
فعَتوْا ( ٤٤ )
فاستكبروا
فأخذتهم الصّاعقة ( ٤٤ )
فأهلكتهم صيحة أو نارٌ من السماء
بنيْناها بأيدٍ ( ٤۷ )
بقوّة و قدْرَة
إنّـا لموسعون ( ٤۷ )
لقادرون
الأرض فرشناها ( ٤۸ )
مهّدناها و بَسَطناها كالفراش للاستقـرار عليها
فنِعْمَ الماهدون ( ٤۸ )
المسَوّون المُصلحون
خلقنا زوجين ( ٤۹ )
