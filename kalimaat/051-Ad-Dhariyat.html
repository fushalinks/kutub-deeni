۵۱ - سورة الذاريات
( ۱ ) و الذاريات ذروا :(قسَمٌ) بالرياح تذرو و تفرّق التراب و غيره ذروا
( ۲ ) فالحاملات وقرا :السّحب تحمل الأمطار حَمْلاً
( ۳ ) فالجاريات يُسرا :السّـفن تجري على الماء جَرْيا سهلا
( ٤ ) فالمقسمات أمرا :الملائكة تقسّم المقدّرات الرّبانيّة
( ۵ ) إن ما توعدون :مِنَ البعث (جواب القسم)
( ٦ ) إن الدين :الجزاء بعد الحساب
( ۷ ) ذات الحُبك :الطرق التي تسير فيها الكواكب
( ۸ ) قوْل مختلف :متناقضٍ فينا كلـّـفتم الإيمان به
( ۹ ) يؤفك عنه :يُصْرَف عن الحقّ الآتي به الرسول
( ۱۰ ) قتل الخرّاصون :لـُعِنَ و قُــبّـح الكذابون
( ۱۱ ) غمرة :جهالة غامرة بأمور الآخرة
( ۱۱ ) ساهون :غافلون عمّا أمروا به
( ۱۲ ) أيّـان يوم الدّين ؟ :متى يوم الجزاء ؟ (إنكارٌ له)
( ۱۳ ) يُـفـتـنون :يُحْرَقونو يعذبون
( ۱۷ ) يَهجعون :يَنامون
( ۱۸ ) بالأسحار :أواخر الليل
( ۱۹ ) المحروم :الذي حُرم الصدقة لتعفّـفه عن السؤال مع حاجته
( ۲٤ ) ضيف إبراهيم :أضيافه من الملائكة
( ۲۵ ) قوم منكرون :قاله في نفسه لغرابَتهمْ
( ۲٦ ) فراغ إلى أهله :ذهَبَ إليهم في خِفـيَة من ضَيفه
( ۲۸ ) فأوجس منهم :فأحسّ في نفسه منهم
( ۲۸ ) بغلام عليم :هو هنا إسحاق عند الجمهور
( ۲۹ ) صرّة :صَيْحة و ضَجّـة
( ۲۹ ) فصكّت وجهها :لطمته بيدها تعجّبا
( ۳۱ ) فما خطبكم ؟ :فما شأنكم الخطير ؟
( ۳٤ ) مسَوّمة :مُعْـلـَـمَة بأنّها حجارة عذاب
( ۳۸ ) و في موسى :و جعلنا في قصّة موسى آية
( ۳۹ ) فتولـّـى بركنه :فأعرض فرعون بقـوّتِه و سلطانه عن الإيمان
( ٤۰ ) هو مليم :آتٍ بما يُلام عليه الكُـفْـر
( ٤۱ ) الريح العقيم :المُهلكة لهمْ، القاطعة لنـَـسْـلهمْ
( ٤۲ ) كالرّميم :كالشيء البالي المفـتـّـت الهالك
( ٤٤ ) فعَتوْا :فاستكبروا
( ٤٤ ) فأخذتهم الصّاعقة :فأهلكتهم صيحة أو نارٌ من السماء
( ٤۷ ) بنيْناها بأيدٍ :بقوّة و قدْرَة
( ٤۷ ) إنّـا لموسعون :لقادرون
( ٤۸ ) الأرض فرشناها :مهّدناها و بَسَطناها كالفراش للاستقـرار عليها
( ٤۸ ) فنِعْمَ الماهدون :المسَوّون المُصلحون
( ٤۹ ) خلقنا زوجين :صِنـفَـيْـن و نوعين مختلفين
( ۵۰ ) فـفـرّوا إلى الله :فاهربوا من عقابه إلى ثوابه
( ۵۳ ) طاغون :متجاوزون الحدّ في الكفر
( ۵٦ ) ليَعْبدون ِ :ليعرفوني أو ليخْـضعوا لي و يتذلـّـلوا
( ۵۹ ) ذنوبًا :نـَـصيبًـا مِنَ العَذاب
( ٦۰ ) فوَيْـلٌ :هلاكٌ . أو حسْرَةٌ أو شدّة عَـذاب
