۸ - سورة الأنفال
( ۱ ) الْأَنْفَالِ :غنائم بدر
( ۷ ) ذَاتِ الشَّوْكَةِ :ذات السّلاح و القوّة . و هي النّفير
( ۹ ) مُرْدِفِينَ :مُتبِعا بعضهم بعضا آخر منهم
( ۱۱ ) رِجْزَ الشَّيْطَانِ :وسوسته و تخويفه إياكم من العطش
( ۱۲ ) كُلَّ بَنَانٍ :كلّ الأطراف أو كلّ مفصل
( ۱۳ ) شَاقُّوا اللَّهَ وَرَسُولَهُ :خالفوا و عصوْا
( ۱۵ ) زَحْفًا :جيشا زاحفا نحوكم لقتالكم
( ۱٦ ) مُتَحَرِّفًا لِقِتَالٍ :مظهرا الفرار خِدعة ثم يكرّ
( ۱٦ ) مُتَحَيِّزًا إِلَى فِئَةٍ :منضمّا إليها ليقاتل العدوّ معها
( ۱٦ ) بَاءَ بِغَضَبٍ مِنَ اللَّهِ :رجع متلبّسا به مستحقّا له
( ۱۷ ) لِيُبْلِيَ الْمُؤْمِنِينَ :ليُنعم عليهم بالنّصر و الأجر
( ۳۵ ) مُكَاءً وَتَصْدِيَةً :صفيرا و تصفيقا
( ۳۷ ) فَيَرْكُمَهُ جَمِيعًا :فيجمعه ملقى بعضه على بعض
( ٤۲ ) بالعدوة الدّنيا :بحافّة الوادي و ضفّته الأقرب للمدينة
( ٤۲ ) الرَّكْبُ :عير قريش فيها أموالهم
( ٤٦ ) تَذْهَبَ رِيحُكُمْ :تتلاشى قوّتكم أو دولتكم
( ٤۷ ) بَطَرًا :طغيانًا أو فخرا و أشرّا
( ٤۸ ) نَكَصَ عَلَى عَقِبَيْهِ :رجع القهقري و ولّى مُدبرا
( ۵۲ ) كَدَأْبِ آلِ فِرْعَوْنَ :كعادة . .
( ۵۷ ) تَثْقَفَنَّهُمْ :تصادفنّهم و تظفرنّ بهم
( ۵۷ ) فَشَرِّدْ بِهِمْ :ففرّق و بدّد و خوّف بهم
( ٦۰ ) رِبَاطِ الْخَيْلِ :حبسها للجهاد في سبيل الله
( ٦۱ ) جَنَحُوا لِلسَّلْمِ :مالوا للمُسالمة و المصالحة
( ٦۷ ) حَتَّى يُثْخِنَ فِي الْأَرْضِ :يُبالغ في القتل حتى يذل الكفر
