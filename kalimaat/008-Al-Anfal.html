۸ - سورة الأنفال
( ۱ ) الأنفال :غنائم بدر
( ۱ ) لله و الرّسول :مفوّضٌ إليهما أمرُها
( ۱ ) ذات بينكم :أحوالكم التي يحصل بها اتّصالكم
( ۲ ) وجلت قلوبهم :فزعت و رقّت استعظاما و هيبة
( ۲ ) يتوكّلون :يعتمدون و إلى الله يُفوّضون
( ۷ ) الطائفين :هما العير و النّفير
( ۷ ) ذات الشّوكة :ذات السّلاح و القوّة . و هي النّفير
( ۷ ) دابر الكافرين :آخرهم و المراد جميعهم
( ۹ ) مُردفين :مُتبِعا بعضهم بعضا آخر منهم
( ۱۱ ) يغشّيكم النّعاس :يجعله غاشيا عليكم كالغطاء
( ۱۱ ) أمنة منه :أمنا من الله و تقوية لكم
( ۱۱ ) رجز الشّيطان :وسوسته و تخويفه إياكم من العطش
( ۱۱ ) ليربط :يشدّ و يقوّي باليقين و الصّبر
( ۱۲ ) أنِّي معكم :مُعينكم على تثبيت المؤمنين
( ۱۲ ) الرّعب :الخوف و الفزع و الإنزعاج
( ۱۲ ) كلّ بنان :كلّ الأطراف أو كلّ مفصل
( ۱۳ ) شاقّوا :خالفوا و عصوْا
( ۱۵ ) زحفا :جيشا زاحفا نحوكم لقتالكم
( ۱٦ ) مُتحرّفا :مظهرا الفرار خِدعة ثم يكرّ
( ۱٦ ) مُتحيّزا إلى فئة :منضمّا إليها ليقاتل العدوّ معها
( ۱٦ ) باء بغضب :رجع متلبّسا به مستحقّا له
( ۱۷ ) لِيُبليَ المؤمنين :ليُنعم عليهم بالنّصر و الأجر
( ۱۸ ) موهن . . :مُضعف . .
( ۱۹ ) تستفتحوا :تطلبوا النّصر لأهدى الفئتين
( ۲٤ ) يُحييكم :يورثكم حياةً أبديّة في نعيم سرمديّ
( ۲٦ ) يتخطّفكم النّاس :يستلبوكم و يصطلموكم بسرعة
( ۲۸ ) فتنة :ابتلاء و محنة أو سبب في الإثم و العقاب
( ۲۹ ) فرقانا :هداية و نورا أو نجاة . أو مخرجًا
( ۳۰ ) ليُثبتوك :ليحبسوك أو ليُقيّدوك بالوثاق
( ۳۰ ) يمكر الله :يعاملهم معاملة الماكرين
( ۳۱ ) أساطير الأوّلين :أكاذيبهم المسطورة في كتبهم
( ۳۵ ) مُكاءً و تصدية :صفيرا و تصفيقا
( ۳٦ ) حسرة :ندما و تأسّفا
( ۳۷ ) فيركمه جميعا :فيجمعه ملقى بعضه على بعض
( ۳۸ ) سُنّة الأوّلين :عادة الله في المُكذّبين لرسله
( ۳۹ ) فتنة :شرك أو بلاء
( ٤۱ ) لله خُمسه :و الأربعة الأخماس للغانمين
( ٤۱ ) يوم الفرقان :بين الحقّ و الباطل (يوم بدر)
( ٤۲ ) بالعدوة الدّنيا :بحافّة الوادي و ضفّته الأقرب للمدينة
( ٤۲ ) الرّكب :عير قريش فيها أموالهم
( ٤۳ ) لَفشلتهم :لجُبنتُم عن القتال و هبتموه
( ٤٦ ) تذهب ريحكم :تتلاشى قوّتكم أو دولتكم
( ٤۷ ) بطرًا :طغيانًا أو فخرا و أشرّا
( ٤۸ ) إنِّي جار لكم :مُجير و مُعين و ناصر لكم
( ٤۸ ) نكص على عقبيه :رجع القهقري و ولّى مُدبرا
( ۵۲ ) كدأب :كعادة . .
( ۵۷ ) تثقفنّهم :تصادفنّهم و تظفرنّ بهم
( ۵۷ ) فشرِّد بهم :ففرّق و بدّد و خوّف بهم
( ۵۸ ) من قوم :قد عاهدوك
( ۵۸ ) فانبذ إليهم :فاطرح إليهم عهدهم و حاربهم
( ۵۸ ) على سواء :على استواء في العلم بنبذه
( ۵۹ ) سبقوا :خلصوا و أفلتوا من العذاب
( ٦۰ ) قوّة :كل ما يُتقوّى به في الحرب
( ٦۰ ) رباط الخيل :حبسها للجهاد في سبيل الله
( ٦۱ ) جنحوا للسّلم :مالوا للمُسالمة و المصالحة
( ٦۲ ) حسبك الله :كافيك في دفع خديعتهم
( ٦۵ ) حرِّض المؤمنين :بالغ في حثّهم
( ٦۷ ) يُثخن :يُبالغ في القتل حتى يذل الكفر
( ٦۷ ) عرض الدّنيا :حطامها بأخذكم الفدية
( ۷۱ ) فأمكن منهم :فأقدرك عليهم يوم بدر
( ۷۵ ) أولوا الأرحام :ذووا القربات
( ۷۵ ) أوْلى :بالميراث من الأجانب
