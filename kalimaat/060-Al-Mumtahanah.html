٦۰ - سورة الممتحنة
أولياء ( ۱ )
أعوانا توادونهم و تناصحونهم
أن تؤمنوا ( ۱ )
لإيمانكم أو كراهة إيمانكم
يثقفوكم ( ۲ )
يَظفروا بكم . أو يصادفوكم
يبسطوا إليكم ( ۲ )
يمدّوا إليكم
أسروة حسنة ( ٤ )
قدوة حميدة في التـَّـبَـرّي من الضالين
بُرَآءُ منكم ( ٤ )
أبرياء منكم
إليك أنـَــبْنا ( ٤ )
إليك رَجَعْنا تائبين
لا تجعلنا فـتنة ( ۵ )
مفتونين بهم معذبين بأيديهم
تبرّوهم ( ۸ )
تحْسنوا إليهم و تكرموهم
تـُـقسِطوا إليهم ( ۸ )
تـُـفـضوا إليهم بالقسْط و العدل
ظاهروا ( ۹ )
عاوَنوا الذين قاتلوكم و أخرجوكم
أن توَلّوْهمْ ( ۹ )
أن تتخذوهم أولياء
فامْتـَـحنوهنّ ( ۱۰ )
فاختبروهن و كان ذلك بالتحْـليف
أجورهنّ ( ۱۰ )
مُهورهنّ
بعِصم الكوافر ( ۱۰ )
بعقود نكاح المشركات
فاتكم شيء ( ۱۱ )
انفلتَ أحدٌ بردّة
فعاقبتمْ ( ۱۱ )
فَغَزوتم فغَنمتم منهمْ
ببُـهتان ( ۱۲ )
بإلصاق اللّـقـطاء بالأزواج
يفترينه ( ۱۲ )
يَختـَـلِقـنه
لا تتولوْا ( ۱۳ )
لا تتخذوا أولياء
قوْمًـا ( ۱۳ )
هم اليَهود ، أو الكفار عامّة
