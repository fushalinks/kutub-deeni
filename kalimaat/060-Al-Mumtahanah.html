٦۰ - سورة الممتحنة
( ۱ ) أولياء
أعوانا توادونهم و تناصحونهم
( ۱ ) أن تؤمنوا
لإيمانكم أو كراهة إيمانكم
( ۲ ) يثقفوكم
يَظفروا بكم . أو يصادفوكم
( ۲ ) يبسطوا إليكم
يمدّوا إليكم
( ٤ ) أسروة حسنة
قدوة حميدة في التـَّـبَـرّي من الضالين
( ٤ ) بُرَآءُ منكم
أبرياء منكم
( ٤ ) إليك أنـَــبْنا
إليك رَجَعْنا تائبين
( ۵ ) لا تجعلنا فـتنة
مفتونين بهم معذبين بأيديهم
( ۸ ) تبرّوهم
تحْسنوا إليهم و تكرموهم
( ۸ ) تـُـقسِطوا إليهم
تـُـفـضوا إليهم بالقسْط و العدل
( ۹ ) ظاهروا
عاوَنوا الذين قاتلوكم و أخرجوكم
( ۹ ) أن توَلّوْهمْ
أن تتخذوهم أولياء
( ۱۰ ) فامْتـَـحنوهنّ
فاختبروهن و كان ذلك بالتحْـليف
( ۱۰ ) أجورهنّ
مُهورهنّ
( ۱۰ ) بعِصم الكوافر
بعقود نكاح المشركات
( ۱۱ ) فاتكم شيء
انفلتَ أحدٌ بردّة
( ۱۱ ) فعاقبتمْ
فَغَزوتم فغَنمتم منهمْ
( ۱۲ ) ببُـهتان
بإلصاق اللّـقـطاء بالأزواج
( ۱۲ ) يفترينه
يَختـَـلِقـنه
( ۱۳ ) لا تتولوْا
لا تتخذوا أولياء
( ۱۳ ) قوْمًـا
هم اليَهود ، أو الكفار عامّة
