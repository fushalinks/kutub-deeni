۱۰۲ - سورة التكاثر
ألهاكم ( ۱ )
شَغَلَكم عنْ طاعَة ربّكم
الـتـّـكاثر ( ۱ )
التـّـباهي بكثرة متاع الدنيا
زُرْتم المقابر ( ۲ )
متـّـمْ و دُفِنـنـْـتم في القبور
لو تعلمون عِلم اليقين ( ۵ )
لوْ تعلمون مآلكم عِلما يَقـينا لـَـمَـا ألهاكم التـّـكاثر
لترونّ الجحيم ( ٦ )
و الله لترَوُنّ الجحيم
عيْن اليقين ( ۷ )
نـَـفـْـسَ اليَقين و هو المشاهدة
النّعيم ( ۸ )
الذي ألهاكمْ عنْ طاعة رَبّكمْ
