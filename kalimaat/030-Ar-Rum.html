۳۰ - سورة الروم
( ۲ ) غُلبت الروم :قهرت فارس الرّوم
( ۳ ) أدْنى الأرض :أقرب أرض الرّوم إلى فارس
( ۳ ) غَلَبهم :كونهم مغلوبين
( ۸ ) أجل مسمّى :وقت مُقدّر أزلا لِبقائها
( ۹ ) أثاروا الأرض :حرَثوها و قلـّـبوها للزّراعة
( ۱۰ ) السّوآى :العقوبة المُتناهية في السّوء (النار)
( ۱۲ ) يُبلسُ المجرمون :تنقطع حجّتهم . أو يَيْأسون
( ۱۵ ) يُحبرون :يُسرّون . أو يُكْرَمون
( ۱٦ ) في العذاب مُحضرون :لا يغيبون عنه أبدا
( ۱۸ ) حين تُظهرون :تدخلون في وقت الظهيرة
( ۲۰ ) تنتشرون :تتصرفون في شؤون معايشِكم
( ۲۱ ) لتسكنوا إليها :لِتميلوا إليها و تألفوها
( ۲٦ ) له قانتون :مُطيعون مُنقادون لإرادته
( ۲۷ ) له المثل الأعلى :الوصف الأعلى في الكمال و الجلال
( ۳۰ ) فأقم وجهك :قوّمْهُ و عدّلْهُ
( ۳۰ ) للدّين :دين التوحيد و الإسلام
( ۳۰ ) حنيفا :مائلا إليه مُستقيما عليه
( ۳۰ ) فِطرة الله :الزموها و هي دين الإسلام
( ۳۰ ) فطَر النّاس عليها :جبَلهم و طَبَعهم عليها
( ۳۰ ) لخلق الله :لدينه الذي فطرهم عليه
( ۳۰ ) ذلك الدّين القيّم :المُستقيم الذي لا عِوَج فيه
( ۳۱ ) مُنيبين إليه :راجعين إليه بالتـّــوبة و الإخلاص
( ۳۲ ) كانوا شِـيَعا :فِرقا مُختلفة الأهواء
( ۳۵ ) سُلطانا :كتابا أو حُجّة
( ۳٦ ) فرحوا بها :بَطِروا و أشِروا
( ۳٦ ) هم يقنطون :ييْأسون من رحمة الله تعالى
( ۳۷ ) يقدِر :يُضيّقه على من يشاء لحكمة
( ۳۹ ) ربًا :هو الرّبا المُحرّم المعروف
( ۳۹ ) لِيرْبوَ :لِيزيد ذلك الرّبا
( ۳۹ ) فلا يربوَ :فلا يزكو و لا يُبارك فيه
( ۳۹ ) المضعفون :ذوو الأضعاف من الحسنات
( ٤۳ ) للدّين القيّم :المستقيم (دين الفطرة)
( ٤۳ ) لا مردّ له :لا يقدِر أحدٌ على ردّه
( ٤۳ ) يصّـدّعون :يتفرّقون إلى الجنّة و إلى النّار
( ٤٤ ) يمهدُون :يُوطِّـئون مواطن النّعيم
( ٤۷ ) فتُثير سحابا :تحرّكُهُ و تنشره
( ٤۷ ) يجعله كِسفا :قِطعا مُتفرّقة
( ٤۷ ) الودق :المطر
( ٤۷ ) من خلاله :فُـرَجه و وسطه
( ٤۹ ) لَمبلسين :آيسين من نزوله
( ۵۱ ) فرأوهُ مُصفرّا :فرَأوُا النّبات مُصفرّا بعد الخُضرة
( ۵٤ ) شيبة :حال الشّيخوخة و الهرم
( ۵۵ ) يُؤفكون :يُصرفون عن الحقّ و الصّدق
( ۵۷ ) و لا هم يستعتبون :لا يُطلب منهم إزالة عتـْــبه و غَضَبـِــه تعالى عليهم – بالتّوبة و الطّاعة
( ٦۰ ) لا يستخفّـنـّـك :لا يحْمِلنّك على الخفّة و القلق
