( ۳۲ ) طيّبين :طاهرين من دنسالشّرك و المعاصي
( ۳٤ ) حاق بهم :أحاط . أو نزل بهم
( ۳٦ ) اجتنبوا الطّاغوت :كلّ معبود باطل و كلّ داع إلى ضلالة
( ۳٦ ) حقّت :ثبتت و وجبت

( ٤۱ ) لنُبوّئنّهم :لننزلنّهم

( ٤۷ ) تخوّف :مخافة من العذاب . أو تَنَقُّص

( ۵٦ ) تفترون :تكذبونه على الله

( ٦۲ ) لا جرَمَ :حقّ و ثبت . أو لا محالة أو حقّا
( ٦۲ ) مُفرطون :مُقدّمون معجّلٌ بهم إلى النار
( ٦٦ ) لعبرةً :لعظة عظيمة و دلالة على قدرتنا
( ٦٦ ) فرث :ما في الكَرش من الثُّــفْـلِ
( ٦۷ ) سكرًا :خمرا (ثمّ حُرّمت بالمدينة)

( ۷٦ ) هو كلٌّ :عبءٌ و عِيال
( ۷۷ ) كلمح البصر :كخطفة بالبصر و اختلاس بالنّظر
( ۸۰ ) تستخفونها :تجدونها خفيفة الحمل
( ۸۰ ) يوم ظعنكم :وقت ترحالكم
( ۸۰ ) أثاثا :متاعا لبُيوتكم كالفرش

( ۸۱ ) ظلالا :أشياء تستظلون بها كالأشجار
( ۸۱ ) أكنانا :مواضع تستكِنّون فيها (الغيران)
( ۸۱ ) سرابيل :ما يُلبس من ثياب أو دروع
( ۸۱ ) تقيكم بأسكم :الضّرب و الطّعن في حروبكم
( ۸٤ ) لا هو يستعتبون :لا يُطلب منهم إرضاءُ ربّهم
( ۸۵ ) يُنظرون :يُمهَلون و يؤخّرون
( ۸۷ ) السّلَم :الإستسلام و الإنقياد لحكمه تعالى
( ۹۰ ) يأمر بالعدل :بالإعتدال و التّوسّط في الأمور اعتقادا و عملا و خُلُقا
( ۹۰ ) الإحسان :إتقان العمل . أو نفعِ الخلق
( ۹۰ ) الفحشاء :الذنوب المفرطة في القُبح
( ۹۰ ) البغي :التّطاول و التّجبّر على النّاس
( ۹۱ ) كفيلا :شاهدا . رقيبا . ضامِنا
( ۹۲ ) قوّة :إبرام و إحكام
( ۹۲ ) أنكاثا :أنقاضا محلول الفتل

( ۱۰۰ ) يتولّونه :يتّخذونه وليّا مُطاعا
( ۱۰۲ ) روح القدس :الرّوح المطهّر جبريل عليه السلام
