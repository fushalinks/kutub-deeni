۷۲ - سورة الجن
( ۱ ) قرآنا عَجَـبا :عجبًـا بديعا في بلاغته و فصاحته
( ۲ ) الرّشد :الحقّ و الصّــواب . أو التوحيد و الإيمان
( ۳ ) تعالى :ارتفعَ و عَــُظمَ
( ۳ ) جدّ ربّـنا :جلاله . أو سلطانه . أو غِـنـَـاه
( ٤ ) يَقول سَفـيهُـنا :جَـاهِـلنا (إبليس اللعين)
( ٤ ) شططا :قوْلاً مُـفرطًـا في الكذب و الضّـلال
( ٦ ) يَعوذون :يَسْـتعيذون و يَتحيّـرون
( ٦ ) فزادوهمْ رَهَـقـا :إثما . أو طغيانا و سَفَها
( ۸ ) حَرَسًـا شديدا :حُرّاسا أقوياء من الملائكة
( ۸ ) شُهُـبًـا :شُعل نار تـَـنـْـقـَـضّ كالكواكب
( ۹ ) شِـهابًـا رَصَدًا :راصِدًا ، مُـترقّـبا يَرْجُمه
( ۱۰ ) رَشَـدًا :خَيْرًا و صَلاحًا و رحمة
( ۱۱ ) طرائق قِدَدًا :ذوي مذاهب مُتفرقة مُختلِفة
( ۱۲ ) ظنـنّـا :عَـلِـمْـنا و أيقـنّـا الآن
( ۱۳ ) فلا يَخاف بخسا :فلا يَخشى نـَـقـْصًـا منْ ثوَابه
( ۱۳ ) و لا رهقا :غَـشَيَـان ذِلـّـة له
( ۱٤ ) منّـا القاسطون :الجَـائرون بكفرهم العادلون عنْ طريق الحقّ
( ۱٤ ) تـَـحرّوا رشدا :قـَـصَـدوا خَيرًا و صلاحا و هُـدًى
( ۱۵ ) لجهنّم حطبا :للنّـار وقودًا
( ۱٦ ) على الطريقة :طريقة الهدى "ملـّة الإسلام
( ۱٦ ) ماءً غَدَقـًـا :كثيرًا يتـّـسِع بع العيش
( ۱۷ ) لنفـتِنـَهم فيه :لِنـَـخْـتـَـبرَهم فيما أعْطيناهم
( ۱۷ ) يَـسْـلكه :يُدخلـْـه
( ۱۷ ) عَذابًا صَعَدًا :شاقّـا يعلوه و يَغلبُـه فلا يُطيقه
( ۱۹ ) عَـبْدُ الله يَدْعـوه :هو النبي صلى الله عليه و سلم
( ۱۹ ) عَـليْه لِبَدًا :مُـتـَـراكِمِين من ازدحامهم عليه تعجّـبا
( ۲۱ ) رَشَدًا :نفعا أو هداية
( ۲۲ ) لنْ يجيرني من الله :لنْ يمنعَني من عذابه إنْ عَصَـيتـُـه
( ۲۲ ) مُلتـَـحَـدا :مَلْجَـأ أوْ حِرْزًا أرْكَنُ إليه
( ۲۵ ) أمَدًا :زمنا بعيدا
( ۲۷ ) رَصَدًا :حَرَسا من الملائكة يحرسونه
( ۲۸ ) أحاط :عَلِمَ عِـلمًـا تامّـا
( ۲۸ ) أحْصَى :ضَـبَـط ضبطا كاملا
