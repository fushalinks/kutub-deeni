۵٦ - سورة الواقعة
( ۱ ) وقعت الواقعة :قامت القيامة بنفخة البعْث
( ۲ ) كاذبة :نفْسٌ كاذبة تـُـنـْـكر وقوعها
( ۳ ) خافضة رافعة :هي خافضة للأشقياء رافعة للسّعداء
( ٤ ) رُجّت الأرض :زلزلتْ وحُرّكت تحريكا بشدة
( ۵ ) بُسّت الجبال :فـتـّـتْ كالسّويق المَـلْـتوت
( ٦ ) هباءً منبثـّـا :غبارًا متفرقا منتشِرا
( ۷ ) كنتم أزواجا :أصنافا
( ۸ ) فأصحاب الميمنة :اليُـمْن و البركة . أو ناحية اليمين
( ۹ ) أصحاب المشأمة :الشؤم . أو ناحية الشمال
( ۱۳ ) ثـلـّـة :هم أمّة من الناس كثيرة
( ۱۵ ) سُرُر موضونة :منسوجة من الذهب بإحكام
( ۱۷ ) ولدان مُخلّدون :مُبْـقون على هيئة الولدان في البهاء
( ۱۸ ) بأكواب :أقداح لا عُرى لها و لا خراطيم
( ۱۸ ) أباريق :أوان لها عُرى و خراطيم
( ۱۸ ) كأس :خمْر أو قدَح فيه خمْر
( ۱۸ ) من معين :خمْر جارية من العيون
( ۱۹ ) لا يُصدّعون عنها :لا يُصيبهم صداع بشربها
( ۱۹ ) لا يُنزفون :لا تذهَبُ عقولهم بسببها
( ۲۲ ) حورٌ عين :نساء بيض واسعات الأعين حِسانُها
( ۲۳ ) اللؤلؤ المكنون :المصون في أصدافه مما يغيّره
( ۲۵ ) لغْوًا :كلاما لا خير فيه أو باطلا
( ۲۵ ) و لا تأثيما :ولا نسبة إلى الإثم أو لا ما يوجبه
( ۲۸ ) في سِدْر :في شجر النـّـبَـق ينعّمون به
( ۲۸ ) مخضود :مقطوع شوكه
( ۲۹ ) طلح :شجر الموز أو مثله
( ۲۹ ) منضود :نضّد بالحمْل من أسفله إلى أعلاه
( ۳۰ ) ظل ممدود :دائم لا يتقلّص أو ممتدّ منبسط
( ۳۱ ) ماءٍ مسكوب :مصبوب يجري في غير أخاديد
( ۳٤ ) مرفوعة :على الأسرّة أو منضدّة مرتفعة
( ۳۷ ) عُرُبا :متحبّبات إلى أزواجهنّ
( ۳۷ ) أترابا :مسْتويات في السّنّ
( ٤۲ ) سموم :ريح شديدة الحرارة تدخل المَسام
( ٤۲ ) حميم :ماءٍ بالغ غاية الحرارة
( ٤۳ ) يَحْموم :دخان شديد السّواد أو نار
( ٤٤ ) لا كريم :لا نافع من أذى الحرّ
( ٤۵ ) مُترَفين :منعّمين متّبعين أهواء أنفسهم
( ٤٦ ) الحِنث :الذنب العظيم – الشرك
( ۵۲ ) زقوم :شجر كريه جدّا في النار
( ۵۵ ) شرب الهيم :الإبل العِطاش التي لا تروى
( ۵٦ ) هذا نزلهم :ما أعدّ لهم من الجزاء
( ۵٦ ) يوم الدين :يوم الجزاء ( يوم القيامة)
( ۵۸ ) أفرأيتم :أخبروني
( ۵۸ ) ما تـُـمْـنون :المنيّ الذي تقذفونه في الأرحام
( ۵۹ ) تخلقونه :تصوّرونه بشرًا سويّا
( ٦۰ ) بمسبوقين :بمغلوبين عاجزين
( ٦۳ ) ما تحرثون :البَذر الذي تلقونه في الأرض
( ٦٤ ) تزرعونه :تنبتونه حتى يشتدّ و يبلغ الغاية
( ٦۵ ) حُطاما :هشيما متكسّرا لا يُنتفع به
( ٦۵ ) تفكّهون :تتعجّبون من سوء حاله و مصيره
( ٦٦ ) إنّـا لمُغرمون :مُهلكون بهلاك رزقنا
( ٦۷ ) محرومون :ممنوعون الرّزق بالكلّـيّـة
( ٦۹ ) المُـزن :السّحاب أو الأبيض منه
( ۷۰ ) جعلناه أجاجا :مِلحًا زُعاقا أو مُرّا لا يُمكن شربه
( ۷۱ ) النّار التي تورون :تقدحون الزّناد لاستخراجها
( ۷۳ ) تذكرة :تذكيرا لنار جهنّم
( ۷۳ ) متاعا للمقوين :مَنفعَة للمسافرين في القـَـوَاءِ (القـَـفـْـر) أو المُحتاجين إليها
( ۷۵ ) فلا أقسم :فأقسِم و "لا" مزيدة للتأكيد
( ۷۵ ) بمواقع النجوم :بمغاربها . أو منازلها
( ۷۷ ) إنّه لقرآن كريم :نـفّـاعٌ جمّ المنافع . أو رفيع القدْر
( ۷۸ ) كتاب مكنون :مستور مصون عند الله في اللوح المحفوظ من السّوء
( ۷۹ ) لا يَمسّه إلا المطهّرون :صفة أخرى للقرآن
( ۸۱ ) أنتم مُدهنون :مُتهاونون أو مكذبون
( ۸۲ ) تجعلون رزقكم :شكركم على الإنعام به
( ۸۳ ) بلغت الحلقوم :بلغت الرّوح الحلقوم عند المَوْت
( ۸۵ ) نحن أقرب إليه :بعِلمِنا و قدْرَتنا
( ۸٦ ) غير مَدينين :غير مربوبين مقهورين
( ۸۹ ) فرَوْح :فله استراحة أو رحمة
( ۸۹ ) ريحان :رزق حسن
( ۹۳ ) فـنُـزُلٌ :فله قِرًى و ضِيافة
( ۹۳ ) حميم :ماءٍ تناهت حرارته
( ۹٤ ) تصلِية جحيم :مُقاسات لحرّ النار أو إدخال فيها
