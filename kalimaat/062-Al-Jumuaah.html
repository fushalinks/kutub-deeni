٦۲ - سورة الجمعة
( ۱ ) يسبّح لله
يُنـزّهه و يُمَجّده تعالى و يدلّ عليه
( ۱ ) الملك
مالك الأشياء كلّها
( ۱ ) القدوس
البليغ في النّـزاهة عن النّـقائص
( ۱ ) العزيز
القادر الغالب القاهر
( ۲ ) الأميين
العرب المعاصرين له صلى الله عليه و سلم
( ۲ ) يزكّيهم
يُطهّرُهم من أدناس الجاهليّة
( ۳ ) أخرين منهم
من العَرَب
( ۳ ) لمّـا يَلحقوا بهم
لمْ يلحقوا بهمْ بَعد و سيلحقون
( ۵ ) حُـمّـلوا التوراة
كُـلّـفوا العمل بما فيها (اليهود)
( ۵ ) يَحْمِل أسفارًا
كُتـبا عظاما و لا ينتفع بها
( ٦ ) هَادوا
تديّـنوا باليهودية
( ۹ ) ذروا البيْع
اتركوه و تفرّغوا لذكر الله
( ۱۰ ) فانتشروا
تفـرّ قوا للتـّـصرف في حَوَائجكم
( ۱۱ ) انفضّوا إليها
تفرقوا عنك قاصدين إليها
