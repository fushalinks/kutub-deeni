۵۹ - سورة الحشر
( ۱ ) سبّـح لله :نزّهَه و مَجّـدَهُ تعالى و دلّ عليه
( ۲ ) الذين كفروا :هو يهود بني النضير قرب المدينة
( ۲ ) لأوّل الحَشر :في أول إخراج و إجلاء إلى الشام
( ۲ ) فأتاهم الله :فأتاهم أمره و عقابه
( ۲ ) لمْ يحتسِبوا :لم يظنوا و لم يخطر لهم ببَال
( ۲ ) قذف :ألقى و أنزل إنزالا شديدا
( ۳ ) الجَلاء :الخروج من الوَطن
( ٤ ) شاقّوا :عادَوْا و عَصَوْا و حادّوا
( ۵ ) لِـينة :نخلة . أو نخلة كريمة
( ۵ ) على أصولها :على سوقها
( ٦ ) و ما أفاء الله :و ما ردّ و ما أعادَ
( ٦ ) فما أوْجَفتم عليه :فما أجْرَيتم على تحصيله
( ٦ ) ركاب :ما يُركب ن الإبل خاصّة
( ۷ ) دُولة بين الأغنياء :مِلكا مُتداولا بينهم خاصة
( ۹ ) تـبَـوّءوا الدار و الإيمام :تـَـوَطّـنوا المدينة و أخلصوا الإيمان
( ۹ ) حاجَة :حَزَازَةً و حَسَدًا
( ۹ ) خصاصة :فقـْـرٌ و احتياج
( ۹ ) منْ يوق :منْ يُجَنّبْ و يُـكْـفَ
( ۹ ) شحّ نفسه :بُخْـلها مع الحرص على المنع
( ۱۰ ) غِلاّ :حِقـدًا و بُـغـضـا و غِـشّـا
( ۱٤ ) بَأسهم بينهم :قتالهم فيما بينهم
( ۱٤ ) قلوبهم شتـّـى :متفرقة لتـَـعَاديهمْ
( ۱۵ ) وَبال أمرهم :سوء عاقبة كفرهم
( ۱۹ ) نسوا الله :لمْ يراعوا أوامره و نـَـواهيه
( ۱۹ ) فأنـْـساهمْ أنفسهم :فلم يُقـدّموا لها ما ينفعها عنده
( ۲۱ ) خاشِعًا :ذليلا خاضعا
( ۲۱ ) متصدّعًا :مُـتـشـقـّــقـا
( ۲۳ ) المَلِك :المالك لكل شيء المتصرف فيه
( ۲۳ ) القدوس :البَليغ في النـّـزاهة عن الـنّـقائص
( ۲۳ ) السلام :ذو السّـلامة منْ كل عيّب و نقـْص
( ۲۳ ) المؤمن :المصَدّق لرسلِه بالمعجزات
( ۲۳ ) المهيمن :الرقيب على كلّ شيء
( ۲۳ ) العزيز :القوّي الغالب
( ۲۳ ) الجبّار :القهار . أو العظيم
( ۲۳ ) المتكبّر :البَليغ الكبرياء و العظمة
( ۲٤ ) البارئ :المبدع المخترع
( ۲٤ ) المصوّر :خالق الصّور على ما يريد
( ۲٤ ) الأسماء الحسنى :الدّالة على محاسن المعاني
