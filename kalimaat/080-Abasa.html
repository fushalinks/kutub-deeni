۸۰ - سورة عبس
( ۱ ) عَـبَس
قطّـبَ وَجْهه الشريف صلى الله عليه و سلم
( ۱ ) توَلـّـى
أعرَض بوجهه الشريف صلى الله عليه و سلم
( ۳ ) لعَلـّـه يَزكّى
يَـتطهّر بتـَـعليمِك مِنْ دَنـَـس الجهْل
( ٤ ) يَذكّـر
يـتـّـعِـظ
( ٦ ) له تـَـصدّى
تتعرّض له بالإقـبال عليه
( ۸ ) جَاءك يَسْعى
وَصَـل إليك مُـسْرعا ليَتـَـعلـّـم
( ۱۰ ) عنه تـَـلـهّـى
تتلهـّى- تتشاغَـل و تـُـعْرض
( ۱۱ ) كلاّ
حقـّـا أو إرْشادٌ ، بليغٌ لترْك المُـعاوَدَة
( ۱۱ ) إنـّـها تذكِرَة
إنّ آيات القرآن مَوْعِـظة و تذكيرٌ
( ۱۳ ) في صُحف
منتسخةٍ من اللوح المحفوظ
( ۱٤ ) مَرْفوعة
رَفيعة الـقـَـدْر و المَنزلة عنده تعالى
( ۱۵ ) بأيدي سَفـرة
ملائكة ينسخونها من اللوح المحفوظ
( ۱٦ ) بَرَرة
مُطيعين له تعالى أو صادقين
( ۱۷ ) قـُـتِل الإنسان
لُعِن الكافـر . أو عُـذب
( ۱۹ ) فـَـقـدّره
أطوارا أو هيّـأه لما يَصْـلـُـح له
( ۲۰ ) السّبيل يسّره
سَهّـل له طريقي الهدى و الضّلال
( ۲۱ ) فأقـبَرَه
أمَرَ بدفـنِه في قـبْر تكْرمة ً له
( ۲۲ ) أنـْـشَـره
أحْـيـَاه بَعْد موته
( ۲۳ ) لمّـا يَقـْـضِ ما أمره
لمْ يفعَلْ ما أمَرَه الله به بَلْ قـَـصّـر
( ۲٦ ) شقـقـنا الأرض
بالنّـبات أو بالحَرْث
( ۲۸ ) قضْـبًـا
عَـلفـًـا رَطْـبًـا للدّوابّ كالبَـرْسِيم
( ۳۰ ) حَدائق غُـلْـبا
بَساتين عِظامًـا مُتكاثفة الأشجار
( ۳۱ ) أبّـا
كلأ و عُـشْـبًا . أو هُوَ التـِّـبْن خاصّة
( ۳۳ ) جاءت الصاخّة
الصّـيْحَـة تـُـصِمّ الآذان لشِدتِها (النفخة الثانية)
( ۳۸ ) مُسْـفـِرَة
مُـشـْـرقة مُـضيئة (وجوه المؤمنين)
( ٤۰ ) غَـبَـرَة
غبارٌ وَ كُـدُورَة (وجوه الكافرين)
( ٤۱ ) تَرْهَـقها قـترة
تـَـغْـشاها ظُـلـْـمَـة و سواد
