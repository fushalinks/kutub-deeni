٦۱ - سورة الصف
( ۱ ) سَبّح لله . . :نـَـزّهَه و مَجّده تعالى و دلّ عليه
( ۳ ) كَبُرَ مَقـتا :عَظم بغضا بالغ الغاية
( ٤ ) صَفّا :صافّين أنفسهم أو مصفوفين
( ٤ ) بُنيانٌ مَرْصوص :متلاصق مُحكم لا فرجة فيه
( ۵ ) زَاغوا :مالوا باختيارهم عن الحقّ
( ۵ ) أزاغ الله قلوبهم :حَرَمَهم التوفيق لاتـّباع الحقّ
( ۸ ) نور الله :الحقّ الذي جاء به الرسول صلى الله عليه و سلم
( ۸ ) و أخرى :و لكم من النّعم نعمة أخرى
( ۱٤ ) للحواريين :أصْـفِـياء عيسى و خواصّه
( ۱٤ ) فأيّدنا :قوّينا المحقّين بالإيمان
( ۱٤ ) ظاهرين :غالبين بالحجج و البيانات
