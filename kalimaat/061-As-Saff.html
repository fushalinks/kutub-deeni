٦۱ - سورة الصف
سَبّح لله . . ( ۱ )
نـَـزّهَه و مَجّده تعالى و دلّ عليه
كَبُرَ مَقـتا ( ۳ )
عَظم بغضا بالغ الغاية
صَفّا ( ٤ )
صافّين أنفسهم أو مصفوفين
بُنيانٌ مَرْصوص ( ٤ )
متلاصق مُحكم لا فرجة فيه
زَاغوا ( ۵ )
مالوا باختيارهم عن الحقّ
أزاغ الله قلوبهم ( ۵ )
حَرَمَهم التوفيق لاتـّباع الحقّ
نور الله ( ۸ )
الحقّ الذي جاء به الرسول صلى الله عليه و سلم
و أخرى ( ۸ )
و لكم من النّعم نعمة أخرى
للحواريين ( ۱٤ )
أصْـفِـياء عيسى و خواصّه
فأيّدنا ( ۱٤ )
قوّينا المحقّين بالإيمان
ظاهرين ( ۱٤ )
غالبين بالحجج و البيانات
