٤٤ - سورة الدخان
( ۳ ) ليلة مباركة :ليلة القدر من شهر رمضان
( ٤ ) فيها يُفرق :يفصّل و يُـبَـيّـن
( ٤ ) أمرٍحكيم :محْكمٍ مبرم أو ملتبسٍ بالحكمة
( ۱۰ ) فارتقبْ :انتظر بهؤلاء الشاكّين
( ۱۰ ) بدخان :كناية عن إصابتهم بالجَدْب و المجاعة
( ۱۱ ) يغشى الناس :يشملهم و يحيط بهم
( ۱۳ ) أنّى لهم الذكرى؟ :كيف يتذكّرون و يتـّـعِظون ؟
( ۱٤ ) مُعلّم :يعلـّـمه بشر
( ۱٦ ) يوم نبطش :يوم نأخذ بشدّة و عنف (يوم بدر أو يوم القيامة)
( ۱۷ ) فتنّا :ابتلينا و امتحنّا
( ۱۸ ) أدّوا إليّ عباد الله :سلـّـموا إلي بني إسرائيل
( ۱۹ ) لا تعلوا :لا تتكبّروا . أو لا تفتروا
( ۱۹ ) بسلطان :حجّة و برهان على صدقي
( ۲۰ ) إنّي عذت بربّي :استجرت به و التجَأت إليه
( ۲۰ ) ترجمونِ :تؤذوني. أو تقتلوني بالحجارة
( ۲۳ ) فأسر بعبادي ليلا :سِـرْ ليلا ببني إسرائيل
( ۲۳ ) إنكم متبعون :يتبعكم فرعون و جنوده
( ۲٤ ) البحْرَ رهوا :ساكنـًـا . أو منفرجا مفتوحا
( ۲٤ ) جندٌ :جماعة
( ۲۷ ) نعمَةٍ :تنعّم أو نضارة عيشٍ و لذاذتِه
( ۲۷ ) فاكهين :ناعمين متفكّهين
( ۲۹ ) منظرين :ممهلين بالعّذاب إلى وقت آخر
( ۳۱ ) كان عاليا :متكبّرا جبّارا
( ۳۲ ) العالمين :عالمي زمانهم
( ۳۳ ) فيه بلاءٌ مبين :اختبار ظاهر أو نعمة ظاهرة
( ۳۵ ) بمُنشرين :بمبعوثين بعد موتتنا
( ۳۷ ) قوم تبّع :أبـِـي كَــِرب الحِمْـيَـــِريّ ملك اليمن
( ٤۰ ) يوم الفصْل :يوم القيامة و الحساب
( ٤۱ ) لا يغني مولى . . :لا يدفع قريب. و لا صديق
( ٤۳ ) شجرة الزقوم :من أخبث الشّجر تنبت في النّار
( ٤۵ ) كالمهل :دُرْديّ الزّيت . أو المعدن المُذاب
( ٤٦ ) الحميم :الماء البالغ غاية الحرارة
( ٤۷ ) فاعتلوه :فجرّوه بعنف و قهر
( ٤۷ ) سواء الجحيم :وَسَط النّار
( ۵۰ ) به تمترون :فيه تجادلون و تمارون
( ۵۳ ) سندس :رقيق الدّيباج
( ۵۳ ) إستبرق :غليظه
( ۵٤ ) زوّجناهم بحور عين :قرنّاهم بنساءٍ بيضٍ مخلوقات في الجنّة واسعات الأعين حِسَانُها
( ۵۵ ) يدعون فيها :يطلبون فيها
( ۵۹ ) فارتقب :فانتظر ما يحلّ بهم
( ۵۹ ) إنهم مرتقبون :منتظرون ما يحلّ بك
