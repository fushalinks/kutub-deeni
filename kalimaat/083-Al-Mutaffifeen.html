۸۳ - سورة المطففين
( ۱ ) وَيْـل :عَذابٌ أوْ هَلاك أو وادٍ في جهـنـّـم
( ۱ ) للمطـفـفين :المُـنقـِـصين في الكـيْـل ، و مثله الوزن
( ۲ ) اكتالوا :اشْتروْا بالكيل ، و مثله الوزن
( ۳ ) كالوهمْ :أعطَوْا غـيرهم بالكيل
( ۳ ) وَزنوهمْ :أعْطَوْا غـيرهم
( ۳ ) يُـخْـسِرون :يَنـقـصون الكيل و الوَزن
( ٦ ) لربّ العالمين :لأمره و حكمه
( ۷ ) كَـتاب الفجّـار :ما يُـكتب من أعمالهم
( ۷ ) لفي سجين :لمُـثـبَـتٌ في ديوان الشّر
( ۹ ) كتابٌ مَرقوم :بَـيّـن الكتابة أو معلـّـم بعلامة
( ۱۲ ) مُعْـتدٍ :فاجر مُتجاوز عن نـَـهْج الحقّ
( ۱۳ ) أساطير الأوّلين :أباطيلهم المسَـــّطرة في كتبهم
( ۱٤ ) كلاّ :رَدْعٌ و زجْرٌ عن قولهم الباطل
( ۱٤ ) رانَ على قـلوبهم :غَـلبَ و غَـــّطى عليها أو طبع عليها
( ۱٦ ) لصالوا الجحيم :لداخِلوها أو لمقاسو حَرّها
( ۱۸ ) كتاب الأبرار :ما يُكتب من أعمالهم
( ۱۸ ) لفي عليين :لمُثبت في ديوان الخيْر
( ۲۳ ) الأرائك :الأسِرّة في الحجال(جمع حجلة محركة – بيت يزيّن بالقباب و الأسرة و الستور)
( ۲٤ ) نضْرة النّعيم :بهجته و رونقه و بَهاءَه
( ۲۵ ) رَحيق :أجْوَد الخَمْر و أصْـفاه
( ۲۵ ) مَختوم :إناؤه حتى يَفـكّه الأبرار
( ۲٦ ) خِـتامه مسْـك :ختام إنائه المِسْـك بَدَل الــّطين
( ۲٦ ) فلـيـتنافس :فـلـيَـتـَـسارعْ . أو فـلـيَسْـتـَـبـِـقْ
( ۲۷ ) مِزَاجُه :ما يُـمْـزَجَ بهِ و يُـخْـلـَـط
( ۲۷ ) تسْـنيم :عَـيْـن ٍ عاليةٍ شَرابُـها أشرف شراب
( ۲۸ ) يَشرَب بها :يَشرَب منها
( ۳۰ ) يَغامَـزون :يُشيرون إليهم بلأعين استهزاء
( ۳۱ ) فـكهين :مُـتـلذذين باستِخفافهم بالمؤمنين
( ۳٦ ) ثـُـوّب الكـفار :جُوزُوا بسُخريّـتهم بالمؤمنين
