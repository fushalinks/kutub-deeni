٦۵ - سورة الطلاق
( ۱ ) فطلّـقوهن لعدتهنّ :مُستقـبلاتٍ لعدتهنّ (الــّطهْر)
( ۱ ) أحصوا العدّة :اضبطوها و أكمِلوها ثلاثة قروء
( ۱ ) بفاحشة مبَيّـنة :بمعصيةٍ كبيرةٍ ظاهرة
( ۲ ) بَلَغْنَ أجلهنّ :قاربْن انقضاء عدتهنّ
( ۲ ) مخرجا :من كلّ شدة و ضيق و بلاء
( ۳ ) لا يحتسب :لا يَخطر بباله و لا يكون في حسابه
( ۳ ) فهو حسبه :كافيه ما أهمّه في جميع أموره
( ۳ ) قـَـدْرًا :أجَلاً ينتهي إليه أو تقديرا أزَلاً
( ٤ ) يَـئِـسْنَ :انقـطع رَجاؤهنّ لِكِبَرهنّ
( ٤ ) و اللائي لم يَحِضن :لِصِغرهنّ عدتهن ثلاثة أشهر
( ٤ ) يُسْرا :تيْسيرا و فَـرَجًا
( ٦ ) وُجْدِكم :وُسْعِكم و طاقـَـتِكم
( ٦ ) ائتمِروا بيْنكم :تشاوَروا في الأجرة و الإرضاع
( ٦ ) تـَـعاسَرتم :تضايَـقـْـتم و تشاحنتم فيهما
( ۷ ) ذو سَعَة :غـنـًى و طاقة
( ۷ ) قـُـدِرَ عليه :ضُـيّـق عليه
( ۸ ) كأيّّـنْ مِنْ قـَـرية :كثيرٌ من أهل قرْية
( ۸ ) عـتـَـتْ :تجبّرت و تكبّرتْ و أعْرَضتْ
( ۸ ) عذابا نُـكرا :مُـنـْكَرًا شنيعا في الآخرة
( ۹ ) وَبال أمْرها :سوء عَاقـبة عُـتوّها
( ۹ ) خُـسرا :خسراناً و هَلاكاً
( ۱۰ ) ذِكْرا :قرآنا
( ۱۱ ) رَسولا :أرْسل رسولا ، أو جبريل
( ۱۲ ) يتنزّل الأمر :يَجْري قضاؤه و قـَـدَره أو تدبيره
