۳٦ - سورة يس
( ۷ ) لقد حقّ القول :و اللهِ لقد ثبت ووجب العقاب
( ۸ ) أغلالا :قيودا تشدّ أيديهم إلى أعناقهم
( ۸ ) فَهُم مُّقْمَحُونَ :رافعوا الرّءوس غاضّوا الأبصار
( ۹ ) سدّا :حاجزا و مانِعا
( ۹ ) فأغشيناهم :فألبسنا أبصارهم غشاوة
( ۱۲ ) آثارهم :ما سنّوه من حسن أو سيّء
( ۱۲ ) أحصيناه :أثبتناه و حفظناه
( ۱۲ ) إمام مبين :أصل بيّن (اللوح المحفوظ)
( ۱۳ ) القرية :أنطاكية
( ۱٤ ) فعززنا بثالث :فقوّيناهما و شددناهما به
( ۱۸ ) تطيّرنا بكم :تشاءمنا بكم
( ۱۹ ) طائركم معكم :شؤمكم كُفركم المصاحب لكم
( ۱۹ ) أئن ذكّرتم :أئن وُعِـظتم تطيّرتم
( ۲۰ ) يسعى :يُسرع في مشيه لنصح قومه
( ۲۲ ) فطرني :خلقني و أبدعني
( ۲۳ ) لا تغن عنّي :لا تدفع عنّي
( ۲۹ ) صيحة واحدة :صوتـًا مُهلكًا من السّماء
( ۲۹ ) خامدون :مَيّـتون كما تخمد النّار
( ۳۰ ) يا حسرة :يا وَيْلاً . أو يا تـَـنـَـدّماً
( ۳۱ ) كم أهلكنا :كثيرا أهلكنا
( ۳۱ ) القرون :الأمم
( ۳۲ ) لمّا جميع :إلا مجموعون
( ۳۲ ) مُحضرون :نحضرهم للحساب و الجزاء
( ۳٤ ) فجّرنا فيها :شققنا في الأرض
( ۳٦ ) خلق الأزواج :الأصناف و الأنواع
( ۳۷ ) نسلخ منه النّهار :ننزع من مكانه الضّوْء
( ۳۹ ) قدّرناه منازل :قـدّرنا سيره في منازل و مسافات
( ۳۹ ) كالعرجون القديم :كعود عِـذ ْق النّخلة العتيق
( ٤۰ ) ولا اللّيل :و لا آية الليل (القمر)
( ٤۰ ) سابق النّهار :سابق آية النهار (الشمس)
( ٤۰ ) يسبحون :يسيرون بانبساط أو يدورون
( ٤۱ ) ذرّيّتهم :أولادهم و ضعفاءَهم
( ٤۱ ) المشحون :المملوءِ المُوقر
( ٤۳ ) فلا صريخ لهم :فلا مغيث لهم من الغَرَق
( ٤۹ ) صيحة واحدة :نفخة الموت
( ٤۹ ) هم يخِصّمون :يخْتصِمون في أمورهم غافلين
( ۵۱ ) نفخ في الصّور :نفخة البَعْث
( ۵۱ ) الأجداث :القبور . .
( ۵۱ ) ينسلون :يُسرعون في الخروج
( ۵۳ ) صيحة واحدة :نفخة البعث
( ۵۳ ) مُحضرون :نحضِرُهم للحساب و الجزاء
( ۵۵ ) شُغُلٍ :نعيم عظيم يُلهيهمْ عمّا سواه
( ۵۵ ) فاكهون :مُتلذذون. أو فَرحون
( ۵٦ ) الأرائك :السّرر في الحِجال (جمع حجلة محركة- بيت يزين بالثياب و الأسرّة و الستور)
( ۵۷ ) لهم ما يدّعون :ما يتمنّونه أو ما يطلبونه
( ۵۹ ) امتازوا :تميّزوا و انفردوا عن المؤمنين
( ٦۰ ) أعْهَدْ إليكم :أوصِكمْ. أو أكلـّـفكمْ
( ٦۲ ) جـِــبـِـلاّ :خلقـًـا . أو جماعة عظيمة
( ٦٤ ) اصلوها :ادخلوها . أو قاسوا حرّها
( ٦٦ ) لطمسنا :لصيّرناها ممسوحة لا يُرى لها شق
( ٦٦ ) فاستبقوا الصّراط :ابْتـَـدروا الطّريق ليَجوزوه
( ٦٦ ) فأنّى يبصرون ؟ :فكيف يُبصرون الطريق؟
( ٦۷ ) على مكانتهم :في مكان معاصيهمْ
( ٦۸ ) من نعمّره :نطِلْ عُمُرَه
( ٦۸ ) ننكّسه في الخلق :نرُدّه إلى أرذل العمر
( ۷۲ ) ذلـّـلناها لهم :صيّرناها مسخّرة مُنقادة لهم
( ۷۵ ) و هم لهم جند مُحضرون :و الأصنام جُندٌ مُعَدّون للكفار نحضِرْهم معهم في النّار لعذابهم
( ۷۷ ) هو خصيم :مُبالغ في الخصومة بالباطل
( ۷۸ ) هي رميم :بالية أشدّ البلى
( ۸۱ ) بلى :هو قادر على خلق مثلهم
( ۸۳ ) ملكوت :هو المُـلـْـك التـّـام
