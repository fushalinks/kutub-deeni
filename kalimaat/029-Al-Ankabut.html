۲۹ - سورة العنكبوت
( ۲ ) لا يُفتنون :لا يُمتحنون بالمشاقّ و الشّدائد ليتميّز المُخلص من المنافق
( ٤ ) أن يسبقونا :أن يُعجزونا و يفوتونا
( ۵ ) أجل الله :الوقت المعيّن للبعث و الجزاء
( ۸ ) وصّينا الإنسان :أمرناه
( ۸ ) حسنا :برّا بهما و عطفا عليهما
( ۱۰ ) فتنة النّاس :ما يُصيبه من أذاهم و عذابهم
( ۱۲ ) خطاياكم :أوزاركم
( ۱۳ ) أثقالهم :خطاياهم الفادحة
( ۱۳ ) يفترون :يختلقونه من الأباطيل و الأكاذيب
( ۱۷ ) تخلقون إفكا :تكذبون أو تنحتون كذبا
( ۲۱ ) إليه تُقلبون :تـُــردّون و ترجعون لا إلى غيره
( ۲۲ ) بمعجزين :فائتين من عذابه بالهرب
( ۲۵ ) مودّة بينكم :للتوادّ و التّواصل بينكم لاجماعكم على عبادتها
( ۲۵ ) مأواكم النّار :منزلكم الذي تأوون إليه النّار
( ۲۹ ) تقطعون السّبيل :بمقارفة المعاصي و القبائح
( ۲۹ ) ناديكم :مجلسكم الذي تجتمعون فيه
( ۳۲ ) من الغابرين :من الباقين في العذاب كأمثالها
( ۳۳ ) سيء بهم :اعتراه الغمّ بمجيئهم خوفا عليهم
( ۳۳ ) ضاق بهم ذرعا :ضعُـفـتْ طاقته عن تدبير خلاصهم
( ۳٤ ) رجزا :عذابا شديدا
( ۳٦ ) لا تعثوْا :لا تُفسِدوا أشدّ الإفساد
( ۳۷ ) فأخذتهم الرّجفة :الزلزلة الشديدة بسبب الصّيحة
( ۳۷ ) جاثمين :هامدين ميّـتين لا حراك بهم
( ۳۸ ) كانوا مستبصرين :عقلاء متمكّنين من التدبّر
( ۳۹ ) سابقين :فائتين من عذابه تعالى
( ٤۰ ) حاصبا :ريحا عاصفا ترميهم بالحصباء
( ٤۰ ) أخذته الصّيحة :صوتٌ من السّماء مُهلك مُرجف
( ٤۱ ) العنكبوت :حشرة معروفة
( ۵۳ ) أجل مسمّى :هُو يوم القيامة
( ۵۳ ) بغتة :فجأة
( ۵۵ ) يغشاهم العذاب :يُـجلـّـلهم و يُحيط بهم
( ۵۸ ) لنبوأنّهم :لنُنزلنّهم على وجه الإقامة
( ۵۸ ) غرفا :منازل رفيعة عالية
( ٦۰ ) كأيّ من دابّة :كثيرٌ من الدّواب
( ٦۱ ) فأنى يُؤفكون؟ :فكيف يُصرفون عن توحيده ؟
( ٦۲ ) يقدر له :يُضيّـقه على من يشاء لحكمة
( ٦٤ ) لهو و لعب :لذائد مُتصرّمة ، و عبث باطل
( ٦٤ ) لَهي الحيوان :لَهِيَ دار الحياة الدّائمة الخالدة
( ٦۵ ) الدّين :العبادة و الطّاعة
( ٦۷ ) يُتخطـّــف النّاس :يُسلبون قتـْـلا و أسرا
( ٦۸ ) مثوا للكافرين :مكانٌ يَثـْوُون فيه و يقيمون
