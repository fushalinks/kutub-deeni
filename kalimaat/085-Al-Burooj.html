۸۵ - سورة البروج
( ۱ ) و السماء
(أَقسَمَ) اللهُ بها وبما بعدها
( ۱ ) ذات البروج
ذات المنازلِ المعروفة للكواكب
( ۲ ) اليوم الموعود
يومِ القيامة
( ۳ ) شاهد
من يَشهدُ على غيرِه فيه
( ۳ ) مشهودٍ
من يَشهدُ عليه غيرُه فيه
( ٤ ) قُـتِـل
لقد لُعِنَ أشدَّ اللَّعْنِ (جواب القسم)
( ٤ ) الأخدود
الشِّقِّ العظيم ؛ كالخَنْدَق
( ۸ ) ما نقموا
ما كَرِهُوا وما عابُوا وما أنكَروا
( ۱۰ ) فـَـتَـنوا
عَذبُوا أو أَحرَقُوا
( ۱۲ ) بَطْش ربّـك
أخْذَه الجبابرةَ والظلمةَ بالعذاب
( ۱۳ ) هو يُبدئ
يَخلُقُ ابتِداءً بقدرته
( ۱۳ ) يُعيد
يَبعثُ الموتى يوم القيامة بقدرته
( ۱٤ ) الوَدود
المُتودِّدُ إلى أوليائِه بالكرامة
( ۱۵ ) المجيد
العظيمُ الجليلُ المُتعالِى
