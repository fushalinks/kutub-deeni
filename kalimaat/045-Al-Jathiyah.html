٤۵ - سورة الجاثية
( ٤ ) يبثّ :يَنشر و يفرّق
( ۵ ) تصريف الرياح :تقليبها في مهابّها و أحوالها
( ۷ ) ويْلٌ :هلاكٌ ، أو حَسرة أو شدة عذاب
( ۷ ) أفّاك أثيم :كذاب كثير الإثم
( ۹ ) اتـّـخذها هزوا :سخريّة أو مهزوءً بها
( ۱۰ ) لا يغني عنهم . . :لا يدفع عنهم . .
( ۱۱ ) رجْزٌ :أشدّ العذاب
( ۱٤ ) لا يرجون أيام الله :لا يتوقعون وقائعه بأعدائه
( ۱۷ ) يغيا بينهم :حسدا و عداوة بينهم
( ۱۸ ) شريعة من الأمر :طريقة و منهاج من أمر الدين
( ۱۹ ) لن يغنوا عنك :لن يدفعوا عنك
( ۲۰ ) بصائر للناس :بيّـنات تبصّرُهم سبيل الفلاح
( ۲۱ ) اجترحوا السيئات :اكتسبوا المعاصي و الكفر
( ۲۳ ) أفرأيت :أخبرني
( ۲۳ ) غشاوة :غطاءً حتى لا يُـبْـصِرَ الرّشد
( ۲۸ ) جاثية :باركة على الرّكب لشدة الهوْل
( ۲۸ ) كتابها :صحائف أعمالها
( ۲۹ ) نستنسخ . . :نأمر الملائكة بنسخ . .
( ۳۳ ) حاقَ بهم :نزَل أو أحاط بهم
( ۳٤ ) ننساكم :نترككم في العذاب
( ۳٤ ) مأواكم النار :منزلكم و مقرّكم النّـار
( ۳۵ ) غرتكم . . :خدعتكم ببهرَجها . .
( ۳۵ ) يستعتبون :يُطلب منهم الرجوع إلى ما يرضي الله
( ۳۷ ) له الكبرياء :العظمة و الملك و الجلال
