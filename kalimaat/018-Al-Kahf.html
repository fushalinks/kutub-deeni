۱۸ - سورة الكهف
( ۱ ) لم يجعل له عوجا :اختلالا و لا اختلافا و لا انحرافاعن الحقّ و لا خروجا عن الحكمة
( ۲ ) قيِّما :مُستقيما معتدلا أو بمصالح العباد
( ۲ ) بأسا :عذابًا آجلا أو عاجلا
( ۵ ) كبُرت كلمة :ما أعْظمَها في القُبح كلمة
( ٦ ) باخع نفسك :قاتِلها و مُهلِكها أو مُجهدُها
( ٦ ) أسفا :غصبا. و حُزنا عليهم أو غيظا
( ۷ ) لِنبلوهم :لِنختبرَهم مع علمنا بحالهم
( ۷ ) أحسن عملا :أزهد فيها و أسرع في طاعتِنا
( ۸ ) صعيدا جُرزا :تُرابا أجْرَد لا نبات فيه
( ۹ ) أم حسبت :بل أظنَنْـتَ
( ۹ ) أصحاب الكهف :النّقب المُتسع في الجبل
( ۹ ) الرّقيم :اللوح فيه أسماؤهم و قصّتهم
( ۱۰ ) أوى الفتية :التجئوا هَرَبا بِدِينهم . .
( ۱۰ ) رشدا :اهتداءً إلى طريق الحقّ
( ۱۲ ) فضربنا على آذانهم :أنمناهم إنامة ثقيلةً
( ۱۲ ) بعثناهم :أيقظناهم من نومهم
( ۱۲ ) أمدا :مدّة و عدد سِنين أو غاية
( ۱٤ ) ربطنا :شدَدْنا و قوّينا بالصّبر
( ۱٤ ) شططا :قولا مُفرطافي البعد عن الحقّ
( ۱٦ ) مرفقا :ما تنتفعون به في عيشكم
( ۱۷ ) تزاور :تميل و تعدل

( ۱۷ ) فجوة منه :مُتسع من الكهف
( ۱۸ ) بالوصيد :بفِناء الكهف أو عتبة بابه
( ۱۸ ) رُعبا :خوفا و فزعا
( ۱۹ ) بعثناهم :أيقظناهم من نومتهم الطّويلة
( ۱۹ ) بوَرقكم :بدارهمكم المضروبة
( ۱۹ ) أزكى طعاما :أحلّ، أو أجود طعاما
( ۲۰ ) يظهروا عليكم :يطّلعوا عليكم أو يغلبوا
( ۲۱ ) أعثرنا عليهم :أطلَعنا النّاس عليهم
( ۲۲ ) رجما بالغيب :قذفًا بالظّن غير يقين
( ۲۲ ) فلا تمارفيهم :فلا تُجادل في عدّتهم و شأنهم
( ۲۲ ) إلا مراء ظاهرا :بمجرّد تلاوة ما أوحي إليك في أمرهم
( ۲٤ ) رشدا :هداية و إرشادًا للنّاس
( ۲٦ ) أبصر به :ما أبصر الله بكلّ موجود
( ۲۷ ) مُلتحدا :مَلجأ و مَوْئلا
( ۲۸ ) اصبر نفسك :احبسْها و ثبّتها
( ۲۸ ) لا تعد عيناك عنهم :لا تصرف عيناك النّظر عنهم
( ۲۸ ) أغفلنا قلبه :جعلناه غافلا ساهيًا
( ۲۸ ) فرُطا :إسرافًا . أو تضْييعا و هلاكا
( ۲۹ ) سُرادقها :فُسْطاطُها . أو لهبُها و دُخانها
( ۲۹ ) كَالمهل :كدُرديّ الزّيت أو كالمُذاب من المعادن

( ۳۱ ) جنات عَدْن :جنات إقامة و إستقرار

( ۳۱ ) إستبرق :غليظ الدّيباج
( ۳۱ ) الأرائك :السّرُر في الحجال (جمع حجلة محركة – بيت يزين بالثياب و الأسرّة و الستور)
( ۳۲ ) جنّتين :بُستانين
( ۳۲ ) حففناهُما :أحطناهما و أطفناهما
( ۳۳ ) أُكُلها :ثمرها الذي يُؤكل

( ۳۳ ) فجّرنا خِلالهما :شقـَقـْنا و أجْرَينا وسَطهما
( ۳٤ ) ثمرٌ :أموالٌ كثيرة مُثمّرة
( ۳٤ ) أعزّ نفرا :أقوى أعوانا أو عشيرة
( ۳۵ ) تبيد :تهلك و تفنى و تخرب
( ۳٦ ) مُنقلبا :مرْجعا و عاقِيةً
( ۳۸ ) لكنّا هو الله ربّي :لكنْ أنا أقول: هو الله ربّي
( ٤۰ ) حُسبانا :عذابا كالصّواعق و الآفات
( ٤۰ ) فتُصبح صعيدا زلقا :رمْلا هائلا أو أرضا جُرُزا لا نبات فيها يُزلق عليها لِملاسَتِها
( ٤۱ ) غورا :غائرا ذاهبا في الأرض

( ٤۲ ) يُقلّب كفّيه :كِماية عن النّدم و التّحسّر
( ٤۲ ) خاوية على عُرُوشها :ساقطة على سُقوفها التي سَقطت
( ٤٤ ) الولاية لله :النّصرة له تعالى وحده
( ٤٤ ) خيرٌ عقبا :عاقبة لأوليائه
( ٤۵ ) هشيما :يابسا متفتّتا بعد نضارته
( ٤۵ ) تذروه الرّياح :تُفرّقه و تنسِفه
( ٤۷ ) بارزة :ظاهرة لا يسترها شيء
( ٤۸ ) موعدا :وقتا لإنجازنا الوعد بالبعث و الجزاء
( ٤۹ ) وُضع الكتاب :صُحُف الأعمال في أيدي أصحابها
( ٤۹ ) مُشفقين :خائفين وَجلين
( ٤۹ ) يا ويلتنا :يا هلاكنا
( ٤۹ ) لا يُغادر :لا يترُك و لا يُبقي
( ٤۹ ) أحصاها :عدّها و ضبطها و أثبتها
( ۵۰ ) اسجدوا لآدم :سجود تحية و تعظيم لا عبادة
( ۵۱ ) عضُدا :أعوانا و أنصارا
( ۵۲ ) مَوْبقا :مهلِكا يشتركون فيه و هو النّار
( ۵۳ ) مُواقعوها :واقعون فيها أو داخلون فيها
( ۵۳ ) مصرفا :معدلا و مكانا ينصرفون إليه
( ۵٤ ) صرّفنا :كرّرنا بأساليب مُختلفة
( ۵٤ ) كلّ مثل :معنى غريب بديع كالمثل في غرابته
( ۵۵ ) سنّة الأوّلين :عذاب الاستئصال إذا لم يُؤمنوا
( ۵۵ ) قُبُلا :أنواعا و ألوانا أو عيانا و مقابلة
( ۵٦ ) لِيدخضوا :لِيبطلوا و يُـزيلوا
( ۵٦ ) هُـزوا :استهزاء و سُخريّة
( ۵۷ ) أكنّة . . :أغطية كثيرة مانعة . .
( ۵۷ ) وقرا :صممًا و ثقلا في السّمع عظيما
( ۵۸ ) موئِلا :منجى و ملجأ و مخلصا
( ۵۹ ) لمَهلِكهم :لهلاكهم
