۳۲ - سورة السجدة
( ۳ ) افتراه :اختلق القرآن من تِلقاء نفسه
( ٤ ) استوى على العرش :استواءً يليق بكماله و جلاله تعالى
( ۵ ) يعرُج إليه :يصعد الأمر و يرتفع إليه بعد تدبيره
( ۷ ) أحسَن كلّ شيء :أحكمه و أتقنه
( ۸ ) سلالة :خُلاصة
( ۸ ) ماءٍ مهين :منيّ ضعيف حقير
( ۹ ) سوّاه :قوّمه بتصوير أعضائه و تكميلها
( ۱۰ ) ضللنا في الأرض :ضِعنا فيها و صرنا ترابا
( ۱۲ ) ناكِسوا رءوسهم :مُطْرقُها خِزياً و حياءً و ندماً
( ۱۳ ) حقّ القول :ثبت و تحقّق و نفذ القضاء
( ۱۳ ) الجـِـنّة :الجنّ
( ۱٦ ) تتجافى جنوبهم :ترتفع و تتنحّى للعبادة
( ۱٦ ) عن المضاجع :الفُرُش التي يُضطجع عليها
( ۱۷ ) من قرّة أعين :من موجـِــبات المسرّة و الفرح
( ۱۹ ) نـزُلاً :ضِيافة . و عطاءً. و تكرمة
( ۲۳ ) في مِريَة :في شكّ
( ۲۳ ) من لقائه :تلقّيه إياه بالرّضا و القبول
( ۲٦ ) أوَ لم يهدِ لهم ؟ :أغفلوا و لم يُبيّن لهم مآلهم ؟
( ۲٦ ) كم أهلكنا . . :كثرة إهلاكنا الأمم قبلهم
( ۲٦ ) القرون :الأمم الخالية
( ۲۷ ) الأرض الجُرز :اليابسة الجرداء التي قُطِعَ نباتها
( ۲۸ ) هذا الفتح :النّصْرُ علينا ، أو الفصل للخصومة
( ۲۹ ) يُنظَرون :يُمهَلون ليُؤمنوا
