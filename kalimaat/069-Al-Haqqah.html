٦۹ - سورة الحاقة
( ۱ ) الحاقّة :السّاعة يَتحقـّـق فيها ما أنكَرُوه
( ۲ ) ما الحاقّة :أيّ شيءٍ هِيَ في أهْوَالها
( ٤ ) بالقارعة :بالقيامة تـقـْـرَع القـلوب بأفزَاعِها
( ۵ ) بالطاغية :بالصّـيْحَة المُجَاوزة للحدّ في الشّدة
( ٦ ) بريح صرصر :شديدة السّموم أو البرْد أو الصّوْت
( ٦ ) عاتِـيَة :شديدة العصْف
( ۷ ) سَخّرها عليهم :سَلـّـطها عليهم بقدْرتِه تعالى
( ۷ ) حُسُوما :مُـتـتـَـابعَاتٍ . أو مَشئوماتٍ
( ۷ ) أعْجَاز نخل :جُذوع نـَخل بلا رُءُوس
( ۷ ) خاوية :ساقِطة أو فارغة أو بالِيَة
( ۹ ) المؤتـفـكات :قرى قوم لوط (أهلها)
( ۹ ) بالخاطئة :بالفَعَلات ذات الخَطَـأ الجَسيم
( ۱۰ ) أخذة رَابيَة :زَائدة في الشدّة على الأخذات
( ۱۱ ) الجَاريَة :سَفينة نوح عليه السّـلام
( ۱۲ ) تـَـذكِرة :عِـبْرَة و عِظة
( ۱۲ ) وَتـَـعِـيَها :و لِتحْـفَـظها
( ۱۳ ) نـَـفـْـخة ٌ واحدة :النّـفخة الأولى لخراب العالم
( ۱٤ ) حُـمِلتِ الأرض :رُفِعَت من أماكنها بأمْرنا
( ۱٤ ) فدُكّـتـا :فدقّـتا و كُسِرَتا . أو فسُوِّيتا
( ۱۵ ) وَقَعَت الواقِعة :قامت القيامة
( ۱٦ ) انشَقّـت السّماء :تـفَـــّطرَتْْ و تـَـصدّعَتْ من الهَوْل
( ۱٦ ) وَاهـية :ضعيفة مُتداعِـيَة بعد الإحْكام
( ۱۷ ) على أرْجائها :جَوانبها و أطرافها
( ۱۸ ) يَـوْمـَـئِـذ تـُـعْـرَضون :بعد النّـفخة الثانية للحساب و الجزاء
( ۱۹ ) هاؤم :خُذوا أو تعالـَـوْا
( ۱۹ ) كِـتـَـابيَهْ :كتابي ، و الهاء للسّـكـت
( ۲۱ ) راضِـية :مَرْضِيّة لا مكروهة
( ۲۳ ) قطوفها دانية :ثمارها قريبة التـّـناول إذ تـُـجْـنى
( ۲٤ ) هنيئا :أكْلاً غيْر مُنغّص و لا مكَدّر
( ۲۷ ) كانت القاضية :الموْتة القاطعة لأمْري و لم أبْعَث
( ۲۸ ) ما أغنى عـنّي :ما دَفَعَ العَذاب عَنّي
( ۲۸ ) مَالـِـيَهْ :الذي كان لي مِنْ مالٍ و نحوه
( ۲۹ ) سُـلْـطانِيَهْ :حُجّـتِي أو تسلـّـطي و قـوّتي
( ۳۰ ) فَغـلـّـوه :اجْعلوا الغلّ في يَدَيْه و عُنقِه
( ۳۱ ) الجَحيم صَـلـّـوه :أدْخِلوه . أو احرقوه فيها
( ۳۲ ) فاسْـلكوه :فأدخِلوا فيها
( ۳٤ ) لا يَحُضّ :لا يَحُثّ و لا يُحَرّض
( ۳۵ ) حَمـِـيمٌ :قريبٌ مُشفـِـق يَحْميه مِنَ العَذاب
( ۳٦ ) غِسلين :صَديد أهْل النّـار
( ۳۷ ) الخاطِـئون :الكافرون
( ۳۸ ) فَلاَ أقـْـسِـم :أقسِمُ. و "لا" مزيدة
( ٤۰ ) إنّه لَقوْل رَسول :يُـبَـلـّـغُهُ عن الله أوحِيَ إليْه
( ٤٤ ) تقوّلَ عَـلـيْـنا :اخْتـَـلقَ و افترى علينا
( ٤۵ ) باليَمين :بيَمينه . أوْ بالقـُـوّة و القُـدْرَة
( ٤٦ ) الوَتين :نِيَاط القـَـلب . أو نُخاع الظهر
( ٤۷ ) عـنه حَاجزين :مانِعين الهلاك عَـنه
( ۵۰ ) لَحَسْرَةٌ :نـَـدامَة عظيمة
( ۵۲ ) فسبّح باسم ربّـك :نـَـزّهْهُ عَمّـا لا يَليق به تعالى
