۵۰ - سورة ق
( ۱ ) و القرآن :قسم جوابه لتبعثنّ
( ۳ ) رَجْع بعيدٌ :رجوعٌ إلى الحياة غير ممكن
( ۵ ) أمرٍ مريج :مختلط مضطرب
( ٦ ) فروج :فتوق و شقوق
( ۷ ) الأرض مددناها :بسطناها للاستقرارعليها
( ۷ ) رواسيَ :جبالا ثوابت تمنعها المَيَدَان
( ۷ ) زوْج بهيج :صِنـْـف حسن نضير
( ۸ ) عبد منيب :راجع إلينا مذعن ٍ بقدرتنا
( ۹ ) حبّ الحصيد :حبّ الزّرع الذي يحصد
( ۱۰ ) النخل باسقات :طوالاً . أو حوامل
( ۱۰ ) لها طلعٌ :هو ثمرها ما دام في وعائه
( ۱۰ ) نضيد :متراكمٌ بعضه فوق بعض
( ۱۱ ) كذلك الخروج :من القبور أحياءً عند البعث
( ۱۲ ) أصحاب الرّس :البئر ، رسّوا نبيّهم فيها فأهلكوا
( ۱٤ ) أصحاب الأيكة :سُـكّـان الغيضة الكثيفة الملتفّة الشجر (قوم شعيب)
( ۱٤ ) قوْم تبّع :أبي كَــِربٍ الحِمْـيَريّ ملِك اليَمَن
( ۱۵ ) أفعيينا بالخلق :أفعجزنا عنه – كلاّ
( ۱۵ ) في لـَـبْس ٍ :خَلـْط و شبهةٍ و شكّ
( ۱٦ ) حَبْـل الوريد :عِرْق كبير في العنق
( ۱۷ ) يَتلـقّى المتلقّـيان :يحفظ و يكتب الملكان
( ۱۷ ) قعيدٌ :مَلـَـكٌ قاعد
( ۱۸ ) رقيبٌ عتيدٌ :مَلـَـكٌ حافظ لأقواله مُعدّ حاضرٌ
( ۱۹ ) سَكْرة الموت :شِدّته و غَمْرَته الذاهبة بالعقل
( ۱۹ ) تحيد :تميل عنه و تفرّ منه و تـَـهْرُب
( ۲۲ ) غِطاءَك :حِحاب غفلتك عن الآخرة
( ۲۲ ) حديد :نافِذ قويّ
( ۲۳ ) عتيد :مُعدّ حاضر مُهَـيّـأ للعرْض
( ۲٤ ) عنيد :شديد العِناد و المجافاة للحقّ
( ۲۵ ) مُعتد :ظالم متجاوز للحدّ
( ۲۵ ) مُريب :شاكّ في الله و في دينه
( ۲۷ ) ما أطْغيته :ما قهَرْتـُـهعلى الطغيان و الغَواية
( ۳۱ ) أزلفت الجَنة :قـُـــِرّبَتْ و أدْنِيَتْ
( ۳۲ ) أوّاب :رجّاع إلى الله بالتوبة
( ۳۲ ) حفيظ :لما استودعه الله من حقّه
( ۳۳ ) بقلب منيب :مُخلص مُقبل على طاعة الله
( ۳٦ ) كمْ أهلكنا :كثيرا أهلكنا
( ۳٦ ) قرنٍ :أمّة
( ۳٦ ) بَطشًا :قوّة أو أخْذا شديدا في كلّ شيء
( ۳٦ ) فنقـّـبوا في البلاد :طوّفوا في الأرض حَذرَ الموت
( ۳٦ ) محيص :مَهْرَبٍ و مفرّ من الله
( ۳۸ ) لُغوب :تعب و إعياءٍ
( ۳۹ ) سبّح بحمد ربّـك :نـَــِزّهْهُ تعالى عن كلّ نقص أو صَلّ له تعالى حامدًا له
( ٤۰ ) أدبار السّجود :أعقاب الصلوات
( ٤۲ ) يسْمعون الصيحة :نفخة البعث
( ٤٤ ) تشقّق الأرض. . :تنفلق و تتصدّع . .
( ٤٤ ) سِراعًا :مُسرعين إلى الدّاعي
( ٤۵ ) بجبّار :بمُسلـّط تجْـبـرُهمْ على الإيمان
