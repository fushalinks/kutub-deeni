۸۸ - سورة الغاشية
( ۱ ) الغاشية :القيامةِ تَغشى الناسَ بأهوالِها
( ۲ ) خاشعة :ذَليلةٌ خاضِعةٌ من الخِزْي
( ۳ ) عامِلة :تَجُرُّ السلاسلَ والأغلالَ فى النار
( ۳ ) ناصبة :تـَعِبَة ٌ ممَّا تـُلاقِيه فيها من العذاب
( ٤ ) تصلى نارًا حامية :تدخُلُ أو تـُقاسِي نارًا تـَناهَى حَرُّها
( ۵ ) عيْن آنـيَـة :بَلَغَت أنـَاهَا (غايَـتـَها) فى الحرارة
( ٦ ) ضريع :شيءٍ فى النار ، كالشَّوكِ مُرّّ مُنـْتـِنٍ
( ۷ ) لا يُغْـني منْ جوع :لا يَدفَعُ عـنهم جُوعًا
( ۸ ) ناعمة :ذاتُ بهجةٍ وحُسنٍ ونَضارةٍ
( ۱۱ ) لاغِية :لَغوًا وباطِلاً
( ۱۳ ) سُرُرٌ مرفوعة :مُرتـَفعةُ السَّمكِ أو رفيعة ُ القـَدْرِ
( ۱٤ ) أكواب مَوْضوعة :أقداحٌ بين أيديهم للشرب منها
( ۱۵ ) نـَـمارق مَصْـفوفة :وسائدُ ومَرافقُ يَـتـَّـكِأُ عليها مَوضُوعٌ بَعضُها إلى جَنـْبِ بعض
( ۱٦ ) زرابيّ مَبْـثوثة :بُسطٌ فاخِرةٌ مُـفـَرّقة فى المجالس
( ۱۷ ) يَـنظرون :يَتأمَّـلـُـون فيُدْركُون
( ۲۲ ) بمُـسيْـطر :بِمُـتـَـسـلّـِطٍ جبَّـارٍ
( ۲۵ ) إيابَـهم :رجُوعَهم بعد الموت بالبعث
