۸۸ - سورة الغاشية
الغاشية ( ۱ )
القيامةِ تَغشى الناسَ بأهوالِها
خاشعة ( ۲ )
ذَليلةٌ خاضِعةٌ من الخِزْي
عامِلة ( ۳ )
تَجُرُّ السلاسلَ والأغلالَ فى النار
ناصبة ( ۳ )
تـَعِبَة ٌ ممَّا تـُلاقِيه فيها من العذاب
تصلى نارًا حامية ( ٤ )
تدخُلُ أو تـُقاسِي نارًا تـَناهَى حَرُّها
عيْن آنـيَـة ( ۵ )
بَلَغَت أنـَاهَا (غايَـتـَها) فى الحرارة
ضريع ( ٦ )
شيءٍ فى النار ، كالشَّوكِ مُرّّ مُنـْتـِنٍ
لا يُغْـني منْ جوع ( ۷ )
لا يَدفَعُ عـنهم جُوعًا
ناعمة ( ۸ )
ذاتُ بهجةٍ وحُسنٍ ونَضارةٍ
لاغِية ( ۱۱ )
لَغوًا وباطِلاً
سُرُرٌ مرفوعة ( ۱۳ )
مُرتـَفعةُ السَّمكِ أو رفيعة ُ القـَدْرِ
أكواب مَوْضوعة ( ۱٤ )
أقداحٌ بين أيديهم للشرب منها
نـَـمارق مَصْـفوفة ( ۱۵ )
وسائدُ ومَرافقُ يَـتـَّـكِأُ عليها مَوضُوعٌ بَعضُها إلى جَنـْبِ بعض
زرابيّ مَبْـثوثة ( ۱٦ )
بُسطٌ فاخِرةٌ مُـفـَرّقة فى المجالس
يَـنظرون ( ۱۷ )
يَتأمَّـلـُـون فيُدْركُون
بمُـسيْـطر ( ۲۲ )
بِمُـتـَـسـلّـِطٍ جبَّـارٍ
إيابَـهم ( ۲۵ )
رجُوعَهم بعد الموت بالبعث
