۸٤ - سورة الإنشقاق
( ۱ ) السماء انشقّـت :انصَدعت عند قيام القيامة
( ۲ ) أذنت لربها :اسْـتـمَـعَتْ و انـْـقادَتْ له تعالى
( ۲ ) حُـقـّـتْ :حَقّ الله عليها الاستماع و الانقـياد
( ۳ ) الأرْض مُدّتْ :بُـسِطتْ و سُـوّيَتْ كمَـدّ الأديم
( ٤ ) ألـْـقـَت ما فـيها :لـَـفـَـظتْ ما في جَوْفِها منَ الموتى
( ٤ ) تخلـّـتْ :خَـلـت عنه غاية الخُـلـُـوّ
( ٦ ) كادِح إلى ربّـك :جاهدٌ في عملك إلى لقاء ربّـك
( ٦ ) فـمُلاقـيه :فمُـلاقٍ لا محالة جزاء عملك
( ۱۱ ) يَدْعو ثبورا :يُنادي هَـلاكًـا قائلا يَاثـُـبُـراه
( ۱۲ ) يَصلى سعيرا :يدْخلها أو يُـقاسي حرّها
( ۱٤ ) لنْ يَحور :لن يرْجع إلى ربّـه تكْذيبا بالبَعْث
( ۱٦ ) فلا أقسِم :أقسِم و "لا" مزيدة
( ۱٦ ) بالشّـفـق :بالحُمْرَة في الأفق بعد الغروب
( ۱۷ ) ما وَسَق :ما صَمّ و جَمَـعَ ما انتشر بالنـّهار
( ۱۸ ) اتـّـسق :اجْـتـمَعَ و تكامل و تمّ نوره
( ۱۹ ) لتـركبنّ :لتـُـلاقُـنّ أيّها النـّاس ( جواب القسم)
( ۱۹ ) طبَـقـًـا :أحْوَالا بَعْدَ أحْوَال مُتطابقة في الشّدة
( ۲۳ ) يوعـون :يُضمِرونه أو يَجمعونه من السّيئات
( ۲۵ ) غيْرُ مَمْـنون :غير مقطوع عنهم
