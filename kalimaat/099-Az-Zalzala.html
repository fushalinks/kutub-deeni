۹۹ - سورة الزلزلة
( ۱ ) زلزلت الأرض
حـُرّكت تحْريكا عَـنيفا مُتكرّرا عند النفخة الأولى
( ۲ ) أثقالها
كُنـوزها و مَوْتاها في النّـفخة الثانية
( ٤ ) تـُحَـدّث أخبارها
تدُلّ بحالِها على ما عُملَ عليها
( ۵ ) أوْحى لها
جَعَل في حالِها دِلالة على ذلك
( ٦ ) يصدر الناس
يَخْـرجونَ منْ قُـبُورهمْ إلى المَحْـشَر
( ٦ ) أشتاتا
مُـتـفرّقين على حَسَب أحْـوالِهمْ
( ۷ ) مِثقال ذرّة
وَزْنَ أصْـغر نملةٍ أو هَـباءَةٍ
