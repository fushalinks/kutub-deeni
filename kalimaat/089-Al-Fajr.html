۸۹ - سورة الفجر
( ۱ ) و الفجر
(أَقْسَمَ تعالى) بالوقتِ المعروف
( ۲ ) و ليالٍ عشْر
العَشْرِ الأُوَلِ من ذِى الحِجَّة
( ۳ ) و الشّـفـع و الوتر
يومِ النَّحْرِ ، ويَومِ عَرَفَة
( ٤ ) و الليل إذا يَسْر
إذا يَمضِى ويَذهَبُ أو يُسارُ فيه
( ۵ ) هَلْ في ذلك
المذكورِ الذى أَقـسَمْـنـَا به
( ۵ ) قـَـسَـمٌ لذي حِجْر ؟
مُقسَمٌ به حَقيقٌ بالتعظيمِ لدى العُقلاءِ ـ نعم ـ (وجواب القسم) لـَنُعذبَنَّ الكافرين
( ٦ ) بـِعادٍ
قَومِ هُودٍ ؛ سُمُّوا باسمِ أبيهم
( ۷ ) إرَم
هو اسمُ جَدِّهم وبه سُمِّـيَت القبيلة
( ۷ ) ذات العِماد
الشدةِ أو الأبْنِية الرَّفيعةِ المُحكَمَة بالعَمَد
( ۹ ) جابوا الصّخْـر
قَـطَعُوه ونـَحَتـُوا فيه بيوتـَهم
( ۱۰ ) ذي الأوتاد
الجُيُوشِ الكثيرةِ التى تَشُدُّ مُـلـْـكَه
( ۱۳ ) سَوْط عذاب
عذابًا شديدًا مُؤلِمًا دائمًا
( ۱٤ ) إنّ ربّـك لبالمرصاد
يَرقـُـبُ أعمَالَهم ويُجازيهم عليها
( ۱۵ ) ابْـتلاه ربّه
امْـتـَحَنه واخـتـَـبَره بالنعم أو الـنـِّقم
( ۱٦ ) فقدر عليه رزقه
فَضَـيَّـقـَه عليه ولم يَبسُطْه له
( ۱۷ ) كلاّ
رَدْعٌ للإنسان عمَّا قاله فى الحالَيْن
( ۱۷ ) بلْ
لكم أعمالٌ أسوأ ُ من ذلك
( ۱۸ ) لا تحاضّون
لا يَحُثُّ بعضُكم بعضًا
( ۱۹ ) تأكلون التـّـراث
مِيراثَ النساءِ والصِّغار
( ۱۹ ) أكْلاً لمّـا
جَمعًا بين الحلال والحرام
( ۲۰ ) حُـبّـا جَمّـا
كثيًرا مع حِرصٍ وشَرَهٍ
( ۲۱ ) دُكّتِ الأرض
دقَّـتْ وكُـسِرَتْ بالزَّلازل
( ۲۱ ) دكّـا دكّـا
دَكًّا مُـتـَتـَابـِعًا حتى صارت هَباءً
( ۲۲ ) و المَـلـَـك
ملائكة ُ كلِّ سماءٍ
( ۲۳ ) أنـّـى له الذكرى
من أين له مَنفـَعَـتـُها ؟ هـَيْهات
( ۲٦ ) لا يوثِـق
لا يَشُـدُّ بالسلاسلِ والأغلال
