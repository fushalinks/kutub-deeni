۷۰ - سورة المعارج
( ۱ ) سَأل سائل :دَعَـا دَاع على نـَـفـسِه و قوْمِه
( ۳ ) ذي المعارج :ذي السّمـوات مَصَـاعِـد الملائكة
( ٤ ) تعرُج الملائكة :تصْعَـدُ في تلك المعارج
( ٤ ) الرّوح :جبريل عليه السلام
( ٤ ) في يوم :هو يوم القيامة
( ٤ ) مِقداره :في حقّ الكفار
( ۵ ) صَبْرًَا جميلا :لا شكوى فيه لغيْره تعالى
( ۸ ) السّماء كالمُهل :كالمعدن المُذاب أو دُرْدُريّ الزّيت
( ۹ ) الجبال كالعِهن :كالصّوف المصبوغ ألوانا
( ۱۰ ) حَميم :قريبٌ مُشْـفـقٌ لِشدّة الهَوْل
( ۱۱ ) يُـبـَصّرونهم :يُعَرَّف الأحْماء أحْمَاءَهم
( ۱۳ ) فَصيلته :عشيرته الأقربين المنفصل عنهم
( ۱۳ ) تـُـؤويه :تـَـضمّه في النّسَـب . أو عِـند الشدّة
( ۱۵ ) إنّها لظى :جَهنّم . أو الدركة الثانية مِنـْـها
( ۱٦ ) نـَـزّاعة للشّوى :قلاّعة للأطراف أوْ جلدِ الرّأس
( ۱۸ ) فأوْعى :أمْسَك ما له في وعاءٍ حرصًـا و تأميلاً
( ۱۹ ) هلوعًـا :كثيرَ الجَزَعَ ، شديد الحِرْص
( ۲۰ ) جَزوعًـا :كثيرَ الجَزَعِ و الأسَى
( ۲۱ ) مَنوعًا :كثيرَ المَنـْـعِ و الإمساك
( ۲۵ ) المَحْروم :مِنَ العَطاء لتعفـّـفهِ عَنِ السّؤال
( ۲۷ ) مُشفِـقـون :خائفون اسْـتِعظامًا لله تعالى
( ۳۱ ) العادون :المُـجَاوزون الحلال إلى الحَرَام
( ۳٦ ) مُهطِعين :مُسْرعين ، مادّي أعْناقِهم إليك
( ۳۷ ) عِزين :جماعات مُتفرقين
( ۳۹ ) ممّا يعلمون :مِنْ نُطفٍ مَهينـَـة مَذِرَة
( ٤۰ ) فلا أقسمُ :أقْـسِم . و "لا" مزيدة
( ٤۱ ) بمَسْبوقين :مَغْـلوبين عاجزين
( ٤۲ ) فَـذرْهُم :فدعْهُم و خَـلّـهمْ غير مُـكْـترثٍ بهم
( ٤۲ ) يَخوضوا :يَنغمِسُوا في باطلِهمْ
( ٤۳ ) مِنَ الأجداث :مِنَ القـبُور
( ٤۳ ) سِراعًـا :مُسْرعـين إلى الدّاعي
( ٤۳ ) نُصُب :أحْجَار عَـــّظموها في الجاهلية
( ٤۳ ) يوفِضون :يُسْرعون
( ٤٤ ) خاشِعة أبْصارُهم :ذليلة مُنـْـكسِرة لا يَرْفَعونها
( ٤٤ ) ترْهَقهم ذلـّـة :تغْـشاهمْ مَهَانة شديدة
