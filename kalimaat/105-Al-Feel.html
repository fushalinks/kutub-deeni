۱۰۵ - سورة الفيل
( ۱ ) بأصحاب الفـيل :وَقـَـعَت القصّة أوّل عام مولده صلى الله عليه و سلم
( ۲ ) يَجْعَلْ كَيْدَهمْ :سَعْـيَهمْ لتخْريب الكَعْـبَة
( ۲ ) تـَـضْـليل :تـَـضييع و إبْطال و خَسار
( ۳ ) طَيْرًا أبابيل :جماعاتٍ مُتفرّقة مُتتابعة
( ٤ ) سِجّـيـل :طين مُـتحجّر مُحْرَق (آجُرٍّ )
( ۵ ) كَعَصْفٍ مأكول :كَـتِـبْـنِ أكَلـتـْه الدّوابّ فراثـَتـْـه
