۵۵ - سورة الرحمن
( ۲ ) علـّـمَ القرآن :عَـلـّـم الإنسان القرآن
( ۵ ) بحُـسبان :يجريان بحساب مقدّر في بروجهما
( ٦ ) النّجم :النبات الذي ينجم و لا ساق له
( ٦ ) يسجدان :ينقادان لله فيما خُـلِقا له
( ۷ ) وضع الميزان :شرَع العدل و أمر به الخلق
( ۸ ) ألا تطْغَوْا :لئلاّ تتجاوزوا العدل و الحقّ
( ۹ ) بالقسْط :بالعدْل
( ۹ ) لا تـُـخْـسِروا الميزان :لا تنقصوا مَوْزون الميزان
( ۱۰ ) الأرض وضعها :خلقها محفوظة عن السماء
( ۱۱ ) ذات الأكمام :أوعية الثمر و هي الطلْع
( ۱۲ ) ذو العَصْف :القِـشْـر أو التـّـبْن أو الوَرَق اليابس
( ۱۲ ) الرّيحان :النبات المشموم الطيّب الرّائحة
( ۱۳ ) آلاء ربّكما :نعمِه تعالى
( ۱۳ ) تكذبان :تكفران أيها الثقلان
( ۱٤ ) صلصال :طين يابس يُسمع له صلصلة
( ۱٤ ) كالفخّار :هو الطين يُحرَق حتى يتحجّر
( ۱۵ ) مارج :لهبٍ صافٍ لا دخان فيه
( ۱۹ ) مَرَج البحرين :أرْسَل العـذب و المِلح في مجاريهما
( ۱۹ ) يلتقِـيان :يتجاوران أو يلتقي طرفاهما
( ۲۰ ) بيْنهما برْزخ :حاجز أرضي أو من قدرته تعالى
( ۲۰ ) لا يبْغِيان :لا يطغى أحدهما على الآخر بالمُـمَـازحَة
( ۲٤ ) له الجوار :السّـفـن الجارية
( ۲٤ ) المنشئات :المرفوعات الشّرُع (القلوع)
( ۲٤ ) كالأعلام :كالجبالالشّاهقة أو القصور
( ۲٦ ) فان :هالك
( ۲۷ ) ذو الجلال :العظمة و الاستغناء المطلق
( ۲۷ ) الإكرام :الفَـضْـل التام
( ۲۹ ) في شأن :يأتي بأحوالٍ و يذهب بأحوالٍ بالحكمة
( ۳۱ ) سنفرُغ لكم :سنقصِد لمحاسَبتكم بعد الإهمال
( ۳۱ ) أيها الثقلان :الإنس و الجنّ
( ۳۳ ) تنفذوا :تخرجوا هربا من قضائي
( ۳۳ ) فانفذوا :فاخرجوا (أمر تعجيز)
( ۳۳ ) بسلطان :بقوّة و قهر، و هيهات . . !
( ۳۵ ) شواظ :لهب خالص لا دخان فيه
( ۳۵ ) نحاس :صُـفْـرٌ مُذاب أو دخان بلا لهَب
( ۳۷ ) فكانت وردة :كالوردة في الحمرة
( ۳۷ ) كالدّهان :كدُهْن الزيت في الذوبان
( ٤۱ ) بسيماهم :بسواد الوجوه ، و زرقة العيون
( ٤۱ ) فيؤخذ بالنّواصي :بشعور مقدّم الرّءوس
( ٤٤ ) حميم آن :ماءٍ حارتناهى حرّه
( ٤٦ ) جنّـتان :بستان داخل القصر و آخر خارجه
( ٤۸ ) ذواتا أفنان :أغصان . أو أنواع من الثمار
( ۵۰ ) عينان :التسْـنيم و السلسبيل
( ۵۲ ) زوجان :صِنفان : معروف و غريب
( ۵٤ ) استبرق :غليظ الديباج
( ۵٤ ) جنى الجنـّـتـيْـن :ما يُجنى من ثمارهما
( ۵٤ ) دان :قريب من يد المتناول
( ۵٦ ) قاصرات الطّرف :قصَرْنَ أبصارَهنّ على أزواجهنّ
( ۵٦ ) لم يطمثهنّ :لمْ يفـتضّهنّ قبل أزواجهنّ
( ٦۲ ) و من دونهما جنـّـتان :أعلى أو أدنى من السابقـتـيْن
( ٦٤ ) مدهامّـتان :خضراوان شديدتا الخضرة
( ٦٦ ) نضّاختان :فوّارتان بالماء لا تنقطعان
( ۷۰ ) خيرات حسان :خيّرات الأخلاق حِسان الوُجوه
( ۷۲ ) حور :نساءٌ بيض حِسان
( ۷۲ ) مقصورات في الخِيام :مخدّرات في بيوت من اللؤلؤ
( ۷٦ ) رفرف :وسائد أو فُرُش مُرتفعة
( ۷٦ ) عبقريّ :بُسطٍ ذات خمْل رقيق
( ۷۸ ) تبارك :تعالى . أو كثـُـرَ خيْرُه و إحسانه
( ۷۸ ) ذي الجلال :العظمة و الإستغناء المُطلق
( ۷۸ ) الإكرام :الفضل التام و الإحسان
