۳۵ - سورة فاطر
( ۱ ) فاطر . . :مُبدع و مُخترع . .
( ۲ ) ما يفتح الله :ما يُرسل الله
( ۳ ) فأنّى تؤفكون ؟ :فكيف تصرفون عن توحيده ؟
( ۵ ) فلا تغرنّكم :فلا تخدعنّكم و لا تلهينّكم بالزّخارف و الملذات
( ۵ ) الغرور :ما يغرّ و يخدع من شيطان و غيره
( ۸ ) فلا تذهب نفسك عليهم حسرات :فلا تهلك نفسك عليهم غموما و أحزانا لكفرهم
( ۹ ) فتثير سحابا :تحرّكه و تهيّجه
( ۹ ) النشور :بعث الموتى من القبور للجزاء
( ۱۰ ) يريد العزّة :الشّرف و المنعة
( ۱۰ ) الكلِم الطيّب :كلمة و التوحيد و جيمع عبادات اللّسان
( ۱۰ ) العمل الصّالح يرفع :يرفع الله العمل الصّالح و يقبله
( ۱۰ ) يبور :يفسد و يبطل
( ۱۱ ) أزواجا :ذكورا و إناثا
( ۱۱ ) معمّرٍ :طويل العمر
( ۱۲ ) عذبٌ فرات :طيّب حُلو شديد العذوبة
( ۱۲ ) سائغ شرابه :مريء سهلٌ انحداره
( ۱۲ ) ملح أجاج :شديد الملوحة أو المرارة
( ۱۲ ) حلية :اللؤلؤ و المرجان من الملح
( ۱۲ ) مواخر :جواري بريح واحدة
( ۱۳ ) يولج :يُدْخل
( ۱۳ ) لأجل مسمّى :مقدّر لفنائهما (يوم القيامة)
( ۱۳ ) قطمير :هو القشرة الرقيقة على النّواة
( ۱۸ ) لا تزر وازرة . . :لا تحمل نفس آثمة . .
( ۱۸ ) مُثقلة :نفس أثقلتها الذنوب
( ۱۸ ) حِملها :ذوبها التي أثقلتها
( ۱۸ ) تزكّى :تطهّر من الكفر و المعاصي
( ۲۱ ) الحَرُور :شدّة الحرّ ليلا كالسّموم
( ۲۵ ) بالزّبر :بالكتب المكتوبة كصحف إبراهيم و موسى عليهما السلام
( ۲٦ ) كان نكير :إنكاري عليهم بالتدمير
( ۲۷ ) جددٌ :ذات طرائق و خطوط مختلفة الألوان
( ۲۷ ) غرابيب سود :متناهية في السّواد كالأغربة
( ۲۹ ) لن تبور :لم تكسُدَ و تفسُدَ ، أوَ لن تهلِك
( ۳۲ ) ظالم لنفسه :رجحت سيّـئاته على حسناته
( ۳۲ ) مقتصد :استوت حسناته و سيّـئاته
( ۳۲ ) سابق بالخيرات :رجحت حسناته على سيّـئاته
( ۳٤ ) الحَزن :كلّ ما يُحزن و يَغمّ
( ۳۵ ) دار المُقامة :دار الإقامة الدّائمة (الجنّة)
( ۳۵ ) نَصَبٌ :تعب و مشقّة
( ۳۵ ) لغوبٌ :إعياء من التعب و فتور
( ۳۷ ) هم يصطرخون :يستغيثون و يصيحون بشدّة
( ۳۹ ) جعلكم خلائف :خلفاء من كان قبلكم
( ۳۹ ) مقتا :أشدّ البغض و الغضب و الاحتقار
( ۳۹ ) خسارا :هَلاكا و خسرانا
( ٤۰ ) أرأيتم شركاءكم :أخبروني عن شُركائكم
( ٤۰ ) أم لهم شِرك ؟ :بل ألـَـهم شرِكة مع الله تعالى في الخلق؟
( ٤۰ ) غرورا :باطلا . أو خداعا
( ٤۲ ) جهد أيمانهم :مجتهدين في الحلف بأغلظها و أوكدها
( ٤۲ ) نفورا :تباعدا عن الحقّ و فرارا منه
( ٤۳ ) و مكر السّيّء :و المكر السّيء (الكيد للرّسول)
( ٤۳ ) لا يحيق :لا يُحيط أو لا ينزل
( ٤۳ ) فهل ينظرون :فما ينتظرون
( ٤۳ ) سنّة الأوّلين :سنّة الله فيهم بتعذيبهم بتكذيبهم
