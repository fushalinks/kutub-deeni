۹۸ - سورة البينة
مـنفكّين ( ۱ )
مُـزايلين ما همْ عليه من الدّين
تأتيَهُم البيّـنة ( ۱ )
الحُجّة الواضحة و هي الرّسول
صُحُـفـًـا ( ۲ )
مكتوبا فيها القرآن العظيم
مطهّرة ( ۲ )
مُنزّهة عن الباطل و الشّبهات
فيها كتب ( ۳ )
آيات و أحكام مكتوبة
قيّمة ( ۳ )
مُسْـتقـيمة حَقة عادلة مُحْكَمَة
ما تفرّق ( ٤ )
في الرّسول بين مُؤْمِن و جَاحد
جاءتهم البينة ( ٤ )
بالهُدى و كان الحقّ أو لا يتفـرقـوا
الدين ( ۵ )
العِبادة
حُنفاء ( ۵ )
مائلين عن الباطل إلى الإسلام
دين القيمة ( ۵ )
المِلـّـة المستقيمة أو الكتب القيّمة
البَريّة ( ٦ )
الخلائق أو البَشَر
