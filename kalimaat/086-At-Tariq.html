۸٦ - سورة الطارق
( ۱ ) و الطارق :(قسمٌ) بالنَّجم الثَّاقِب يَطلُعُ ليلاً
( ۳ ) النـّـجم الثاقب :المُضِىءُ المُتوهِّجُ أو المُرتفِعُ العالى
( ٤ ) إن كلّ نـَـفس :ما كلُّ نَفسٍ (جواب القسم)
( ٤ ) لمّـاعَـليها :إلا عليها
( ٤ ) حافِـظٌ :مُهَيْمنٌ ورَقِيبٌ وهو الله تعالى
( ٦ ) ماءٍ :مُمتَزجٍ من مائَي الرجلِ والمرأة
( ٦ ) دَافِـق :مَصبُوبٍ بِدَفْعٍ وسُرعةٍ فى الرَّحِم
( ۷ ) مِـنْ بين الصلب :ظَهْر كلٍّ من الرجل والمرأة
( ۷ ) و التـّرائب :عِظامِ الصَّدرِ أو الأطرافِ مِن كلٍّ منهما ، أو يَخرجُ من كل البَدَنِ مِنهما ، والصُّلْبُ والترائب كِنايةٌ عنه
( ۸ ) رَجْعِه :إعادَةِ الإنسانِ بعدَ فنائه
( ۹ ) تبْلى السّرائر :تـُكشَفُ مَكنوناتُ القلوب
( ۱۱ ) ذات الرّجْـع :المطرِ لِرُجُوعِه إلى الأرض مِرارًا
( ۱۲ ) ذات الصّدْع :النباتِ الذى تـَنشَقُّ عنه
( ۱۳ ) لـَـقوْلٌ فصل :فاصلٌ بين الحق والباطل
( ۱٦ ) أكيدُ كيْدًا :أُجازيهم على فِعلِهم بالاسْتدْراج
( ۱۷ ) فمهّل الكافرين :فلا تـَستَعجـِلْ بالانتقام منهمْ
( ۱۷ ) أمْهلهُمْ روَيْدا :إمْهالاً قريبًا ، أو قليلاً حتى يَأتِيَهم العذاب
