۱۱۱ - سورة المسد
( ۱ ) تـَـبّتْ
هَلكَتْ أو خَسِرَتْ أو خابَتْ
( ۱ ) و تبَّ
و قَدْ هَـلـَكَ أو خَسِرَ أوْخـَابَ
( ۲ ) ما أغنى عنه
ما دَفَـعَ التـّـبَابَ عنه
( ۲ ) ما كَسَب
الذي كَسَبَـهُ بـِـنـَـفـْسِهِ
( ۳ ) سَيَصلى نارًا
سَـيَـدْخلـها أوْ يُقاسي حَرّها
( ۵ ) في جيدها
في عُـنـُـقِـها
( ۵ ) مِنْ مَسَدٍ
مـمّـا يُـفـْـتـَـلُ قـَويّـا مِنَ الحِبَال
