٤۳ - سورة الزخرف
( ٤ ) أمّ الكتاب :اللّوح المحفوظ . أو العلم الأزليّ
( ۵ ) أفنضرب عنكم الذكر :أفنترك تذكيركم و إلزامكم الحجّة بإنزال القرآن
( ۵ ) صَفحًا :إعراضا أو مُعرضين عنكم
( ۵ ) أن كنتم قوما مسرفين ؟ :لكونكم مُفرطين في الجهالة و الضّلالة ؟ لا نتركه
( ٦ ) كم أرسلنا :كثيرا أرسلنا
( ٦ ) في الأولين :في الأمم السّابقة
( ۸ ) بطْشًا :قوة
( ۸ ) مثل الأوّلين :صفتهم أو قصّـتهم العجيبة
( ۱۰ ) الأرض مهدا :فِراشا ممهّدا للاستقرار عليها
( ۱۰ ) سُبُـلاً :طُرُقا تسلكونها . أو معايش
( ۱۱ ) ماءً بقدَر :بتقدير مُحكمٍ أو بمقدار الحاجة
( ۱۱ ) فأنشرنا به :فأحيينا بالماء
( ۱۲ ) خلَقَ الأزواج :أوجد أصناف المخلوقات و أنواعها
( ۱۲ ) و الأنعام :و من الأنعام و هو الإبل
( ۱۳ ) لتستووا :لتستقرّوا . و تستعلوا
( ۱۳ ) سخّر :ذلـّـل
( ۱٦ ) مُقرنين :مُطيقين و غالبين أو ضابطين
( ۱٦ ) أصفاكم بالبنين :أخلصكم و آثركم بهم
( ۱۷ ) مثلا :شِبْهاً و مُما ثِلا
( ۱۷ ) هو كظيم :مملوء في قلبه غيظا و غمّـا
( ۱۸ ) يُنشّـأ في الحلية :يُربّى في الزّينة و النعمة (البنات)
( ۱۸ ) في الخصام :المُخاصمة و الجدال
( ۲۰ ) يخرصون :يكذبون فيما قالوه
( ۲۲ ) على أمّة :على دين و طريقة تـُـؤمّ و تـُـقـْـصد
( ۲۳ ) قال مُترفوها :مُـتنعّموها المنغمسون في شهواتهم
( ۲٦ ) إنّـني براء :بريءٌ
( ۲۷ ) فطرني :خلقـَـني و أبدعني
( ۲۸ ) كلمة باقية :كلمة التوحيد ، أو البراءة
( ۲۸ ) في عَقِبه :ذرّيته إلى يوم القيامة
( ۳۱ ) من القريتين :من إحدى القريتين مكّة و الطائف
( ۳۲ ) سُخريّـا :مسخّرا في العمل ، مُسْتخدَما فيه
( ۳۳ ) أمّة واحدة :مُطْبـِـقـَـة ً على الكفر حبّـا للدنيا
( ۳۳ ) معارج :مصاعِد و مَراقي و دَرَجا من فضّة
( ۳۳ ) يظهرون :يَـصعدون و يرتقون
( ۳۵ ) زخرُفا :ذهبا ، أو زينة مزوقة ً
( ۳۵ ) لمّا متاع . . :إلاّ متاع . .
( ۳٦ ) من يعْشُ :من يتعامَ و يُعْرض و يتغافل
( ۳٦ ) نقيّض له :نسبّـب . أو نتِـحْ له
( ۳٦ ) له قرين :مُصاحبٌ له لا يُفارقه
( ٤٤ ) إنه لذكر :إن القرآن لشرفٌ عظيم
( ٤۹ ) بما عَهِد عندك :منْ كشف العذاب عمّن اهتدى
( ۵۰ ) ينْـكثون :ينقضون عهدهم بالإهتداء
( ۵۲ ) هوَ مَهين :ضعيف حقيرٌ
( ۵۲ ) يُبين :يُفصِح الكلام لِلـُـثغَة في لسانه
( ۵۳ ) مُقترنين :مقرونين به يُصدقونه
( ۵٤ ) فاستخفّ قومه :وجدهم خفاف العقول
( ۵۵ ) آسفونا :أغْضَبونا أشدّ الغضب بأعمالهم
( ۵٦ ) سَلَفا :قدْوَة للكفّار في استحقاق العقاب
( ۵٦ ) مثلا للآخرين :عِـبْرَة و عظة للكفّار بعدهم
( ۵۷ ) منه يصدّون :من أجله يضجّون و يصيحونَ فرَحا و جَذلاً
( ۵۸ ) قوم خصِمون :لُدّ شداد الخصومة بالباطل
( ۵۹ ) مثلا :آية و عبرة عجيبة كالمثل السّـائر
( ٦۰ ) لجعلنا منكم :بذلكم . أو لولـّـدْنا منكم
( ٦۱ ) إنه لعلمٌ للسّـاعة :يُعلَم قربُها بنزوله (عليه السلام)
( ٦۱ ) فلا تمترنّ بها :فلا تشكّنّ في قيَامها
( ٦۵ ) فويل :هلاك أو حسرة أو شدّة عذاب
( ٦٦ ) هل ينظرون :هل ينتظرون
( ٦٦ ) بغتتة :فجأة
( ٦۷ ) الأخلاء :الأحباء في غير ذات الله
( ۷۰ ) تحبَرون :تسرّون سرورا ظاهرَ الأثر
( ۷۱ ) أكوابٍ :أقداح لا عُرَى لها و لا خراطيم
( ۷۵ ) لا يُفتـّـر عنهم :لا يُخـفّـف عنهم
( ۷۵ ) مُبلسون :ساكنون أو حزينون من شدّة اليأس
( ۷۷ ) ليقضِ علينا ربّك :ليُمتـْـنا حتـّى نخلُـصَ من هذا العذاب
( ۷۹ ) أمْ أبرموا أمرًا :بلْ أأحكموا كيْدا له صلى الله عليه و سلم
( ۸۰ ) نجواهم :تناجيهم فيما بينهم
( ۸۳ ) يخوضوا :يدخلوا مدَاخل الباطل
( ۸٤ ) في السّمـاء إله :هو معْبودٌ في السّماء
( ۸۵ ) تبارك الذي . . :تعالى أو تكاثر خيْرُه و إحسانه
( ۸۷ ) فأنّى يُأفكون :فكيف يُصْرُفون عن عبادته تعالى
( ۸۸ ) وَ قِـيـلِه :و عنده علم قول الرسول صلى الله عليه و سلم
( ۸۹ ) فاصفحْ عنهم :فأعرض عنهم
( ۸۹ ) سلامٌ :أمري تـَـسلـّــمٌ و مُـتـَـاركة ٌ لكم
