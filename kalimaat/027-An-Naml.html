۲۷ - سورة النمل
( ۲ ) هدًى :هادٍ من الضّلالة
( ٤ ) فهم يعمهون :يعمون عن الرّشد أو يتحيّرون
( ۷ ) آنستُ نارًا :أبصرتها إبصارا بيـّــنا
( ۷ ) بشهاب قبس :بشُعلة نار ساطعة مقبوسة من أصلها
( ۷ ) تصطلون :تستدفئون بها من البرد
( ۸ ) بُورك :قدّس و طُهّرَ و زيد خيرا
( ۸ ) من في النار و من حولها :الذين في ذلك الوادي الذي بدا فيه النور و هم موسى و الملائكة
( ۱۰ ) تهتزّ :تتحرّك بشدة و اضطراب
( ۱۰ ) كأنّها جانّ :حيّة خفيفة في سرعة حركتها
( ۱۰ ) لم يُعقّب :لم يرجع على عقِبه أو لم يلتفت
( ۱۲ ) في جيبك :فتحة القميص حيث يُدخل الرّأس
( ۱۲ ) بيضاء :نيّرة يغلب نورها نور الشمس
( ۱۲ ) غير سوء :غير داء برص و نحوه
( ۱۳ ) مبصرة :واضحة بيّـنة هادية
( ۱٤ ) عُلوّا :ترفّعا و استكبارا عن الإيمان بها
( ۱٦ ) منطِق الطّير :فهم أغراضه كلّها من أصواته
( ۱۷ ) فهم يوزعون :يوقف أوائلهم لتحقهم أواخرهم
( ۱۸ ) لا يحطمنّكم :لا يكسرنّكم و يُهلِكنّكم
( ۱۹ ) أوزعني . . :ألهمني و حرّضني و اجعلني . .
( ۲۱ ) بسلطانٍ مبين :بحجّة تبيّن عُذره في غيبته
( ۲۵ ) يُخرج الخبء :يُظهر المخبوء المستور أيّا كان
( ۲۸ ) تولّى عنهم :تنحّ عنهم قليلا
( ۳۱ ) لا تعلو عليّ :لا تتكبّروا عليّ
( ۳۱ ) مُسلمين :مؤمنين . أو مُنقادين مستسلمين
( ۳۲ ) تشهدونِ :تحضرون . أو تشيروا عليّ
( ۳۳ ) أولوا بأس :أصحاب نجدة و بلاءٍ في الحرب
( ۳۷ ) لا قِبَل لهم بها :لا طاقة لهم بمقاومتها
( ۳۷ ) هم صاغرون :ذليلون بالأسر و الإستعباد
( ٤۰ ) الذي عنده علم :آصَـفُ أو جبريل أو ملك آخر
( ٤۰ ) طرْفك :نظرك . أو جفن عينك بعد فتحه
( ٤۰ ) ليبلُوَني :لِيخْـتبرني و يمتحنني
( ٤۱ ) نكّروا :غيّروا
( ٤٤ ) ادخلي الصّرح :القصر . أو ساحته أو بركته
( ٤٤ ) حسبته لُجّة :ظنّـتـه ماءً غزيرا
( ٤٤ ) صرح ممرّد :مُمَـلّس مُسوّى
( ٤٤ ) من قوارير :زجاج شفّاف
( ٤۷ ) اطّيّرنا :تشاءمنا حيث أصبنا بالشّداد
( ٤۷ ) طائركم عند الله :شؤمكم عملكم المكتوب عليكم عنده تعالى
( ٤۷ ) قوم تُفتنون :يفتنكم الشّيطان بوسوسته
( ٤۸ ) تسعة رهط :أشخاص من الرّؤساء مع كلّ رهط
( ٤۹ ) تقاسموا بالله :تحالفوا بالله . أو احلِفوا به
( ٤۹ ) لنُبيّـتـنّه و أهلَه :لنقتلنّهم ليلا بغتة
( ٤۹ ) مهْلِك أهله :هلاكهم
( ۵۱ ) دمّرناهم :أهلكناهم
( ۵۲ ) خاوية :خالية خربَة أو ساقطة متهدّمة
( ۵٤ ) أنتم تُبصرون :لا تبالون إظهارها مَجَانَة
( ۵٦ ) يتطهّرون :يزعمون التـّـنـزّه عمّا نفعل
( ۵۷ ) قدّرناها :حكمنا عليها
( ۵۷ ) من الغابرين :بجعلها من الباقين في العذاب
( ۵۸ ) مطرا :حجارة من السّماء مُهلِكَة
( ٦۰ ) حدائق ذات بهجة :بساتين ذات حُسن و رونق
( ٦۰ ) قوم يعدِلون :ينحرفون عن الحقّ إلى الباطل
( ٦۱ ) الأرض قرارا :مستقرّا بالدّحو و التـّسوية
( ٦۱ ) رواسي :جبالا ثوابت لئلاّ تميد
( ٦۱ ) حاجزا :فاصلا يمنع اختلاطهما
( ٦۳ ) رحمته :المطر الذي به تـَحـْيَى الأرض
( ٦٦ ) إدّارك علمهم في الآخرة :تكامل و استحكم علمهم بأحوالها و هو تهكّم بهم لفرْط جهلهم بها
( ٦٦ ) عمون :عُمي البصائر عن دلائلها البيّـنة
( ٦۸ ) أساطير الأوّلين :أكاذيبهم المُسطّر في كتبهم
( ۷۰ ) ضيق :حرج و ضيق صدر
( ۷۲ ) ردِف لكم :لحقكم و وصل إليكم
( ۷٤ ) ما تكنّ صدورهم :ما تُخفي و تستر من الأسرار
( ۷۵ ) غائبة :شيء يغيب و يخفى عن الخلق
( ۸۲ ) وقع القول :دنت الساعة و أهوالها الموعودة
( ۸۲ ) دابّة :هي من أشراط السّاعة الكبرى
( ۸۳ ) فوجا :جماعة و زمرة
( ۸۳ ) فهو يوزعون :يُوقف أوائلُهم لتلحقهم أواخرهم ثمّ يُساقون جميعا
( ۸۷ ) ففزع :خاف خوفا يستتبع الموت
( ۸۷ ) داخرين :صاغرين أذلاّء بعد البعث
( ۹۰ ) فكُـبـّـت وجوههم :ألقوا منكوسين
