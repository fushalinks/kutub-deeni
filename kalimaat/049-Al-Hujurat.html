٤۹ - سورة الحجرات
لا تقدّموا ( ۱ )
لا تقطعوا أمرا و تجزموا به
أن تحبط أعمالكم ( ۲ )
كراهة أن تبطل أعمالكم
يغضون أصواتهمْ ( ۳ )
يَخفضونها و يُخافتون بها
امتحن الله قلوبَهم ( ۳ )
أخلصها و صفّـاها
الحجرات ( ٤ )
حجرات زوجاته صلى الله عليه و سلم
لعَنتـّـمْ ( ۷ )
لأثمْتم و هَلـَـكتمْ
بَغَتْ ( ۹ )
اعتدت و اسْتطالت و أبَتِ الصلح
تفيء ( ۹ )
تَرْجع
أقسطوا ( ۹ )
اعدلوا في كل أموركم
المقسطين ( ۹ )
العادلين فيُـحْسِنُ جزاءَهمْ
لا يسْخرْ ( ۱۱ )
لا يهْـزأ و لا ينتقصْ
لا تلمزوا أنفسكم ( ۱۱ )
لا يَعِبْ و لا يَطعَن بعضكم بعضا
لا تنابزوا بالألقاب ( ۱۱ )
لا تداعوْا بالألقاب المستكرَهة
كثيرًا من الظنّ ( ۱۲ )
هو ظنّ السوء بأهل الخير
لا تجسّـسوا ( ۱۲ )
لا تتبعوا عوْرات المسلمين
فكرهتموه ( ۱۲ )
فقد كرهتموه فلا تفعلوه
آمنّـا ( ۱٤ )
صدقنا بقلوبنا و ألسنتنا
لمْ تؤمنوا ( ۱٤ )
لمْ تصدقوا بقلوبكم
أسْلمْـنـَـا ( ۱٤ )
استسلمنا خَوفا و طمعًا
لا يَلتكمْ ( ۱٤ )
لا ينْقصكمْ
أتعلـّـمون الله بدينكم ( ۱٦ )
أتخبرونه بقوْلكم آمنّا
