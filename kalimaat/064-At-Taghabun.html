٦٤ - سورة التغابن
( ۱ ) يُسبّح لله . .
يُنـزّهه و يُمَجّده تعالى و يدلّ عليه
( ۱ ) له المُلك
التـّصرف المطلق في كلّ شيء
( ۳ ) بالحقّ
بالحِـكمة البالغة
( ۳ ) فأحسن صُوَركم
أتقـَنـَـها و أحكمها
( ۵ ) وَبال أمرهم
سوء عاقبة كفرهم في الدنيا
( ٦ ) تـَـولّوْا
أعرضوا عن الإيمان بالرسل
( ۸ ) النور
القرآن
( ۹ ) ليَوْم الجمع
في يوم القيامة حيث تجتمع الخلائق للحساب و الجَزاء
( ۹ ) يوم التغابن
يَظهر فيه غَـبْن الكافر بتـَركِه الإيمان و غبن المؤمن بتقصيره في الإحسان
( ۱۱ ) بإذن الله
بإرادته و قضائه و قـَـدَره تعالى
( ۱۱ ) يَهْدِ قـَـلبَه
يوفّـقه لليَقين و الصّـبر و التـّـسليم
( ۱۵ ) فِـتنـَـة ٌ
بلاءٌ و مِحنة و اختبار
( ۱٦ ) يُوق شُحّ نفسِه
يُـكْفَ بُخلَـهَا الشديد مع حِرْصها
( ۱۷ ) قـَـرْضًا حَسَنا
احتسابا بطيبة نفس و إخلاص
