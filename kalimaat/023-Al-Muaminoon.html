۲۳ - سورة المؤمنون
( ۱ ) أفلح المؤمنون :فازوا وسعِدوا ونجَوْا
( ۲ ) خاشعون :متُذللون خائفون ساكنون
( ۳ ) اللغو :مالا يَجْمُـلُ من القول والفعل
( ۷ ) العادُون :المُجاوزون الحلال إلى الحرام
( ۱۱ ) الفردوس :أعلى الجنان و أوسطها و أفضلها
( ۱۲ ) سُلالة :خُلاصة ( مائية مكونة من الغذاء )
( ۱۳ ) قرار مكين :مستقر مُـتمكن وهو الرحم
( ۱٤ ) علقة :دما متجمدا
( ۱٤ ) مُضغة :قطعة لحم قـَـدْر ما يُمضغ
( ۱٤ ) خلقا آخر :مباينا للأول بنفخ الروح فيه
( ۱٤ ) فتبارك الله :فتعالى . أو تكاثر خيره وإحسانه
( ۱٤ ) أحسن الخالقين :أتقن الصانعين . أو المصورين
( ۱۷ ) سبع طرائق :سبع سموات طباقا أو طُرُقا للملائكة أو للكواكب في مسيرها
( ۱۸ ) بقدَر :بمقدار الحاجة والمصلحة
( ۲۰ ) شجرة :هي شجرة الزيتون
( ۲۰ ) بالدهن :مُـلتبسًا ثمرها بالزيت
( ۲۰ ) صبغ للآكلين :إدام لهم يغمس فيه الخبز
( ۲۱ ) الأنعام :الإبل والبقر والضأن والمعز
( ۲۱ ) لعبرة :لعظة وآية على القدْرَة والرّحمة
( ۲۲ ) وعليها :وعلى الإبل منها
( ۲٤ ) الملأ :وُجوه القوْم و سَادَتهم
( ۲٤ ) يتفضل عليكم :يترأس ويَشْرُف عليكم
( ۲۵ ) به جـِـنّة :به جنون أو جنّ يَخْـبَـلونه
( ۲۵ ) فتربصوا به :انتظروا و اصبروا عليه
( ۲۷ ) بأعْـيُـننا :برعايتنا وكلاءَتِنا
( ۲۷ ) فار التنور :نبع الماء من التنور المعروف
( ۲۷ ) فاسْلك فيها :فأدخل في الفلك
( ۲۹ ) مُنزلا :إنزالا . أو مكان إنزال
( ۳۰ ) لمُبتلين :لمُختبرين عبادنا بهذه الآيات
( ۳۱ ) قرنا آخرين :هم عادٌ الأولى قوم هود
( ۳۳ ) أتـْـرفـناهم :نعمناهم و وَسّعْـنا عليهم فبَطِـروا
( ۳٦ ) هيهات :بعُد وقوع ذلك الموعود
( ٤۱ ) فأخذتهم الصيحة :صيحة جبريل أو العذاب المُصْــــَطـلِـم
( ٤۱ ) فجعلناهم غثاء :هالكين كغثاء السيل ( حَمِيله )
( ٤۱ ) فبُعدا :هلاكا .. أو بُعدا من الرحمة
( ٤۲ ) قرونا آخرين :أمَمًا أخرى
( ٤٤ ) تتـْـرى :متتابعين على فترات
( ٤٤ ) جعلناهم أحاديث :مجرد أخبار للتعَجّب والتلهّي
( ٤۵ ) سُلطان مبين :برهان بَيّن ٍ مُظهر للحق
( ٤٦ ) قوْما عالين :متكبرين أو متطاولين بالظلم
( ۵۰ ) آويناهُما :صيّرناهُما وأوْصَـلناهُما
( ۵۰ ) إلى رَبوة :إلى مكان مرتفع من البلاد
( ۵۰ ) معين :ماءٍ جَار ٍ ظاهر للعيون
( ۵۲ ) أمتكم :مِلـّـتكم وشريعتكم
( ۵۳ ) فتقطعوا أمرهم :تفرقوا في أمْر دينهم
( ۵۳ ) زبُرا :قطعا وفرقا وأحزابا مختلفة
( ۵٤ ) غَمْرَتِهم :جَهالتهم وضلالتهم
( ۵۵ ) أنّ ما نمدهم به :ما نجعله مدَدا لهمْ
( ۵۷ ) مُشفقون :خائفون حَذِرون
( ٦۰ ) يُؤتون ما آتوْا :يُعطون ما أعْـــَطوْا من الصّدقات
( ٦۰ ) قلوبهم وجلة :خائفة ألا تقبل أعمالهم
( ٦۲ ) وسعها :قدر طاقتها من الأعمال
( ٦۳ ) غمرة :جهالة وغفلة وغطاء
( ٦٤ ) مترفيهم :مُنعّميهم الذين أبْطرَتـْـهم النِّعَم
( ٦٤ ) يجأرون :يصرُخون مستغيثين بربّهم
( ٦٦ ) تنكِصون :ترجعون معرضين عن سماعها
( ٦۷ ) مستكبرين به :مستعظمين بالبيت الحرام
( ٦۷ ) سامرا :سُمّارًا حوْله بالليل
( ٦۷ ) تهجرون :تهْذون بالطعن في القرآن
( ۷۰ ) به جـِـنَّة :به جنون
( ۷۱ ) بذِكرهم :بفخرهم وشرفهم وهو القرآن
( ۷۲ ) خَرْجا :جُعلا و أجْرًا من المال
( ۷٤ ) لناكبون :لعادِلون عن الحق زائغون
( ۷۵ ) للجّوا في طغيانهمْ :لتمادوْا في ضلالهم وكفرهم
( ۷۵ ) يعمهون :يعمون عن الرشد أو يتحيرون
( ۷٦ ) فما استكانوا :فما خضعوا وأظهروا المسكنة
( ۷٦ ) ما يتضرعون :ما يتذللون له تعالى بالدعاء
( ۷۷ ) مُبلسون :مُتحيرون آيسون من كل خير
( ۷۹ ) ذرأكم :خلقكم وبثكم بالتناسل
( ۸۳ ) أساطير الأولين :أكاذيبهم المسطورة في كتبهم
( ۸۸ ) ملكوت :هو الملك الواسع العظيم
( ۸۸ ) هو يجير :يُغيث ويحمى من يشاءُ ويمنع
( ۸۸ ) لا يُجار عليه :لا يُغاث أحد منه ولا يُمنع
( ۸۹ ) فأنى تـُـسحرون :فكيف تـُـخدعون عن توحيده؟
( ۹۷ ) أعوذ بك :أعتصم وأمتنع بك
( ۹۷ ) همزات الشياطين :نزغاتِهم و وَساوسهم المُغْرية
( ۱۰۰ ) من ورائهم :أمامهم
( ۱۰۰ ) برزخ :حاجزٌ دون الرّجعة
( ۱۰٤ ) تلفح :تحرق
( ۱۰٤ ) كالِحون :عابسون أو متقـلـّـصو الشفاه عن الأسنان من أثر اللفح
( ۱۰٦ ) غلبت علينا :استولت علينا وملكتنا
( ۱۰٦ ) شِقوتـُـنا :شقاوَتنا . أو لـَـذاتنا وشَهواتنا
( ۱۰۸ ) اخسئوا فيها :انزجـِـروا و ابْعُدُوا كالكلاب
( ۱۱۰ ) سِخريّـا :مهزُوءًا بهم
( ۱۱٦ ) فتعالى اللهُ :ارتفع بعظمته وتنزّه عن العَبَث
