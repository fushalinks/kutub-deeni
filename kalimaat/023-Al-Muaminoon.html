هلاكا .. أو بُعدا من الرحمة

أمَمًا أخرى
( ٤٤ ) تتـْـرى
متتابعين على فترات
( ٤٤ ) جعلناهم أحاديث
مجرد أخبار للتعَجّب والتلهّي

برهان بَيّن ٍ مُظهر للحق
( ٤٦ ) قوْما عالين
متكبرين أو متطاولين بالظلم

فما خضعوا وأظهروا المسكنة

( ۷۷ ) مُبلسون

( ۷۹ ) ذرأكم
