٤۲ - سورة الشورى
( ۵ ) يتـفطّـرن :يتشقـّـقـْـنَ من عظمته تعالى و جلاله
( ٦ ) أوْلياءَ :معْبودات يزعمون نصْرَتـَـها لهم
( ٦ ) الله حفيظ عليهم :رقيبٌ على أعمالهم و مُجَازيهمْ
( ٦ ) بوَكيل :بمَوْكولٍ إليك أمرهمْ
( ۷ ) أمّ القرى :مَكّة: أي أهلها
( ۷ ) يَوم الجَمع :يومَ القيامة لاجتماع الخلائق فيه
( ۱۰ ) إليه أنيبُ :إليه أرجع في كلّ الأمور
( ۱۱ ) فاطِرُ . . :مبْدع و مخترع . .
( ۱۱ ) من أنفسكم أزْواجًا :حلائل
( ۱۱ ) من الأنعام أزواجًا :أصنافا ذكورا و إناثا
( ۱۱ ) يَذرَؤكمْ فيه :يَـذرؤكم فيه
( ۱۲ ) له مقاليد . . :له مقاليد . .
( ۱۲ ) يَقدِرُ :يَقدِر
( ۱۳ ) شـَـرَعَ لكم . . :شرَعَ لكم . .
( ۱۳ ) ما وصّى :ما وصّى
( ۱۳ ) أقيموا الدّين :أقيموا الدّين
( ۱۳ ) كَـبُـرَ. . :كَبُرَ. .
( ۱۳ ) يَـجْـتـَـبي :يجْتـَـبي
( ۱۳ ) يُنيبُ :ينيب
( ۱٤ ) بَغْياً بَيْـنـَـهمْ :بَغيا بَينهمْ
( ۱٤ ) مُريبٍ :مريب
( ۱۵ ) اسْقِـمْ :استقمْ
( ۱۵ ) لا حُجّة :لا حجّة
( ۱٦ ) استجيب له :استجيبَ له
( ۱٦ ) حُجّـتهم داحضة :حُجّـتـهمْ دَاحِضة
( ۱۷ ) الميزان :العدّل و التـّسوية في الحقوق
( ۱۸ ) مُشفقون منها :خائفون منها مع اعتنائهم بها
( ۱۸ ) يُمَارون في الساعة :يُجَادلون . أوْ يَشكّونَ فيها
( ۱۹ ) لطيف بعباده :بَرٌّ رفيقٌ بهم
( ۲۰ ) حَرْث الآخرة :ثوَابَها الموعود . أو العمل لها
( ۲۱ ) كلمة الفَصْـل :الحكم بتأخير العذاب للآخرة
( ۲۲ ) رَوضات الجنّـات :مَحَاسنها و مَلاذها أو أطيب بـِـقاعها و أنزهِها
( ۲۳ ) يقترف حسنة :يَكـتـَـسِبْ طاعة ً
( ۲۷ ) لبَغَوْا :لـَـطغَـوْا و تجـبّـروا . أو لظلموا
( ۲۷ ) يُـنـَـزّل بـِقـَـدَرٍ :بتقدير حكيم مُحْكَمٍ
( ۲۸ ) قـَـنـَطوا :يَـئـسوا من نزوله
( ۲۹ ) بَثّ فيهما :فرّق و نـَـشرَ فيهما
( ۳۱ ) بمُعْجزينَ :بفائتينَ من العذاب بالهرَب
( ۳۲ ) الجوَار :السّـفن الجَاريَة
( ۳۲ ) كالأعلام :كالجبال . أو القصور العالية
( ۳۳ ) فيظللن رَواكدَ :فيَـصِرْن ثوَابت سَوَاكن
( ۳٤ ) يوبقـْـبهنّ :يُهلكْهنّ بالغرَق أي أهلهنّ
( ۳۵ ) مَحيص :مَهْرَب و مَخلَصٍ من العذاب
( ۳۷ ) الفواحش :مَا عَــُظمَ قبْحه من الذنوب
( ۳۸ ) أمرهمْ شورى :يَتشاوَرون و يَترَاجعون فيه
( ۳۹ ) أصابهم البَغْي :نـَـالهم الظلْم و العدْوَان
( ۳۹ ) ينتصرون :ينتقمون ممّن ظلمهمْ و لا يَعْـتدون
( ٤۲ ) يَبْـغون في الأرض :يُـفـسدون . أو يَتجبّـرون فيها
( ٤۵ ) خاشعين :خاضعين متضائلين
( ٤۵ ) ينظرون منْ طرْفٍ خفيّ :يُسَارقون النّظر منْ شدّة الخوف
( ٤۷ ) نكير :إنكار لذنوبكمْ أو مُنْكر لعَذابكم
( ٤۸ ) فرح بها :بَطِرَ لأجْـلِها
( ۵۲ ) روحا :قرآناً . أو نبُـوّة أو جبريل
( ۵۲ ) الإيمان :الشّرائع الـتـّـفصيلـيّـة التي لا تـُـعْـلـَـم إلا بالوَحْي
( ۵۲ ) صراطٍ مستقيم :دينٍ قويم (دين الإسلام)
