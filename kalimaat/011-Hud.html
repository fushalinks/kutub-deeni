۱۱ - سورة هود
( ۱ ) أُحكمت آياته :نُظمت نظما مُحكما رصينا
( ۱ ) فصّلت :فرّقت في التّنزيل نجوما بالحكمة
( ۵ ) يثنون صدورهم :يطوونها على الكفر و العداوة
( ۵ ) ليستخفوا منه :من الله تعالى جهلا منهم
( ۵ ) يستغشون ثيابهم :يتغطّون بها مُبالغة في الاستخفاء
( ٦ ) يعلم مُستقرّها :موضع استقرارها في الأصلاب ، أو في الأرحام و نحوها
( ٦ ) مستودعها :موضع استيداعها في الأرحام و نحوها ، أو في الأصلاب
( ۷ ) ليبلُوكم :ليختبركم و هو أعلم بأمركم
( ۷ ) أحسن عملا :أطوع لله و أروع عن محارمه
( ۸ ) أمّة معدودة :طائفة من الأيّام قليلة

( ۱۰ ) ضرّاء مسّته :نائبة و نكبة أصابته
( ۱۰ ) إنه لفرح :لَبَطِرٌ بالنّعمة ، مغترّ بها
( ۱۰ ) فخور :على النّاس بما أوتي من النعماء
( ۱۲ ) وكيل :قائمٌ به حافظ له
( ۱۵ ) لا يُبخسون :لا يُنقصون شيئا من أُجور أعمالهم
( ۱٦ ) حبط :بَطَلَ في الآخرة
( ۱۷ ) بيّنة :يقين و برهان واضح و هو القرآن
( ۱۷ ) شاهد :على تنزيله و هو اعجاز نظمه
( ۱۷ ) مرية منه :شكّ من تنزيله من عند الله
( ۱۸ ) الأشهاد :الملائكة و النّبيّون و الجوارح
( ۱۹ ) يبغونها عوجا :يطلبونها معوجّة أو ذات اعوجاج
( ۲۰ ) مُعجزين :فائتين من عذاب الله بالهرب
( ۲۲ ) لا جرَمَ :حقّ و ثبت أو لا محالة أو حقّا
( ۲۳ ) أخبتوا إلى ربّهم :اطمأنّوا على وعده أو خشعوا له
( ۲۷ ) الملأُ :السادة و الرّؤساء
( ۲۷ ) بادي الرأي :ظاهره دون تعمّق و تثبّت
( ۲۸ ) أرأيتم :أخبروني
( ۲۸ ) فعُمّيت عليكم :أُخفيت عليكم
( ۳۱ ) خزائن الله :خزائن رزقه و ماله
( ۳۱ ) تزدري أعينكم :تستحقرهم و تستهين بهم
( ۳۳ ) ما أنتم بمُعجزين :بفائتين من عذاب الله بالهرب
( ۳٤ ) أن يُغويكم :يضلّكم
( ۳۵ ) فعليّ إجرامي :عقاب اكتساب ذنبي
( ۳٦ ) فلا تبتئس :فلا تحزن
( ۳۷ ) بأعيننا :بحفظنا و كلاءتنا الكاملين
( ۳۹ ) يُخزيه :يذلّه و يُهينه
( ۳۹ ) يحلّ عليه :يجب عليه و ينزل به
( ٤۰ ) فار التّنّور :نبع الماء و جاش بشدّة من تنّور الخبز المعروف
( ٤۱ ) مجريها :وقت إجرائها
( ٤۱ ) مُرساها :وقت إرسائها
( ٤۳ ) سآوي :سألتجئُ و أستندُ
( ٤۳ ) لا عاصم :لا مانع و لا حافظ

( ٤٤ ) غيظ الماء :نقص و ذهب في الأرض
( ٤٤ ) استوت على الجوديّ :استقرّت على جبل بقُرب الموصل
( ٤٤ ) بُعدا :هلاكا و سُحقا
( ٤۸ ) بركات :خيرات ثابتة نامية

( ۵٤ ) اعتراك :أصابك
( ۵٤ ) بسوء :بجنون و خَبَل
( ۵۵ ) فكيدوني :فاحتالوا في كيدي و ضرّي
( ۵۵ ) لا تُنظرون :لا تمهلوني
( ۵٦ ) آخذ بناصيتها :مالكها و قادر عليها
( ۵۷ ) حفيظ :رقيب مُهيمن
( ۵۸ ) غليظ :شديد مُضاعف
( ۵۹ ) جبّار :مُتعاظم مُتكبّر
( ۵۹ ) عنيد :طاغٍ معاند للحقّ مُجانب له
( ٦۰ ) بُعدا لعاد :هلاكا و سُحقا لهم
( ٦۱ ) استعمركم فيها :جعلكم عُمّارها و سُكاّنها

( ٦۳ ) أرأيتم :أخبروني

( ٦۳ ) تخسير :خسرانٍ إن عصيته
( ٦٤ ) آية :مُعجزة دالّة على صدق نبوّتي
( ٦۷ ) الصّيحة :صوت من السّماء مُهلك

( ٦۸ ) بُعدا لثمود :هلاكا و سُحقا لهم
( ٦۹ ) بعجل حنيذٍ :مشويّ بالحجارة المحمّاة في حُفرة
( ۷۰ ) نكِرهم :أنكرهم و نفر منهم
( ۷۰ ) أوجس منهم خيفة :أحسّ في قلبه منهم خوفا
( ۷۲ ) يا ويلتا :كلمة تعجّب
( ۷۳ ) مجيد :كثير الخير و الإحسان
( ۷٤ ) الرّوع :الخوف و الفزع
( ۷۵ ) لحليم :مُتأنّ غير عجول
( ۷۵ ) أوّاه :كثير التّأوّه من خوف الله
( ۷۵ ) منيب :راجع إلى الله سبحانه
( ۷۷ ) سيء بهم :نالته المساءة يمجيئهم خوفا عليهم
( ۷۷ ) ضاق بهم ذرعا :ضعُفت طاقته عن تدبير خلاصهم
( ۷۷ ) يوم عصيب :شديد شرّه و بلاؤه
( ۷۸ ) يُهرعون إليه :يُسرعون إليه كأنّهم يُدفعون
( ۷۸ ) هؤلاء بناتي هن أطهر لكم :
( ۷۸ ) لا تُخزون :لا تفضحوني و لا تُهينوني
( ۷۹ ) من حقّ :من حاجة و أَرَب
( ۸۰ ) آوي إلى ركن :أنضمّ إلى قويّ أنتصر به عليكم
( ۸۱ ) بقطعٍ من الليل :بطائفة منه أو من آخره
( ۸۲ ) سجيل :طين طُبخ بالنّار كالفخّار
( ۸۲ ) منضود :متتابع أو مجموع معدّ للعذاب
( ۸۳ ) مسوّمةً :معلمة العذاب
( ۸٤ ) أراكم بخير :بسعة نُغنيكم عن التطفيف
( ۸٤ ) يوم مُحيط :مُهلك

( ۸٦ ) بقيّة الله :ما أبقاه لكم من الحلال
( ۸٦ ) بحفيظ :برقيب فأجازيكم بأعمالكم
( ۸۸ ) أرأيتم :أخبروني
( ۸۸ ) بيّنة :هداية و بصيرة
( ۸۹ ) لا يجرمنّكم :لا يكسبنّكم أو لا يحملنّكم
( ۹۱ ) رهطك :جماعتك و عشيرتك
( ۹۲ ) وراءكم ظهريّا :منبوذا وراء ظهوركم منسيّا
( ۹۳ ) مكانتكم :غاية تمكّنكم من أمركم
