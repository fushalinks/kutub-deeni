۱٤ - سورة ابراهيم
( ۱ ) بإذن ربّهم :بتيسيره و توفيقه لهم أو بأمره
( ۱ ) العزيز :الغالب أو الذي لا مثل له
( ۱ ) الحميد :المحمود الُثنى عليه
( ۲ ) ويلٌ :هلاك أو حسرة أو وادٍ في جهنّم
( ۳ ) يستحبّون :يختارون و يؤثرون
( ۳ ) يبغونها عوجا :يطلبونها مُعوجّة أو ذات اعوجاج
( ۵ ) بأيّام الله :بنعمائه أو وقائعه في الأمم الخالية
( ٦ ) يسومونكم :يذيقونكم و يكلّفونكم
( ٦ ) يستحيون نساءكم :يستبقون بناتكم للخدمة
( ٦ ) بلاء :ابتلاء بالنّعم و النّقم
( ۷ ) تأذّن ربّكم :أعلم اعلاما لا شُبهة معه
( ۹ ) فردّوا أيديهم في أفواههم :عظّوا على أناملهم تغيّظا من الرّسل و كلامهم
( ۹ ) مريبٍ :موقعٍ في الرّيبة و القلق
( ۱۰ ) فاطر. . :مُبدع و مُخترع . .
( ۱۰ ) بسلطان :حجّة و برهان على صدقكم
( ۱٤ ) خاف مقامي :موقفه بين يديّ للحساب
( ۱۵ ) استفتحوا :استنصر الرّسل بالله على الظالمين
( ۱۵ ) خاب كلّ جبّار :خسر و هلك كلّ مُتعاظم متكبّر
( ۱۵ ) عنيدٍ :معاند للحقّ ، مُجانب له
( ۱٦ ) صديدٍ :ما يسيل من أجساد أهل النّار
( ۱۷ ) يتجرّعه :يتكلّف بلعه لحرارته و مرارته
( ۱۷ ) لا يكاد بسيغه :يبتلعه لشدّة كراهته و نتنه
( ۱۸ ) يوم عاصف :شديد هبوب الرّيح
( ۲۱ ) برزوا :خرجوا من القبور للحساب
( ۲۱ ) مُغنون عنّا :دافِعون عنّا
( ۲۱ ) محيصٍ :منجى و مهرب و مزاغ
( ۲۲ ) سلطان :تسلّط أو حجّة
( ۲۲ ) بمصرخكم :بمُغيثكم من العذاب
( ۲۲ ) بمصرخيّ :بمغيثيّ من العذاب
( ۲۵ ) كلمةً طيّبةً :كلمة التوحيد و الإسلام
( ۲۵ ) تُؤتي أُكلها :تعطي ثمرها الذي يُؤكل
( ۲٦ ) كلمة خبيثة :كلمة الكفر و الضّلال
( ۲٦ ) اجتُثّت :اقتُلِعت جثّتها من أصلها
( ۲۷ ) في الحياة الدّنيا :في القبر عند السّؤال
( ۲۸ ) دار البوار :دار الهلاك ( جهنّم)
( ۲۹ ) يصلونها :يدخلونها أو يُقاسون حرّها
( ۳۰ ) أندادا :أمثالا من الأوثان يعبدونها
( ۳۱ ) لا خِلالٌ :لا مُخالّة و لا مُوادّة
( ۳۳ ) دائبين :دائمين في منافعهما لكم
( ۳٤ ) لا تُحصوها :لا تطيقوا عدّها لعدم تناهيها
( ۳۵ ) اجنبني :أبعِدني و نحّني
( ۳۷ ) تهوي إليهم :تسرع إليه شوقا و وٍدادا
( ٤۲ ) تشخص فيه الأبصار :ترتفع دون أن تطرف من الهول
( ٤۳ ) مهطعين :مسرعين إلى الدّاعي بذلّة
( ٤۳ ) مقنعي رءوسهم :رافعيها مديمي النّظر للأمام
( ٤۳ ) أفئدتهم هواء :قلوبهم خالية لا تعي لفرط الحيرة
( ٤۸ ) برزوا لله :خرجوا من القبور للحساب
( ٤۹ ) مقرّنين :مقرونا بعضهم مع بعض
( ٤۹ ) الأصفاد :القيود أو الأغلال
( ۵۰ ) سرابيلهم :قمصانهم أو ثيابهم
( ۵۰ ) تغشى وجوههم :تغطّيها و تجلّلها
( ۵۲ ) بلاغ للنّاس :كفاية في الغظة و التذكير
