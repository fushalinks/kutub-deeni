۱۰ - سورة يونس
( ۲ ) قَدَم صدق :سابقَةَ فضل ، و منزلة رفيعة
( ۳ ) استوى على العرش :استواءً يليق به سبحانه
( ٤ ) بالقسط :بالعدل
( ٤ ) حميم :ماءٍ بالغ غاية الحرارة
( ۵ ) قدّره منازل :صيّر القمر ذا منازل يسير فيها
( ۷ ) لا يرجون لقاءنا :لا يتوقّعونه لإنكارهم البعث
( ۱۰ ) دعواهم :دُعاؤهم
( ۱۱ ) لَقُضي إليهم أجلهم :لأُهلِكوا و أبيدوا
( ۱۱ ) في طُغيانهم :في تجاوزهم الحدّ في الكفر
( ۱۱ ) يعهمون :يعمون عن الرّشد أو يتحيّرون
( ۱۲ ) الضّرّ :الجهد و البلاء و الشّدّة
( ۱۲ ) دعانا لجنبه :استغاث بنا لكشفه ملقى لجنبه
( ۱۲ ) مرّ :استمرّ على كُفره و لم يتّعظ
( ۱۳ ) القرون :الأمم كقوم نوح و عاد و ثمود
( ۱۳ ) ظلموا :بالكفر و تكذيب الرّسل
( ۱٤ ) جعلناكم خلائف :استخلفناكم بعد اهلاك أولئك
( ۱٦ ) لا أدراكم به :لا أعلمكم الله به بواسطتي
( ۱۷ ) لا يُفلح المجرمون :لا يفوزون بمطلوب
( ۱۸ ) سبحانه :تنزيها له تعالى
( ۲۱ ) ضرّاء مسّتهم :نائبة أصابتهم (الجوع و القحط)
( ۲۱ ) لهم مكر :دفع و طعن و استهزاء
( ۲۱ ) الله أسرع مكرا :أعجلُ جزاءً و عقوبة
( ۲۲ ) ريح عاصف :شديدة الهبوب
( ۲۲ ) أحيط بهم :أحدق بهم الهلاك
( ۲۳ ) يبغون :يُفسدون
( ۲٤ ) مثل الحياة الدّنيا :حالها في سرعة تقضيها و زوالها
( ۲٤ ) زخرفها :نظارتها و بهجتها بألوان النّبات
( ۲٤ ) أمرُنا :ما يجتاحها من الآفات و العاهات
( ۲٤ ) حصيدا :كالنّبات المحصود بالمناجل
( ۲٤ ) لم تغْنَ :لم تمكث زروعها و لم تُقم
( ۲٦ ) الحسنى :المنزلة الحُسنى (الجنّة)
( ۲٦ ) زيادة :النّظر إلى وجه الله الكريم فيها
( ۲٦ ) لا يرهق وجوههم :لا يغشى وجوههم و لا يعلوها
( ۲٦ ) قتر :غُبار ما فيه سواد
( ۲٦ ) ذلّة :أثر هوان ما
( ۲۷ ) عاصم :مانع يمنع سُخطه و عذابه
( ۲۷ ) أغشيت وجوههم :كُسيت و ألبست
( ۲۸ ) مكانكم :الزموا مكانكم و اثبتوا فيه
( ۲۸ ) فزيّلنا بينهم :فرّقنا بينهم و قطعنا وُصَلَهم
( ۳۰ ) تبلو :تخبُر . أو تعلم . أو تُعاين
( ۳۲ ) ربّكم الحقّ :الثابتة ربوبيّته بالبرهان ثبوتا لا ريب فيه
( ۳۲ ) فأنّى تُصرفون؟ :فكيف تستجيزون العدول عن الحقّ إلى الكفر و الضّلال ؟
( ۳۳ ) حقّت :ثبتت و وجبت
( ۳٤ ) فأنّى تؤفكون ؟ :فكيف تُصرفون عن طريق الرّشد ؟
( ۳۵ ) لا يهِدّي :لا يهتدي بنفسه
( ۳۹ ) يأتهم تأويله :يتبيّن لهم عاقبته و مآل وعيده
( ٤۳ ) ينظر إليك :يُعاين دلائل نبوّته الواضحة
( ٤۷ ) بالقسط :بالعدل في الدّنيا أو يوم الجزاء
( ۵۰ ) أرأيتم :أخبروني عن عذاب الله
( ۵۰ ) بياتاً :وقت بيات أي ليلا
( ۵۱ ) آلآن ؟ :آلآن تُؤمنون بوقوع عذابه ؟
( ۵۳ ) يستنبئونك :يستخبرونك مُستهزئين عن العذاب
( ۵۳ ) إي و ربّي :نعم و ربّي
( ۵۳ ) و ما أنتم بمعجزين :بفائتين من عذاب الله بالهرب
( ۵٤ ) أسرّوا النّدامة :أخفوا الغمّ و الحسرة
( ۵۹ ) أرأيتم :أخبروني
( ۵۹ ) أذِن لكم :أعلمكم بهذا التحليل و التّحريم
( ۵۹ ) تفترون :تكذبون في نسبة ذلك إليه
( ٦۱ ) تكون في شأن :في أمر هامّ مُعتنى به
( ٦۱ ) تُفيضون فيه :تشرعون و تخوضون فيه
( ٦۱ ) لَا يَعْزُبُ عَنْهُ مِثْقَالُ ذَرَّةٍ :ما يبعُد و ما يغيب
( ٦۱ ) مثقال ذرّة :وزن أصغر نملة أو هباءة
( ٦۵ ) إنّ العزّة لله :إن القهر و الغلبة له تعالى في مُلكه
( ٦٦ ) يخرصون :يكذبون فيها ينسبونه إليه تعالى
( ٦۸ ) سبحانه :تنزيها له تعالى عما نسبوه إليه
( ٦۸ ) سلطان :حجّة و برهان
( ۷۱ ) كبُر عليكم :عظُم و شقّ عليكم
( ۷۱ ) مقامي :اقامتي بينكم دهرا طويلا
( ۷۱ ) فأجمعوا أمركم :اعزموا و صمّموا على كيدكم
( ۷۱ ) و شُركاءكم :مع شركائكم
( ۷۱ ) غمّة :ضيقا شديدا . أو مُبهما مُلتبِسا
( ۷۱ ) اقضوا إليّ :أدّوا إليّ ما تُريدونه
( ۷۱ ) لا تُنظرون :لا تُمهلوني
( ۷۳ ) جعلناهم خلائف :يخلُفون المُغرقين
( ۷٤ ) نطبع :نختم
( ۷۸ ) لتلفتنا :لِتلوينا و تصرفنا
( ۸۳ ) أن يفتنهم :أن يبتليهم و يعذّبهم
( ۸۵ ) لا تجعلنا فتنة :موضع عذاب
( ۸۷ ) تبوّءا لقومكما . . :اتّخذا و اجعلا لهم . .
( ۸۷ ) قبلة :مساجد نحو الكعبة أو مُصلّى
( ۸۸ ) اطمس على أموالهم :أهلكها و أذهِبها . أو أتلِفها
( ۸۸ ) اشدد على قلوبهم :اطبع عليها
( ۹۰ ) بغيا و عدوا :ظلما و اعتداءًا
( ۹۱ ) آلآن ؟ :آلآن تؤمنوا حين أيقنت بالهلاك ؟
( ۹۲ ) آية :عبرة و نكالا
( ۹۳ ) بوّأنا :أنزلنا و أسكنّا
( ۹۳ ) مُبوّأ صدق :منزلا صالحا مرضيّا
( ۹٤ ) الممترين :الشّاكّين المتزلزلين
( ۹۸ ) عذاب الخزي :الذّلّ و الهوان
( ۱۰۰ ) يجعل الرّجس :العذاب . أو السّخط
( ۱۰۵ ) أقم وجهك للدّين :اصرف ذاتك كلّها للدّين الحنيفيّ
( ۱۰۵ ) حنيفا :مائلا عن الأديان الباطلة كلّها
( ۱۰۸ ) بوكيل :بحفيظ موكولٍ إليّ أمركم
