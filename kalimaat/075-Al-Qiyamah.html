۷۵ - سورة القيامة
( ۱ ) لا أقسِم :أقْـسِم . و "لا" مزيدة
( ۲ ) بالنـّـفس الـلـّـوّامة :كثيرة الـلـّـوْم و النّـدَم على ما فات
( ٤ ) بَلـَـى :نجمَعُهَا بَعْدَ التـّـفـرّق و البـِـلـَـى
( ٤ ) نُسوّي بَنـَـانه :أطراف أصابعه فنـَرُدّ عِظامَها كما كانت على صِغَرها بقـُـدْرَتنا فكيف بكِبارها
( ۵ ) لِـيَـفْجُر أمامه :لِيَـدزم على فجوره مُدّة عُـمْره
( ۷ ) بَرقَ البَصر :دَهِشَ و تـَـحَـيّرَ فَزَعًا مما رأى
( ۸ ) خَسَف القمَر :ذهَب ضوءُه
( ۹ ) جُمِعَ الشّمس و القمَر :في الطلوع من المغْرب مُظلمَيْن
( ۱۰ ) أيْنَ المَـفـرّ ؟ :المَهْرَب مِن العذاب أو الهَوْل
( ۱۱ ) لا وَزَر :لا مَلْجَأ و لا مَنـْـجَى له مِنَ الله
( ۱٤ ) بَصيرَة :حُجّة بَيّـنـَـة أو عَيْنٌ بَصيرة
( ۱۵ ) لوْ ألقى مَعَـاذيرَه :لوْ جاءَ بكلّ عُذر لم يَنفعْه
( ۱۷ ) جَـمَـعَه :في صَدْرك و حِفْـظِكَ إيّـاه
( ۱۷ ) قرآنه :أنْ تـَقـْرَأه بلسانِك مَتى شِئت
( ۱۸ ) قرأناه :أتمَمْنا قراءَته عليك بلسام جبريل
( ۱۹ ) بَيَانه :تـَـفـْـسير ما أشكل مِنْ مَعانيه
( ۲۲ ) ناضِـرَة :حَسَنة مُشْرقة مُتهلـّـلة
( ۲٤ ) باسِـرَة :شديدة الكلوحَة و العُبُوس
( ۲۵ ) فَاقِـرَة :داهية عظيمة تـقـْـصِم فَـقـَــار الظهْر
( ۲٦ ) بَلغَتِ التـّـراقي :وَصَلتِ الرّوح لأعلى الصّـدْر
( ۲۷ ) منْ راق ؟ :مَنْ يُـداويه و ينجيه من الموت؟
( ۲۹ ) التفّـت. . :الـْتـَـوَتْ. أو الْـتـَـصَقت . .
( ۳۰ ) المَساق :سَوْق العِبَـاد للجزاء
( ۳۳ ) يَتمَطى :يَـتبَختر في مِشيَـتِه اخْتِـيالا
( ۳٤ ) أوْلى لك :قارَبَك ما يُهْـلِـكُـك
( ۳٦ ) يُـتـْـرَك سُـدًى :مُهْمَلا فلا يُكلـّـف و لا يُجَـازَى
( ۳۷ ) مَنِيّ يُمْـنـَى :يُصَبّ في الرّحم
( ۳۸ ) فَسَوّى :فعَدّله و كَمّـله و نـَـفَخَ فـيه الرّوح
