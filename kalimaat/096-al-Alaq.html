۹٦ - سورة العلق
( ۲ ) عَلق :دَمٍ جامِدٍ استحالَ إليه المَنِىُّ
( ٤ ) علـّـم :عَلَّـمَ الإنسانَ الكتابةَ بالقلم
( ٦ ) كلاّ :حقـّـًـا
( ٦ ) ليَطغَى :لَيُجَاوزُ الحدَّ فى العِصيان
( ۸ ) الرّجْعى :الرُّجُوعَ فى الآخرة للجزاء
( ۹ ) أرَأيت :أخْبـِرْني
( ۱۵ ) لنـَـسْـفَـعن بالناصية :لـَـنـَسحَبَـنـَّه بناصِيـتِه إلى النار
( ۱۷ ) فليَدْع نادِيَه :أهلَ مَجلِسِه من قَومِه وعَشِيَرتِه
( ۱۸ ) سَندْع الزبانية :ملائكةَ العذابِ لِجَرِّه إلى النار
