۸۷ - سورة الأعلى
سبّح اسم ربّك ( ۱ )
نَزِّهُه ومَجِّدهُ تعالى عمَّا لا يَليق به
خَلق ( ۲ )
أَوْجَدَ كلَّ شىءٍ بقدرتِه
فسَوّى ( ۲ )
بَينَ خَلْقِه فى الإحْكام والإتقان
قـَـدّر ( ۳ )
جعلَ الأشياءَ على مَقادِيرَمَخصوصهٍ
فهَـدَى ( ۳ )
فَوجَّه كلَّ واحدٍ منها إلى ما يَنبَغى له
أخْرَجَ المرعى ( ٤ )
أنـْـبَتَ العُشبَ رَطْبًا غَضًّا
فجَعَله غـُـثاءً ( ۵ )
يابِسًا هَشِيمًا من بعدُ كالغُثاءِ(هو ما يَحمِلُه السَّيلُ من البَالِى من ورقِ الشجرِ مُخالِطًا زَبَدَه)
أحْوَى ( ۵ )
أسْودَ أو أسْمَرَ بعد الخُضرة
سنـُـقرؤك ( ٦ )
ما نُوحِى إليكَ بواسطةِ جبريل عليه السلام
فلا تنسى ( ٦ )
أبَدًا من قوةِ الحفظِ والإتقان
نـُـيـسّـرُك لليُـسرى ( ۸ )
نُوفِّقُكَ للطريقةِ اليُسرى فى كل أمْرٍ
يَصْـلى النّار الكبرى ( ۱۲ )
يَدخُلُ جهنَّمَ أو يُقاسِى حَرَّها
أفْـلح ( ۱٤ )
فازَ بالبُغْيةِ
تزَكّى ( ۱٤ )
تَطهِّرَ من الكفر والمعاصى
إنّ هَذا ( ۱۸ )
المذكورَ (الآياتِ الأربعَ السابقة)
