۸۷ - سورة الأعلى
( ۱ ) سبّح اسم ربّك :نَزِّهُه ومَجِّدهُ تعالى عمَّا لا يَليق به
( ۲ ) خَلق :أَوْجَدَ كلَّ شىءٍ بقدرتِه
( ۲ ) فسَوّى :بَينَ خَلْقِه فى الإحْكام والإتقان
( ۳ ) قـَـدّر :جعلَ الأشياءَ على مَقادِيرَمَخصوصهٍ
( ۳ ) فهَـدَى :فَوجَّه كلَّ واحدٍ منها إلى ما يَنبَغى له
( ٤ ) أخْرَجَ المرعى :أنـْـبَتَ العُشبَ رَطْبًا غَضًّا
( ۵ ) فجَعَله غـُـثاءً :يابِسًا هَشِيمًا من بعدُ كالغُثاءِ(هو ما يَحمِلُه السَّيلُ من البَالِى من ورقِ الشجرِ مُخالِطًا زَبَدَه)
( ۵ ) أحْوَى :أسْودَ أو أسْمَرَ بعد الخُضرة
( ٦ ) سنـُـقرؤك :ما نُوحِى إليكَ بواسطةِ جبريل عليه السلام
( ٦ ) فلا تنسى :أبَدًا من قوةِ الحفظِ والإتقان
( ۸ ) نـُـيـسّـرُك لليُـسرى :نُوفِّقُكَ للطريقةِ اليُسرى فى كل أمْرٍ
( ۱۲ ) يَصْـلى النّار الكبرى :يَدخُلُ جهنَّمَ أو يُقاسِى حَرَّها
( ۱٤ ) أفْـلح :فازَ بالبُغْيةِ
( ۱٤ ) تزَكّى :تَطهِّرَ من الكفر والمعاصى
( ۱۸ ) إنّ هَذا :المذكورَ (الآياتِ الأربعَ السابقة)
