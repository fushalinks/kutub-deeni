۳۱ - سورة لقمان
( ٦ ) لَهْو الحديث :الباطل المُلهي عن الخير و العبادة
( ٦ ) هُـزوا :سُخريةً – مهزوءًا بها
( ۷ ) ولّى مُستكبرا :أعرضَ مُتكبّرا عن تدبّرها
( ۷ ) وقْرا :صممًا مانعًا من السّماع
( ۱۰ ) بغير عَمَد :بغير دعائم و أساطينَ تُقيمُها
( ۱۰ ) رواسي :جبالا ثوابت
( ۱۰ ) أن تميد بكم :لِئلاّ تضطرب بكم
( ۱۰ ) بثّ فيها :نَشَرو فرَّق و أظهَرَ فيها
( ۱۰ ) زوج كريم :صِـنـْــفٍ حسنٍ كثير المنفعة
( ۱۲ ) لُقمان :كان صالحا حكيما و ليس نبيّا
( ۱۲ ) الحكمة :العقل و الفهم و الفطنة و إصابة القول
( ۱٤ ) وصّينا الإنسان :أمرناه و ألزمناه
( ۱٤ ) وهْنا :ضعفا
( ۱٤ ) فصاله :فِطامُـه عن الرّضاع
( ۱۵ ) أناب إليّ :رَجَع إليّ بالإخلاص و الطّاعة
( ۱٦ ) مثقال حبّة . . :وزنَ أصغر شيء. .
( ۱۸ ) لا تُصعّر خدّك للنّاس :لا تُمِلْ وجهَك عنهم كِبْرا و تعاظُما
( ۱۸ ) مرَحا :فَرَحا و بَطَرًا و خُيلاء
( ۱۸ ) مُختال فخور :مُتكبّرٍ، مُباهٍ متطاول بمناقبه
( ۱۹ ) اقصِدْ في مَشيك :توسّط فيه بين الإسراع و الإبطاء
( ۱۹ ) اغضض :اخفضْ و انقصْ
( ۲۰ ) سخّر لكم :لِمنافعكم و مصالحكم
( ۲۰ ) أسْـبَغ :أتمّ و أوسع و أكمل
( ۲۲ ) يُسْلِم وجهه. . :يُفوّض أمره كلّه . .
( ۲۲ ) استمسك :تمسّك و تعلـّـقَ و اعتصمَ
( ۲۲ ) بالعروة الوُثقى :بالعهد الأوثق الذي لا نقْضَ له
( ۲٤ ) عذاب غليظ :شديد ثقيل (عذاب النّار)
( ۲۷ ) يمدّه :يَزيده و يَنْصَبُّ إليه
( ۲۷ ) سبعة أبحُر :مملوءةٍ ماءً
( ۲۷ ) ما نفدت :ما فرغَتْ و ما فَنِيتْ
( ۲۷ ) كلمات الله :مقدوراته و عجائبه أو معلوماته
( ۲۹ ) يولج :يُدْخِل
( ۳۲ ) غَشِيهم موْجٌ :علاهم و غطّاهم
( ۳۲ ) كالظـّــلل :كالسّحاب . أو الجبال المظلـّـة
( ۳۲ ) فمِنهم مُقتصد :موفٍ بعهده . شاكر لله
( ۳۲ ) ختـّار كفور :غدّارٍ جَحود للنّعم
( ۳۳ ) يوما لا يجزي. . :لا يقضي فيه شيئا . .
( ۳۳ ) فلا تغرنّـكمْ :فلا تخدعنّكم و تُلهينّكم بلذاتها
( ۳۳ ) الغَرور :ما يغرّ و يخدع من شيطان و غيره
