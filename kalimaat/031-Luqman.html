لَهْو الحديث ( ٦ )

هُـزوا ( ٦ )
سُخريةً – مهزوءًا بها
ولّى مُستكبرا ( ۷ )

وقْرا ( ۷ )

بغير عَمَد ( ۱۰ )
بغير دعائم و أساطينَ تُقيمُها
رواسي ( ۱۰ )
جبالا ثوابت
أن تميد بكم ( ۱۰ )
لِئلاّ تضطرب بكم
بثّ فيها ( ۱۰ )
نَشَرو فرَّق و أظهَرَ فيها
زوج كريم ( ۱۰ )
صِـنـْــفٍ حسنٍ كثير المنفعة
لُقمان ( ۱۲ )
كان صالحا حكيما و ليس نبيّا
الحكمة ( ۱۲ )
العقل و الفهم و الفطنة و إصابة القول

مرَحا ( ۱۸ )
فَرَحا و بَطَرًا و خُيلاء
مُختال فخور ( ۱۸ )
مُتكبّرٍ، مُباهٍ متطاول بمناقبه
اقصِدْ في مَشيك ( ۱۹ )
توسّط فيه بين الإسراع و الإبطاء
اغضض ( ۱۹ )
اخفضْ و انقصْ
سخّر لكم ( ۲۰ )
لِمنافعكم و مصالحكم

بالعهد الأوثق الذي لا نقْضَ له
عذاب غليظ ( ۲٤ )
شديد ثقيل (عذاب النّار)
يمدّه ( ۲۷ )
يَزيده و يَنْصَبُّ إليه

مملوءةٍ ماءً
