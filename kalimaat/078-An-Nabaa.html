۷۸ - سورة النبإ
( ۱ ) عمّ ؟ :عن أيّ شيء عظيمِ الشّـأن؟
( ۲ ) عن النّبإ العظيم :عن القرآن أو البَعْث
( ٤ ) كلاّ :رَدْعٌ و زَجْر عن الإختلاف فيه
( ٦ ) الأرض مِهادًا :فِـرَاشًـا مُوَطّـأ للاستقرار عليها
( ۷ ) الجبال أوتادًا :كالأوتاد للأرض لئلاّ تميد
( ۸ ) خلقناكم أزواجًا :أصنافا ذكورًا و إناثـا للتـّـناسُـل
( ۹ ) نومَكم سُباتا :قـَـطْعًا لأعمالِكُمْ و راحة لأبْدانِكم
( ۱۰ ) الليل لِباسًا :ساتِرا لكم بظلمتِه كاللـّـباس
( ۱۱ ) النّـهار معاشا :تحَصّلون فيه ما تعيشون به
( ۱۲ ) سَبْـعًا شٍـدادًا :سمواتٍ قويّـاتٍ مُحْكَمات
( ۱۳ ) سِرَاجا وهّاجا :مِصْباحا مُنيرا وقـّـادًا (الشّمس)
( ۱٤ ) المُعصِرَات :السّحائب التي حان لها أن تـُمْطِر
( ۱٤ ) ماءً ثجّاجا :مُنـْـصَـبّـا بكثرةٍ مع التـّـتابُع
( ۱٦ ) جنّاتٍ ألفافا :بساتين مُـلتـَـفّة الأشجار
( ۱۸ ) فتأتون أفواجا :أمَمًـا أو جماعات مُختلفة الأحوال
( ۱۹ ) فكانت أبوابا :صارَتْ ذات أبواب و طرُق
( ۲۰ ) فكانت سرابا :فالسّراب الذي لا حَقيقة له
( ۲۱ ) كانت مِرْصادًا :مَوْضِعَ ترَصّدٍ و ترقّب للكافرين
( ۲۲ ) للطاغين مآبًـا :مَرْجـِعًـا و مأوى لهم
( ۲۳ ) أحقابـًا :دهورًا ممتابعة لا نهاية لها
( ۲٤ ) بَرْدًا :نـَوْمًا أو رَوْحًا من حرّ النّـار
( ۲۵ ) حميما :ماءً بالغا نِهاية الحرارة
( ۲۵ ) غسّـاقا :صديدا يسيل من جُـلودِهم
( ۲٦ ) جَزَاءً وفاقـًا :جزَيْنـاهم جَزَاءً مُوافِـقا لأعمالِهِم
( ۲۸ ) كِذابا :تكذيبًـا شديدا
( ۲۹ ) أحصيْناه كِتابا :حَفِظناه و ضبطناه مكتوبًـا
( ۳۱ ) مَفازًا :فوْزًا و ظفَـرًا بكلّ محْبوب
( ۳۳ ) كواعِب :فتيات ناهِدات (نساء الجَنّة)
( ۳۳ ) أترابًا :مُسْتـَـوياتٍ في السّن
( ۳٤ ) كأسًـا دِهاقـا :مُتـْـرَعَة مليئة من خَمْر الجَنّة
( ۳۵ ) لغْوًا :كلاما غير مُعْـتدّ به . أو قبيحًـا
( ۳۵ ) كِذابا :تكذيبًا
( ۳٦ ) عَطاءً حِسابا :إحسانا كافِـيا أو كثيرا
( ۳۷ ) خِطابـًا :إلا بإذنه
( ۳۸ ) الرّوح :جبريل عليه السلام
( ۳۹ ) مآبـا :مَرْجـِـعًـا بالإيمان و الطاعة
( ٤۰ ) كُـنْت تـُـرَابًـا :في هذا اليوم فلا أُعـذ ّب
