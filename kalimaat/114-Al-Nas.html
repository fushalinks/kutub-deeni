۱۱٤ - سورة الناس
( ۱ ) أعوذ :أعْـتصِـمُ و أستجير
( ۱ ) بربّ النـّـاس :مُرَبِّـيهِمْ ومُدبّر أحوالهم
( ۲ ) مَـلِـكِ النّـاس :مالِكِهِمْ مِلـْـكًـا تـامّـا
( ۳ ) إله النّاس :مَعْبُودِهِم الحقّ
( ٤ ) الوَسواس :المُوَسْوس جـِنـِّـيّـا أو إنـْـسِـيّـا
( ٤ ) الخنّاس :المُـتـَوَاري المُـخْـتـَفي
( ٦ ) الجـِنّة :الجـِـنّ
