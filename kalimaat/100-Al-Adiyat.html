۱۰۰ - سورة العاديات
( ۱ ) و العاديات :(قسمٌ) بالخيْـل تـَـعْـدو في الغَزْو
( ۱ ) ضَبْـحًا :هُوَ صَوْتُ أنـْـفاسِها إذا عَدَتْ
( ۲ ) فالموريات قـَـدْحا :المُـخْرجات النار بصكّ حَوافِرها
( ۳ ) فالمغيرات صُبْحا :المباغِتات للعدوّ وَقـْـتَ الصّباح
( ٤ ) فأثرْنَ به نقـْعًـا :هـيّـجْنَ في الصّـبْح غبارًا
( ۵ ) فَوَسَطنَ به جَمْعا :فـتـَـوَسّـــْطنَ فيه مِنَ الأعداء
( ٦ ) إنّ الإنسان :بطبْعِه إلاّ منْ رَحِم الله (جواب القسم)
( ٦ ) لكـَـنـُـودٌ :لـَـكفورٌ جَحودٌ
( ۸ ) إنّه لحُبّ الخَيْر :لأجْـل حُبّ المال
( ۸ ) لشديدٌ :لقويّ مُجدّ في تحصيله مُتـهالكٌ عليه
( ۹ ) بُـعْـثِـرَ :أُثير و أُخْرج و نـُـثِرَ
( ۱۰ ) حُصِّـل :جُمِعَ و اُظْهِر أو مُـيّـزَ
