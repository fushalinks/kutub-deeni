( ۳۳ ) ليُظهره :ليُعليه

( ٦۰ ) العاملين عليها :كالجُباة و الكُتّاب و الحرّاس
( ٦۰ ) في الرّقاب :في فكاك الأرقّاء أو الأسرى
( ٦۰ ) الغارمين :المدينين الذين لا يجدون قضاء
( ٦۰ ) في سبيل الله :في الغزو . أو في جميع القُرب

( ٦۱ ) أذن خير لكم :يسمع الخير و لا يسمع الشرّ

( ٦۹ ) خضتم :دخلتم في الباطل

( ۷۰ ) المأتفكات :المنقلبات (قرى قوم لوط)
