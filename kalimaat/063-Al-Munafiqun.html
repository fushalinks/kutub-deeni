٦۳ - سورة المنافقون
( ۲ ) جُـنّة
وقاية لأنفسهم و أموالهم
( ۳ ) آمنوا
بألسِنـَتهم لا غير
( ۳ ) فطُــبع
خُـتم بسبَب الكفر
( ۳ ) لا يفقهون
لا يعرفون حقيقة الإيمان
( ٤ ) خُشُبٌ مُسنّـدة
إلى الحائط ، أجسامٌ بلا أحلام
( ٤ ) هم العدوّ
الراسخون في العداوة
( ٤ ) أنّى يؤفكون ؟
كيف يُصرَفون عن الحقّ؟
( ۵ ) لوّوْا رؤوسهم
عَطفوها إعرَاضا و استهزاءً
( ۷ ) حتى ينفضّوا
كيْ يَـتفـرقوا عنه
( ۸ ) رَجَعْنـَـا
من غزوة بني المصطلق
( ۸ ) ليُـخْرجَنّ الأعزّ
الأشد و الأقوى يَعنون أنفسهم
( ۸ ) الأذلّ
الأضعف و الأهون . يعنون الرسول و المؤمنين
( ۸ ) و لله العزّة
الغَـلبة و القهْر
( ۹ ) لا تـلهكم
لا تشغَـلْـكم و تصْرفكمْ
( ۹ ) ذِكر الله
عِبادَته و طاعته و مراقبته
( ۱۰ ) لوْلا أخّرتني
هلاّ أمهلتني و أخّرت أجَلي
