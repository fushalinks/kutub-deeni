۹۵ - سورة التين
( ۱ ) و التين و الزيتون :(قسمٌ) بِمنْـبـَتـَيـْهما من الأرض المباركة
( ۲ ) و طور سِنين :جَبَلِ المُناجاة ِللكَلِيم عليه السلام
( ۳ ) البلد الأمين :مكة المكرّمة
( ٤ ) لقـَـدْ خَلـقـْـنا :(جوابُ القسم) بالأربعةِ قَبلَه
( ٤ ) أحْسَنِ تقويم :أكْمَلِ تَعدِيلٍ وأحسنِ صُورَةٍ
( ۵ ) ردَدْناه :ردَدْنا الكافرَ أو جـِنسَ الإنسان
( ۵ ) أسفل سافلين :إلى النار أو الهَرمِ وأرذَلِ العُمُر
( ٦ ) غير ممنون :غيرُ مَقطُوعٍ عنهم
( ۷ ) بالدّين :بالجزاءِ بعد البعث والحساب
