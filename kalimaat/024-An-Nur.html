۲٤ - سورة النور
( ۱ ) فرضناها :أوجبنا أحكامها عليكم
( ۲ ) كلّ واحد :إذا كان حُرّا غير مُحصن
( ٤ ) يرمون المحصنات :يقذفون العفيفات بالزّنى
( ۸ ) يدرأ عنها العذاب :يدفع عنها العقوبة
( ۱۱ ) بالإفك :أقبح الكذب و أفحشه
( ۱۱ ) عصبة منكم :جماعة منكم
( ۱۱ ) تولّى كبره :تحمّل معظمه (رأس المنافقين)
( ۱٤ ) أفضتم فيه :خضتم فيه من حديث الإفك
( ۱۵ ) تحسبونه هيّـنا :تظنونه سهلا لا تبعة له
( ۱٦ ) سبحانك :تعجّب من شناعة هذا الإفك
( ۱٦ ) بهتان :كذب يُحيّر سامعه لفظاعته
( ۲۱ ) خطوات الشيطان :طرقه و أثاره و مذاهبه
( ۲۱ ) بالفحشاء :ما عَظُم قبحه من الذنوب
( ۲۱ ) المنكر :ما ينكره الشّرع و يكرهه الله
( ۲۱ ) ما زكى :ما تطهّر من دنس الذنوب
( ۲۲ ) لا يأتـلِ :لا يحلف أو لا يُقصّر
( ۲۲ ) أولوا الفضل :أصحاب الزيادة في الدّين
( ۲۲ ) السّعة :الغنى
( ۲۳ ) المحصنات :العفائف ، و مثلهن المحصنون
( ۲۵ ) دينهم الحقّ :جزاءهم الثابت لهم بالعدل
( ۲۷ ) تستأنسوا :تستأذنوا ممن يملك الإذن
( ۲۸ ) أزكى لكم :أطهر لكم من دنس الريبة و الدّناءة
( ۲۹ ) جناح :إثم
( ۲۹ ) متاع لكم :منفعة و مصلحة لكم
( ۳۰ ) يغضوا من أبصارهم :يكفوا نظرهم عن المحرّمات
( ۳۱ ) زينتهن :مواضع زينتهن من الجسد
( ۳۱ ) ما ظهر منها :الوجه و الكفين و القدمين
( ۳۱ ) و ليضربن :و ليلقين و يسدلن
( ۳۱ ) بخمورهن :أغطية رؤوسهن ( المقانع)
( ۳۱ ) على جيوبهن :على مواضعها (صدورهن و ما حواليها)
( ۳۱ ) لبعولتهن :لأزواجهن
( ۳۱ ) نسائهن :المختصّات بهن بالصحبة أو الخدمة
( ۳۱ ) أولي الإربة :أصحاب الحاجة إلى النساء
( ۳۱ ) لم يظهروا :لم يبلغوا حد الشهوة
( ۳۲ ) انكحوا الأيامى :من لا زوج لها ، و من لا زوجة له
( ۳۳ ) يبتغون الكتاب :يطلبون عقد المكاتبة المعروف
( ۳۳ ) فتياتكم :إماءكم
( ۳۳ ) البغاء :الزّنى
( ۳۳ ) تحصّـنا :تعفّــفا و تصوّنا عنه
( ۳۵ ) الله نور السّمـوات :منوّرهما أو هادي أهلهما أو موجدهما
( ۳۵ ) كمشكاة :كنور كوّة غير نافذة
( ۳۵ ) مصباح :سراج ضخم ثاقب
( ۳۵ ) زجاجة :قنديل من الزجاج صاف أزهر
( ۳۵ ) كوكب درّي :مضيء متلألئ صافٍ
( ۳٦ ) بيوت :هي المساجد كلّها
( ۳٦ ) أن ترفع :أن تُعظّم و تـُطهّر
( ۳٦ ) بالغدوّ و الآصال :أول النهار و آخره
( ۳۸ ) بغير حساب :بلا نهاية لما يُعطي ، أو بتوسّع
( ۳۹ ) كسراب :شعاع يُرى ظهرا في البرّ عند اشتداد الحرّ كالماء السارب
( ۳۹ ) بقيعة :في منبسطة من الأرض متـسع
( ٤۰ ) بحر لجّي :عميق كثير الماء
( ٤۰ ) يغشاه :يعلوه و يغطّيه
( ٤۰ ) سحاب :غيم يحجب أنوار السّماء
( ٤۱ ) صافات :باسطات أجنحتهن في الهواء
( ٤۳ ) يُزجي سحابا :يسوقه برفق إلى حيث يريد
( ٤۳ ) يجعله ركاما :مجتمعا بعضه فوق بعض
( ٤۳ ) الودق :المطر
( ٤۳ ) من خلاله :من فتوقه و مخارجه
( ٤۳ ) سنا برقه :ضوء برقه و لمعانه
( ٤۹ ) مذعنين :منقادين مُطيعين
( ۵۰ ) أن يحيف :أن يجور
( ۵۳ ) جهد أيمانهم :مجتهدين في الحلف بأغلظها و أوكدها
( ۵۳ ) طاعة معروفة :طاعتكم طاعة معروفة باللسان
( ۵٤ ) ما حُمّل :ما أمر به من التبليغ
( ۵٤ ) ما حُمّلتم :ما أمرتم به من الطاعة و الإنقياد
( ۵۷ ) مُعجزين :فائتين من عذابنا بالهرب
( ۵۸ ) جناح :حرج في الدخول بلا استئذان
( ٦۰ ) القواعد من النساء :العجائز اللاتي قعدن عن الحيض
( ٦۰ ) متبرجات بزينة :مظهرات للزينة الخفيّة
( ٦۱ ) ما ملكتم مفاتحه :مما في تصرّفكم وكالة أو حفظا
( ٦۱ ) أشتاتا :متفرّقين
( ٦۲ ) أمر جامع :أمر مهمّ يجب اجتماعهم له
( ٦۳ ) دعاء الرّسول :دعوته لكم للاجتماع أو نداءكم له
( ٦۳ ) يتسلّلون منكم :يخرجون منكم تدريجا في خفية
( ٦۳ ) لِواذا :يستتر بعضهم ببعض في الخروج
( ٦۳ ) يخالفون عن أمره :يعرضون أو يصدون عنه
( ٦۳ ) فتنة :بلاء و محنة في الدنيا
