۹۳ - سورة الضحى
( ۱ ) و الضحى :(أقسم) بوَقت ارتفاع الشمس
( ۲ ) سجى :سَكنَ أو اشتدّ ظلامه
( ۳ ) ما ودّعك ربّـك :ما تـَـرَك منـْـذ اختارَك (جواب القسم)
( ۳ ) ما قـلى :ما أبْغَـضَـك منذ ُ أحبّـك
( ٦ ) ألمْ يَجدْك . . :ألمْ يَعْـلمْـك ربّـكَ – قَدْ عَـلِـمَك . .
( ٦ ) يَـتيما :طِفلاً مات أبوك و أنتَ جنين
( ٦ ) فآوى :فَـضمّـك إلى مَنْ يَكفـلك و يَرْعاك
( ۷ ) ضالاّ :غافِلاً عنْ أحكام الشّرائع
( ۷ ) فهَدَى :فهَـدَاك إلى مناهجها بما أوحى إليك
( ۸ ) عائلا :فقيرا عَديما
( ۸ ) فأغـنى :فَرضّـاك بما أعطاك و مَنـَـحَك
( ۹ ) فلا تـَقـْـهَرْ :فلا تـَـغْـلِـبُه على ماله و لا تسْـتـَـذلـّـه
( ۱۰ ) فلا تـَـنـْـهَرْ :فلا تـَزْجُرْه ، و ارفـُـقْ به
