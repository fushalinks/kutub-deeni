۷ - سورة الأعراف
( ۲ ) حَرَجٌ مِنْهُ :ضيقٌ من تبليغه خشية التكذيب
( ٤ ) بَيَاتًا :بائتين أو ليلاً و هو نائمون
( ٤ ) هُمْ قَائِلُونَ :مستريحون نصف النهار (القيلولة)
( ۱۸ ) مَذْءُومًا :مذموماً أومعيبا أو محقراً لعيناً
( ۱۸ ) مَدْحُورًا :مطروداً مبعداً
( ۲۰ ) سَوْآتِهِمَا :عوراتهما
( ۲۲ ) فَدَلَّاهُمَا بِغُرُورٍ :فأنزلهما عن رتبة الطاعة بخداع
( ۲۲ ) طَفِقَا يَخْصِفَانِ :شرعا وأخذا يلزقان
( ۲٦ ) رِيشًا :لباس زينة . أو مالاً
( ۲۹ ) وَأَقِيمُوا وُجُوهَكُمْ :توجهوا إلى عبادته مستقيمين
( ۲۹ ) عِنْدَ كُلِّ مَسْجِدٍ :في كل وقت سجود أو مكانه
( ۳۸ ) ادَّارَكُوا فِيهَا :تلاحقوا في النّار واجتمعوا فيها
( ٤۰ ) حَتَّى يَلِجَ الْجَمَلُ فِي سَمِّ الْخِيَاطِ :يدخل الجمل
( ٤۰ ) فِي سَمِّ الْخِيَاطِ :ثقب الإبرة
( ٤۱ ) مِهادٌ :فراشٌ ،أي مستقر
( ٤۱ ) غَوَاشٍ :أغطية كاللُّحُف
( ٤۳ ) غِلٍّ :حقد ٍ وضغنٍ وعداوة
( ٤٦ ) الْأَعْرَافِ :أعالي هذا السور وشرفاته
( ۵۳ ) هَلْ يَنْظُرُونَ إِلَّا تَأْوِيلَهُ :عاقبة مواعيد الكتاب (القرآن) و مآلها من البعث و الحساب و الجزاء
( ۵٤ ) يَطْلُبُهُ حَثِيثًا :يطلب الليل النهار طلباً سريعاً
( ۵۵ ) تَضَرُّعًا :مُظهرين الضراعة والذلة والإستكانة والخشوع
( ۵۷ ) أَقَلَّتْ سَحَابًا :حملته ورفعته
( ۵۸ ) نَكِدًا :عسراً أو قليلاً لا خير فيه
( ٦۹ ) بَسْطَةً :قوةً وعِظم أجسام
( ۷٤ ) وَلَا تَعْثَوْا فِي الْأَرْضِ مُفْسِدِينَ :لا تفسدوا إفسادا شديدا
( ۷۷ ) عَتَوْا عَنْ أَمْرِ رَبِّهِمْ :استكبروا
( ۷۸ ) الرَّجْفَةُ :الزّلزلة الشّديدة . أو الصّيحة
( ۷۸ ) جَاثِمِينَ :هامدين موتى لا حراك بهم

( ۸۳ ) الْغَابِرِينَ :الباقين في العذاب كأمثالها
( ۹۲ ) كَأَنْ لَمْ يَغْنَوْا فِيهَا :لم يقيموا ناعمين في دارهم
( ۹٤ ) يَضَّرَّعُونَ :يتذلّلون و يخضعون
( ۹۵ ) حَتَّى عَفَوْا :كثُروا و نموا عددا و مالا
( ۱۰۰ ) أَوَلَمْ يَهْدِ لِلَّذِينَ يَرِثُونَ الْأَرْضَ :لم يبيّن الله للذين آمنوا
( ۱۰۲ ) وَمَا وَجَدْنَا لِأَكْثَرِهِمْ مِنْ عَهْدٍ :من وفاء بما أوصيناهم
( ۱۰۵ ) حَقِيقٌ عَلَى أَنْ لَا أَقُولَ عَلَى اللَّهِ إِلَّا الْحَقَّ :حريص على أن . . أو خليق بأن . .
( ۱۱۱ ) قَالُوا أَرْجِهْ وَأَخَاهُ : أخّر أمر عقوبتهما و لا تعجل
( ۱۱۷ ) تَلْقَفُ مَا يَأْفِكُونَ :تبتلع أو تتناول بسرعة
( ۱۲٦ ) أَفْرِغْ عَلَيْنَا صَبْرًا :أفض أو صبّ علينا
( ۱۳۰ ) بالسّنين :بالجدوب و القحوط
( ۱۳۱ ) يَطَّيَّرُوا بِمُوسَى وَمَنْ مَعَهُ :يتشاءموا
( ۱۳۱ ) طَائِرُهُمْ عِنْدَ اللَّهِ :شؤمهم عقابهم الموعود في الآخرة
( ۱۳۳ ) وَ الْقُمَّلَ :الدَّبى أو القراد أو القمل المعروف
( ۱۳٤ ) الرِّجْزَ :العذاب بما ذكر من الآيات
( ۱۳۵ ) هُمْ يَنْكُثُونَ :ينقضون عهدهم الذي أبرموه
( ۱۳۷ ) وَمَا كَانُوا يَعْرِشُونَ :من الجنّات أو يرفعون من الأبنية
( ۱۳۹ ) مُتَبَّرٌ مَا هُمْ فِيهِ :مُهلَكٌ مُدمّر
( ۱٤۱ ) يَسُومُونَكُمْ سُوءَ الْعَذَابِ :يذيقونكم أو يكلّفونكم
( ۱٤۱ ) وَيَسْتَحْيُونَ نِسَاءَكُمْ :يستبقون- بناتكم للخدمة
( ۱٤۳ ) فَلَمَّا تَجَلَّى رَبُّهُ لِلْجَبَلِ جَعَلَهُ دَكًّا :مدكوكا متفتّتا
( ۱٤۳ ) خَرَّ مُوسَى صَعِقًا :مغشيّا عليه
( ۱٤۸ ) عِجْلًا جَسَدًا لَهُ خُوَارٌ :مُجسّدا أي أحمر من ذهب
( ۱٤۹ ) وَلَمَّا سُقِطَ فِي أَيْدِيهِمْ :نَدِموا أشدّ النّدم
( ۱۵۰ ) فَلَا تُشْمِتْ بِيَ الْأَعْدَاءَ :فلا تسرّهم بما تنال منّي من المكروه
( ۱۵٦ ) إِنَّا هُدْنَا إِلَيْكَ :تبنا و رجعنا إليك
( ۱۵۷ ) وَيَضَعُ عَنْهُمْ إِصْرَهُمْ :عهدهم بالعمل بما في التوراة
( ۱۵۷ ) الْأَغْلَالَ الَّتِي كَانَتْ عَلَيْهِمْ :التكاليف الشّاقة في التّوراة
( ۱۵۷ ) عَزَّرُوهُ :وقّروه و عظّموه
( ۱٦۰ ) اثْنَتَيْ عَشْرَةَ أَسْبَاطًا :جماعات ، كالقبائل في العرب
( ۱٦۰ ) فَانْبَجَسَتْ مِنْهُ اثْنَتَا عَشْرَةَ عَيْنًا :فانفجرت
( ۱٦۰ ) الْمَنَّ :مادّة صَمغيّة حُلوة كالعسل
( ۱٦۰ ) السَّلْوَى :الطّائر المعروف بالسّماني
( ۱٦۱ ) وَقُولُوا حِطَّةٌ :مسألتنا حطّ ذنوبنا عنّا
( ۱٦۳ ) شُرَّعًا :ظاهرة على وجه الماء كثيرة
( ۱٦۳ ) وَيَوْمَ لَا يَسْبِتُونَ :لا يُراعون أمر السّبت
( ۱٦٦ ) فَلَمَّا عَتَوْا عَنْ مَا نُهُوا عَنْهُ :استكبروا و استعصوا
( ۱٦۷ ) يَسُومُهُمْ سُوءَ الْعَذَابِ :يُذيقهم و يكلّـفهم
( ۱۷۱ ) نتقنا الجبل :رفعناه و قلعناه
( ۱۷۵ ) فَانْسَلَخَ مِنْهَا :فخرج منها بكفره بها
( ۱۷٦ ) أَخْلَدَ إِلَى الْأَرْضِ :ركن إلى الدّنيا و رضي بها
( ۱۷٦ ) تَحْمِلْ عَلَيْهِ يَلْهَثْ :تشدُد عليه و تزجزه
( ۱۷٦ ) يَلْهَثْ :يُخرج لسانه بالنّـفَس الشديد
( ۱۸۰ ) يُلْحِدُونَ فِي أَسْمَائِهِ :يميلون و ينحرفون إلى الباطل
( ۱۸۲ ) سَنَسْتَدْرِجُهُمْ مِنْ حَيْثُ لَا يَعْلَمُونَ :سنستدنيهم إلى الهلاك بالإنعام
( ۱۸٤ ) مَا بِصَاحِبِهِمْ مِنْ جِنَّةٍ :جُنون كما يزعمون
( ۱۸۷ ) أَيَّانَ مُرْسَاهَا :متى إثباتها و وقوعها ؟
( ۱۸۷ ) لَا يُجَلِّيهَا لِوَقْتِهَا :لا يُظهرها و لا يكشف عنها
( ۱۸۷ ) كَأَنَّكَ حَفِيٌّ عَنْهَا :باحث عنها عالمٌ بها
( ۱۹۹ ) خُذِ الْعَفْوَ :ما عفا و تيسّر من أخلاق النّاس
( ۱۹۹ ) وَأْمُرْ بِالْعُرْفِ :بالمعروف حُسنُـه في الشّرع
( ۲۰۰ ) يَنْزَغَنَّكَ مِنَ الشَّيْطَانِ نَزْغٌ :يُصيبنّك . أو يصرفنّك
( ۲۰۰ ) نَزْغٌ :وسوسة . أو صارف
( ۲۰۱ ) مَسَّهُمْ طَائِفٌ مِنَ الشَّيْطَانِ :أصابتهم لمّة أي وسوسة ما
( ۲۰۲ ) يَمُدُّونَهُمْ فِي الْغَيِّ :تُعاونهم الشّياطين في الضّلال
