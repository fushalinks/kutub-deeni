( ۲۰۰ ) نزغ :وسوسة . أو صارف
( ۲۰۱ ) مسّهم طائف :أصابتهم لمّة أي وسوسة ما
( ۲۰۱ ) تذكّروا :أمر الله و نهيه و عداوة الشّيطان
( ۲۰۲ ) يمدّونهم في الغيّ :تُعاونهم الشّياطين في الضّلال
( ۲۰۲ ) لا يُقصِرون :لا يكفّون عن إغوائهم
( ۲۰۳ ) اجتبيتها :اختلقتها و اخترعتها من عندك
( ۲۰۳ ) هذا بصائر :القرآن حجج بيّنة و براهين نيّرة
( ۲۰۵ ) تضرّعا :مُظهرا الضَّراعة و الذلة
( ۲۰۵ ) خيفة :خائفا من عقابه
( ۲۰۵ ) بالغدوّ و الآصال :أوائل النّهار و أواخره . أي في كلّ وقت
( ۲۰٦ ) له يسجدون :يُصلّون و يعبدون (آية سجدة)
