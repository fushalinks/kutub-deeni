۷٤ - سورة المدثر
( ۱ ) المدثر :المتغشّى بثيابه (النبي صلى الله عليه و سلم)
( ۳ ) رَبّـك فكبر :اخصصْ رَبّـك بالتكبير و التعظيم
( ٤ ) ثيابك فطهّر :كناية عن تطهير النفس من المذامّ
( ۵ ) الرجْز فاهجر :اهجرْ المآثم الموجبة للعذاب
( ٦ ) لا تمنن تستكثر :لا تـُـعْـطِ طالبا الكثير عِوَضا عنه
( ۸ ) نقِـر في النّـاقور :نـُـفخ في الصّور للبعث و النّشور
( ۱۱ ) ذرني :دَعْـنِي و خَـلـّـني (تهديد و وعيد)
( ۱۲ ) ملا ممْدودا :كثيرا دائما غير منقطع عنه
( ۱۳ ) بنين شهودًا :حضورا معه ، لا يُفارقونه للتكسّب لِغناهم عنه
( ۱٤ ) مهّدت له :بَسَطتُ له النّعمة و الرّياسة و الجَاه
( ۱٦ ) كلاّ :كلمة رَدْع و زجر عن الطمع الفارغ
( ۱٦ ) لآياتنا عنيدا :معاندا جَاحِدًا أو مُـجانيا للحقّ
( ۱۷ ) سأرهقه صعودا :سَأكلـّـفه عَذابا شاقّـا
( ۱۸ ) قدّر :هيّـأ في نفسِه قَـوْلاً طاعِـنـًـا في القرآن
( ۱۹ ) فقتل :لُعِنَ و عُذب أو قُـبِّـح
( ۲۱ ) نظر :تأمّل فيما قدّرَ و هيّـأ من الــّطـعْن
( ۲۲ ) عَبَس :قـَــّطبَ وَجْهَه لمّا ضاقـَـتْ عليه الحِيَل
( ۲۲ ) بَسَر :اشتدّ في العبوس و كُـلـُـوح الوَجْه
( ۲٤ ) سِحْرٌ يُؤثر :يُرْوَى و يُـتـَـعـلـّـمُ من السَّحَرَة
( ۲٦ ) سأصليه سَقـَـر :سأدْخِلـُـه جهنّم
( ۲۹ ) لوّاحة للبشر :مُسودّة للجلود ، مُحْـرقَة لها
( ۳۱ ) فتنة :سبب فتنة و ضلال
( ۳۱ ) و ما هي :و ما سَقـَـر
( ۳۳ ) و الليل إذ أدبَر :وَلـّـى و ذهَب (قسم)
( ۳٤ ) و الصبح إذا أسْـفَر :أضاء و انكشف (قسَم)
( ۳۵ ) إنها لإحدى الكُـبـَر :لإحدى الدّواهي العظيمة (جوابه)
( ۳۷ ) أنْ يتقـدّم :إلى الخيْر و الطاعة
( ۳۸ ) بما كَسَبَتْ رَهينة :مَرْهونة عنده تعالى بعَمَلِـها
( ٤۲ ) ما سَلكَكم ؟ :أيّ شيء أدْخَلكمْ؟
( ٤۵ ) كنّـا نخوض :نـَـشْرعُ في الباطل لا نُـبالي به
( ٤٦ ) بيوم الدّين :بيَوْمِ البعْثِ و الحِسَاب و الجزاء
( ۵۰ ) حُمُرٌ مُـسْـتـَـنفِـرَة :حُمُـرٌ وَحْشِـيّة ، شديدة الـِنّـفَار
( ۵۱ ) قـَـسْورة :أسدٍ . أو الرّماة القـنّـص
( ۵٦ ) أهل التقوَى :أهل أو يَـتـّـقِـيَهُ عبادُه
