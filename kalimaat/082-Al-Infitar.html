۸۲ - سورة الإنفطار
السماء انفطرت ( ۱ )
انشقـّـتْ عند قِـيَـام الساعة
الكواكب انتثرت ( ۲ )
تساقطتْ مُـتـفـرقة
البحار فجّرت ( ۳ )
شقـّـقتْ جَوانِبـُـها فصارتْ بَحْرًا واحِدًا
القبور بُعثرت ( ٤ )
قـُـلِبَ ترابُها ، و أخْــِرج موتاها
ما غَرّك بربّك ؟ ( ٦ )
ما خدعَك و جَرّأك على عِصْيانه ؟
فسوّاك ( ۷ )
جَعَل أعْضاءَك سَويّة سليمة
فعَدَلك ( ۷ )
جَـعَلك معتدلا متناسب الخَـلـْق
تكذبون بالدّين ( ۹ )
بالبعث أو الجزاء أو بالإسلام
الأبرار ( ۱۳ )
الذين بَرّوا و صدقوا في إيمانهم
يصْـلونها ( ۱۵ )
يَدْخُـلونـَـها ، أو يُـقـاسونَ حَرّها
