سورة العصر
بِسمِ ٱلله الرَّحْمٰنِ الرَّحِيـمِ
ﻣﻜﻴﺔ أو ﻣﺪﻧﻴﺔ وﺁﻳﺎﺗﻬﺎ ﺛﻼث
This sura is either Makkan or Medinan has 3 ayats.
{ والعصر } الدهر أو ما بعد الزوال إلى الغروب أو صلاة العصر.
1. By the Late Afternoon – the word ‘asr means either time, or between midday and sunset, or the ‘Asr prayer!
{ إنَّ الإنسان } الجنس { لفي خُسر } في تجارته.
2. Truly man – meant generically – is in loss in his life-transaction,
{ إلا الذين آمنوا وعملوا الصالحات } فليسوا في خسران { وتواصوا } أوصى بعضهم بعضاً { بالحق } الإيمان { وتواصوا بالصبر } على الطاعة وعن المعصية.
3. …except for those who believe and do right actions – who are not in loss, and urge each other to the truth – which means faith, and urge each other to steadfastness – to obey and not to disobey.
