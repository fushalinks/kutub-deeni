سورة التكاثر
بِسمِ ٱلله الرَّحْمٰنِ الرَّحِيـمِ
ﻣﻜﻴﺔ وﺁﻳﺎﺗﻬﺎ ﺛﻤﺎن
This sura is Makkan has 8 ayats.
{ أَلْهَاكُمْ } شَغَلَكُمْ عَنْ طَاعَة اللَّه { التَّكَاثُر } التَّفَاخُر بِالْأَمْوَالِ وَالْأَوْلَاد وَالرِّجَال
1. Fierce competition for this world distracted you from obeying Allah. This means boasting to one another about wealth, children and followers –
{ حَتَّى زُرْتُمْ الْمَقَابِر } بِأَنْ مُتُّمْ فَدُفِنْتُمْ فِيهَا أَوْ عَدَدْتُمْ الْمَوْتَى تَكَاثُرًا
2. …until you went down to the graves – when you die and are buried in them, or you count the dead in boasting to one another.
{ كَلَّا } رَدْع { سَوْفَ تَعْلَمُونَ }
3. No indeed, you will soon know!
{ ثُمَّ كَلَّا سَوْفَ تَعْلَمُونَ } سُوء عَاقِبَة تَفَاخُركُمْ عِنْد النَّزْع ثُمَّ فِي الْقَبْر
4. Again no indeed, you will soon know the evil end of your mutual boasting, when you die and then are in the grave!
{ كَلَّا } حَقًّا { لَوْ تَعْلَمُونَ عِلْم الْيَقِين } عِلْمًا يَقِينًا عَاقِبَة التَّفَاخُر مَا اِشْتَغَلْتُمْ بِهِ
5. No indeed, if you only knew with the Knowledge of Certainty what the result of the competition which distracted you would be –
{ لَتَرَوُنَّ الْجَحِيم } النَّار جَوَاب قَسَم مَحْذُوف وَحُذِفَ مِنْهُ لَام الْفِعْل وَعَيْنه وَأُلْقِيَتْ حَرَكَتهَا عَلَى الراء
6. …you will certainly see the Blazing Fire!
{ ثُمَّ لَتَرَوُنَّهَا } تَأْكِيد { عَيْن الْيَقِين } مَصْدَر لِأَنَّ رَأَى وَعَايَنَ بِمَعْنًى وَاحِد
7. Then you will certainly see it with the Eye of Certainty. This repetition is for emphasis.
{ ثُمَّ لَتُسْأَلُنَّ } حُذِفَ مِنْهُ نُون الرَّفْع لِتَوَالِي النُّونَات وَوَاو ضَمِير الْجَمْع لِالْتِقَاءِ السَّاكِنَيْنِ { يَوْمئِذٍ } يَوْم رُؤْيَتهَا { عَنْ النَّعِيم } مَا يُلْتَذّ بِهِ في الدنيا من الصحة والفراغ والأمن والمطعم والمشرب وغير ذلك
8. Then you will be asked that Day – when you see this – about the pleasures you enjoyed in this world, in terms of good health, free time, security, food and drink, and other thingS.
