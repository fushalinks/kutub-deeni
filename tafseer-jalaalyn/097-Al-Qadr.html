سورة القدر
بِسمِ ٱلله الرَّحْمٰنِ الرَّحِيـمِ
ﻣﻜﻴﺔ أو ﻣﺪﻧﻴﺔ وﺁﻳﺎﺗﻬﺎ ﺧﻤﺲ أو ﺳﺖ
This sura is Makkan or Madinan and has 5 or 6 ayats.
{ إِنَّا أَنْزَلْنَاهُ } أَيْ الْقُرْآن جُمْلَة وَاحِدَة مِنْ اللَّوْح الْمَحْفُوظ إلَى السَّمَاء الدُّنْيَا { فِي لَيْلَة الْقَدْر } أَيْ الشَّرَف الْعَظِيم
1. Truly We sent it (the Qur’an) down all at once from the Preserved Tablet to the lowest heaven on the Night of Power so called because of its immense honour.
{ وَمَا أَدْرَاك } أَعْلَمَك يَا مُحَمَّد { مَا لَيْلَة الْقَدْر } تَعْظِيم لِشَأْنِهَا وَتَعْجِيب مِنْهُ
2. And what will convey to you, Muhammad, what the Night of Power is – and its immensity and wonder?
{ لَيْلَة الْقَدْر خَيْر مِنْ أَلْف شَهْر } لَيْسَ فِيهَا لَيْلَة الْقَدْر فَالْعَمَل الصَّالِح فِيهَا خَيْر مِنْهُ فِي أَلْف شَهْر لَيْسَتْ فِيهَا
3. The Night of Power is better than a thousand months that do not contain the Night of Power. Righteous actions done during it are better than those done in a thousand months that do not contain it.
{ تَنَزَّل الْمَلَائِكَة } بِحَذْفِ إِحْدَى التَّاءَيْنِ مِنْ الْأَصْل { وَالرُّوح } أَيْ جِبْرِيل { فِيهَا } فِي اللَّيْلَة { بِإِذْنِ رَبّهمْ } بِأَمْرِهِ { مِنْ كُلّ أَمْر } قَضَاهُ اللَّه فِيهَا لِتِلْكَ السَّنَة إِلَى قَابِل وَمِنْ سَبَبِيَّة بمعنى الباء
4. In it the angels and the Ruh (Jibril) descend in the night by their Lord’s authority and command with every ordinance – the decrees of Allah for that coming year.
{ سَلَام هِيَ } خَبَر مُقَدَّم وَمُبْتَدَأ { حَتَّى مَطْلَع الْفَجْر } بِفَتْحِ اللَّام وَكَسْرهَا إِلَى وَقْت طُلُوعه جُعِلَتْ سَلَامًا لِكَثْرَةِ السَّلَام فِيهَا مِنْ الْمَلَائِكَة لا تَمُرّ بِمُؤْمِنٍ وَلَا بِمُؤْمِنَةٍ إِلَّا سَلَّمَتْ عَلَيْهِ
5. It is Peace until the coming (read as matla‘ and matli‘) of the dawn. It is called “Peace”, salaam, because it contains many greetings, salaam, from the angels during it. They do not pass by any believer, man or woman, without greeting them.
