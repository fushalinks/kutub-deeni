سورة القارعة
بِسمِ ٱلله الرَّحْمٰنِ الرَّحِيـمِ
ﻣﻜﻴﺔ وﺁﻳﺎﺗﻬﺎ إﺣﺪى ﻋﺸﺮة ﺁﻳﺔ
This sura is Makkan has 11 ayats.
{ الْقَارِعَة } الْقِيَامَة الَّتِي تَقْرَع الْقُلُوب بِأَهْوَالِهَا
1. The Crashing Blow!
{ مَا الْقَارِعَة } تَهْوِيل لِشَأْنِهَا وَهُمَا مُبْتَدَأ وَخَبَر خَبَر القارعة
2. What is the Crashing Blow? This is the Day of Rising, which will crush the hearts with its terrors. It is repeated to emphasise its dreadfulness.
{ وَمَا أَدْرَاك } أَعْلَمَك { مَا الْقَارِعَة } زِيَادَة تَهْوِيل لَهَا وَمَا الْأُولَى مُبْتَدَأ وَمَا بَعْدهَا خَبَره وَمَا الثَّانِيَة وَخَبَرهَا فِي مَحَلّ الْمَفْعُول الثَّانِي لأدرى
3. What will convey to you what the Crashing Blow is? This question is to emphasize further the terrible nature of that Day.
{ يَوْم } نَاصِبه دَلَّ عَلَيْهِ الْقَارِعَة أَيْ تَقْرَع { يَكُون النَّاس كَالْفِرَاشِ الْمَبْثُوث } كَغَوْغَاء الْجَرَاد الْمُنْتَشِر يَمُوج بَعْضهمْ فِي بَعْض لِلْحِيرَةِ إِلَى أَنْ يدعوا للحساب
4. It is the Day when mankind will be crushed and dispersed like scattered moths – confused locusts, surging into one another out of bewilderment at being summoned for the Reckoning –
{ وَتَكُون الْجِبَال كَالْعِهْنِ الْمَنْفُوش } كَالصُّوفِ الْمَنْدُوف فِي خِفَّة سَيْرهَا حَتَّى تَسْتَوِي مَعَ الْأَرْض
5. …and the mountains like insubstantial tufts of coloured wool.
{ فَأَمَّا مَنْ ثَقُلَتْ مَوَازِينه } بِأَنْ رَجَحَتْ حَسَنَاته على سيئاته
6. As for him whose balance is heavy – him whose good actions outweigh his evil actions,
{ فَهُوَ فِي عِيشَة رَاضِيَة } فِي الْجَنَّة أَيْ ذَات رِضًى بِأَنْ يَرْضَاهَا أَيْ مَرْضِيَّة لَهُ
7. …he will have a most pleasant life in the Garden. He will be delighted with it.
{ وَأَمَّا مَنْ خَفَّتْ مَوَازِينه } بِأَنْ رَجَحَتْ سَيِّئَاته على حسناته
8. But as for him whose balance is light – him whose evil deeds outweigh his good deeds,
{ فأمه } فمسكنه { هاوية }
9. …his motherland (dwelling-place) is Hawiya.
{ وما أدراك ما هيه } أي ما هاوية
10. And what will convey to you what that is?
هي { نار حامية } شَدِيدَة الْحَرَارَة وَهَاء هِيَهْ لِلسَّكْتِ تُثْبَت وَصْلًا ووقفا وفي قراءة تحذف وصلا
11. A raging Fire!
